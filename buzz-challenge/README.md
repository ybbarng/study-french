Buzzvil 사내 챌린지에 분기별로 참여한 기록. 각 분기 폴더에 도전 과제·인증 방법·월별 결과가 정리돼 있다.

## 챌린지 방식
- **Focus Challenge**: 매달 달성 여부를 인증하는 단기집중형.
- 동일/유사 주제로 이어갈 경우 **이전 분기보다 높은 목표**를 설정한다 (규정).
  - Q2(문법) → Q3(작문)은 **영역 자체가 다름**(신규 주제)이라 상향 규정 비해당.

## 인증 원칙
모든 인증은 **이 GitHub 저장소의 파일 링크**로 검증 가능하게 한다.
- Q2: 해당 월 **월말누적평가** 채점 결과 파일 ([`grammar/tests/`](../grammar/tests/)).
- Q3: 해당 월 **누적 단어수·달성 여부** 기록 파일 (작문 원문은 [`writing/exercises/`](../writing/exercises/)에 보관).
