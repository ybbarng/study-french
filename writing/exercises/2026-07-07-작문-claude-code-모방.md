# 2026-07-07 작문 — Claude Code 예찬 (모델 모방) (36단어)

> 화요일 메뉴(모델 모방) · 의무 요소: ce qui + ça/il me permet de + 부정사 · 단어 수 집계: 예
> 모델: « Ce qui me plaît le plus dans la lecture, c'est qu'elle me permet de découvrir d'autres mondes. De plus, lire régulièrement enrichit le vocabulaire. »

## 원문 (사용자)

> Ce qui me plaît le plus dans le utilisation de Claude Code, c'est qu'il me permet de apprendre le français facile. De plus, exerciser régulièrement avec Claude Code fait des progrès de compétence pour le français.

## 교정본

> Ce qui me plaît le plus dans **l'utilisation** de Claude Code, c'est qu'il me permet **d'apprendre** le français **facilement**. De plus, **s'entraîner** régulièrement avec Claude Code **permet de faire des progrès en** français.

## 총평
- **한 줄 평**: 모델 틀(*Ce qui me plaît le plus dans X, c'est que… / me permet de…*)을 **정확히 모방** — 오늘 T1(ce qui)이 바로 살아났음. 최대 약점은 **모음 앞 엘리지옹**과 어휘(exerciser·facile).
- **수준**: 구성 **B1** / 어휘·표기 A2~B1.
- **차원별**:
  - **내용·과제수행**: 모델 틀로 Claude Code 예찬 ✅ (모방 과제 충족)
  - **구성·응집**: Ce qui…, c'est que / De plus ✅
  - **어휘**: exerciser(영어 혼용), de compétence pour(calque) ← 개선 영역
  - **문법**: 모음 앞 엘리지옹 누락(le utilisation, de apprendre), facile→facilement(부사)
  - **표기**: 대체로 정확

## 고칠 것 (❌ 틀림 / △ 덜 자연스러움)

**모음 앞 엘리지옹**
| 표시 | 원문 | 교정 | 설명 |
|---|---|---|---|
| ❌ | dans **le** utilisation | dans **l'**utilisation | le+모음→l' (+ utilisation 여성) |
| ❌ | permet **de** apprendre | permet **d'**apprendre | de+모음→d' |

**어휘**
| 표시 | 원문 | 교정 | 설명 |
|---|---|---|---|
| ❌ | exerciser | **s'entraîner / pratiquer** | 없는 말 (영어 exercise) |
| ❌ | le français **facile** | le français **facilement** | 동사 수식은 부사 |
| ❌ | des progrès **de compétence pour** | des progrès **en** français | faire des progrès **en** + 언어 |

**구조**
| 표시 | 원문 | 교정 |
|---|---|---|
| △ | exerciser… **fait** des progrès | s'entraîner… **permet de faire** des progrès (활동이 주어면 permet de) |

## chunk → Tier
- 🔴 **T1**: 모음 앞 엘리지옹 자동화 (l'/d'/qu'/j'/n'/s')
- 🟡 **T2**: facilement(부사)≠facile, s'entraîner/pratiquer(≠exerciser), faire des progrès en+언어, l'utilisation(여성) / (validated: ce qui↔ce que)
- ⚪ **T4**: —

## 참고
- *J'aime l'idée*는 밋밋 → **J'aime bien cette idée** / **J'aime ça**가 자연스러움. **aimer bien=좋아하다(like)** vs **aimer=사랑하다(love)** (사람엔 큰 차이).
