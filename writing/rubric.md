# 작문 채점표 (rubric) — DELF B1 PI 메인 + B2 확장

> `/write` 월말 평가의 채점 기준. **B1 도달을 정확히 측정**하면서, B1 만점 근처에선 **B2 전환 신호**가 작동하도록 B2 확장 descriptor를 병기.
> 준거참조(criterion-referenced): 항목별 절대 기준으로 채점. 매번 같은 기준점.

## 6항목 채점표 (25점)

### ① 과제 적합성 (respect de la consigne) — 4점
| 점수 | 기준 |
|------|------|
| 4 | 주제·형식·길이 모두 충족, 요구사항 완전 수행 |
| 3 | 대체로 충족, 사소한 누락 |
| 2 | 부분 충족, 일부 요구 미수행 또는 길이 미달 |
| 0\~1 | 주제 이탈 또는 형식 크게 어긋남 |
| **B2 신호** | 단순 의견 진술을 넘어 **입장을 논증**(반론 고려·근거 전개) |

### ② 응집성·정합성 (cohérence et cohésion) — 4점
| 점수 | 기준 |
|------|------|
| 4 | 단락 구성 명확, 연결사로 논리 흐름 매끄러움 |
| 3 | 흐름 있으나 연결사 단조(et/mais/donc 위주) |
| 2 | 문장 나열식, 연결 약함 |
| 0\~1 | 논리 단절, 응집 거의 없음 |
| **B2 신호** | néanmoins·par ailleurs·en revanche 등 **다양한 연결사 + 단락 간 논증 구조** |

### ③ 어휘 (lexique) — 6점
| 점수 | 기준 |
|------|------|
| 5\~6 | 일상+추상 주제에 충분히 풍부, 약간의 반복 허용 |
| 3\~4 | 기본 어휘로 과제 수행, 반복·부정확 일부 |
| 1\~2 | 제한적, 자주 막힘, 직역·영어 혼용 |
| 0 | 과제 수행 불가 수준 |
| **B2 신호** | **정교한 어휘·뉘앙스·관용 표현**, chunk 다양성 → B2 과제로 전환 검토 |

### ④ 문법 (morphosyntaxe) — 6점
| 점수 | 기준 |
|------|------|
| 5\~6 | 시제·법·일치 대체로 정확, 복문 구사 (관계절·접속법) |
| 3\~4 | 기본 구조 정확, 복잡 구조에서 오류 |
| 1\~2 | 잦은 기본 오류 (시제·일치·관사) |
| 0 | 문장 성립 어려움 |
| **B2 신호** | **수동태·간접화법·분사 압축·가정법**을 자연스럽게, 오류 드묾 → B2 |

### ⑤ 표기·철자 (orthographe) — 3점
| 점수 | 기준 |
|------|------|
| 3 | 철자·악센트 거의 정확 (의사소통 무방해 경미 오타 허용) |
| 2 | 일부 오류이나 이해 무방 |
| 0\~1 | 잦은 철자 오류로 가독성 저하 |
| **B2 신호** | 악센트·정서법 거의 완벽, 미묘한 동형이의 구분 |

### ⑥ 사회언어학적 적합성 (registre) — 2점
| 점수 | 기준 |
|------|------|
| 2 | 격식 수준 일관 (글의 장르에 맞는 문어체) |
| 1 | 격식 흔들림 (구어·문어 혼재) |
| 0 | 부적절 |
| **B2 신호** | **격식↔비격식 의식적 조절**, 독자·목적에 맞춘 어조 |

> 오타 채점은 [feedback_typo-scoring-policy] 적용: 정서법 오타·문맥 명백 실수는 면제, 의사소통 방해 오류만 감점.

## B1 → B2 전환 트리거 (천장 효과 방지)

> **3개월 연속 22/25 이상** 또는 **B2 신호가 4개 항목 이상에서 관찰**되면:
> → 다음 평가부터 **B2 과제(250단어 논증문) + B2 채점 기준**으로 전환.
> (B2 채점표는 이 문서의 "B2 신호" descriptor를 만점 기준으로 승격하여 사용 — 새로 안 만듦)

목표상 B2 전환은 **2027 Q1** 예상. Q3(7\~9월)는 B1 측정 유지.
