동사 굴절(conjugaison) 드릴 훈련 세션을 시작한다.

## 절차 (반드시 순서대로 수행)

### 1. 데이터 로드
- `grammar/drills/conjugaison.md` (마스터 테이블)를 읽는다.
- `grammar/drills/conjugaison-log.md` (세션 로그)를 읽는다.
- 마지막 드릴 세션 날짜, 취약 패턴, 각 동사×시제의 숙달 단계를 파악한다.

### 2. 문제 선별 (10~15문제)
아래 비율로 문제를 구성한다. 숙달도 단계와 정답률을 기준으로 선별한다.

| 비율 | 대상 | 설명 |
|------|------|------|
| 40% | 🔴 정답률 < 70% | 가장 취약한 동사×시제 조합 |
| 20% | 취약 패턴 | 로그에서 반복 오류로 기록된 인칭/패턴 |
| 20% | 🟡 정답률 70~90% | 아직 불안정한 항목 |
| 20% | ✅ 유지보수 + 신규 | 숙달 항목 유지 확인 + ⬜ 미드릴 항목 도입 |

#### 혼동 쌍(confusion pairs) 규칙
- Stage 2→3 승격 후보를 테스트할 때, 해당 동사와 혼동하기 쉬운 동사를 연달아 출제한다.
- 예: faire conditionnel 직후 aller conditionnel, pouvoir/vouloir/devoir 연속 비교

### 3. 드릴 진행
- **전체 프랑스어로 진행한다.**
- **한 문제씩** 출제하고, 사용자의 답을 확인한 후 다음으로 넘어간다.
- 문제 유형을 섞어서 출제한다:
  - **활용 빈칸 채우기** (주요 유형, 60%): "Conjuguez *faire* au conditionnel présent, 1re personne du pluriel."
  - **과거분사 직접 질문** (10%): "Donnez le participe passé de *mettre*." — 과거분사 형태만 단독으로 묻는다.
  - **오류 교정** (15%): "Corrigez : *Nous faisont le travail.*"
  - **시제 선택** (15%): 문맥을 주고 올바른 시제를 고르게 한다.
- 정답/오답 판정 후 간결한 피드백을 준다.
  - 오답 시: 올바른 형태를 알려주고, 규칙을 간략히 상기시킨다.
  - 정답 시: "Très bien !" 등 짧게 격려하고 바로 다음 문제로.

### 4. 숙달 단계 관리

#### 단계 정의
| 단계 | 이름 | 승격 조건 |
|------|------|-----------|
| ⬜ | 미드릴 | 아직 한 번도 출제되지 않음 |
| 1 | 학습 | 한 세션 내에서 3회 정답 (간격을 두고, 연속 아닌) |
| 2 | 안정 | 서로 다른 2개 세션에서 첫 시도 정답 (세션 간 1일+ 간격) |
| 3 | 숙달 | 혼동 쌍과 함께 출제하여 정답 |
| ✅ | 완료 | Stage 3 달성 후 유지보수 모드 |

#### 퇴행(regression) 규칙
- ✅ → 3: 유지보수 출제에서 오답
- 3 → 2: 혼동 쌍 출제에서 오답
- 2 → 1: 세션 첫 시도에서 오답
- 퇴행 후 재승격 시, 승격 조건을 1회 줄여서 적용 (예: Stage 1→2 재승격은 1개 세션 정답)

#### 주의사항
- **실제 출제하여 정답인 항목만 승격한다.** 미출제 항목은 절대 자동 승격하지 않는다.
- 인칭별 취약점을 로그에 기록한다 (예: "nous/ils 어미 혼동", "tu impératif -s 탈락 오류").

### 5. 세션 종료
- 사용자가 "stop", "끝", "fin" 등을 말하거나 모든 문제를 완료하면 세션을 종료한다.
- 종료 시 아래를 수행한다:

#### 5-1. 결과 리포트 출력
사용자에게 아래 형식의 결과를 보여준다:
```
📊 Résultat du drill — [날짜]
- Questions : X / Y correctes (Z%)
- Progressions : [승격된 항목 나열]
- Régressions : [퇴행한 항목 나열]
- Points faibles détectés : [취약 패턴]
```

#### 5-2. grammar/drills/conjugaison.md 업데이트
- 출제된 항목의 숙달 단계를 갱신한다.
- 정답/시도 횟수(정답률)를 갱신한다.
- 취약 패턴 목록을 갱신한다.
- 비고란에 특이사항을 기록한다.

#### 5-3. grammar/drills/conjugaison-log.md 업데이트
- 세션 엔트리를 추가한다:
  - 날짜
  - 출제 문제 목록 (동사, 시제, 인칭, 문제 유형)
  - 각 문제의 결과 (⭕/❌)
  - 감지된 취약 패턴
  - 단계 변동 내역

#### 5-4. 커밋
- `grammar/drills/conjugaison.md`와 `grammar/drills/conjugaison-log.md`를 스테이징한다.
- 커밋 메시지 형식: `드릴: 동사 굴절 - 간단 요약`
  - 예: `드릴: 동사 굴절 - futur simple 중심 12문제, 정답률 83%`
- `git push`까지 실행한다.
