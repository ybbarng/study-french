# S9. 소유 대명사 (Pronoms possessifs)

> *C'est **ton** livre? Non, **le mien** est sur la table.* — 같은 "내 것"을 가리키는데 왜 *mon livre*(소유형용사)에서 *le mien*(소유 대명사)으로 바뀌는가? 그리고 ***nôtre*** vs ***notre*** — 시르콩플렉스 하나 차이가 왜 결정적인가? *Je préfère **les miennes*** — 왜 *les miens*가 아니라 *les miennes*인가?

## 핵심 개념

**공리**: 소유 대명사는 **이미 알려진 명사를 반복하지 않기 위해** 소유형용사 + 명사를 통째로 대체하는 형태이다. 항상 ***정관사(le/la/les) + 소유형 어간***의 두 부분으로 구성되며, **가리키는 명사**의 성수에 일치한다.

| 형태 변환 |
|----------|
| ***mon* livre** (소유형용사 + 명사) → ***le mien*** (소유 대명사, 통째 대체) |
| ***ma* voiture** → ***la mienne*** |
| ***mes* livres** → ***les miens*** |
| ***mes* voitures** → ***les miennes*** |

핵심 차이점:
- **소유형용사** (2장 학습 완료): *mon, ton, son, notre, votre, leur* + 명사 — 명사를 수식
- **소유 대명사** (본 자료): *le mien, le tien, le sien...* — 명사 자체를 대체

영어 *mine, yours, his, hers* 등에 대응하지만, **프랑스어는 성수 4형태**로 분화한다.

## 상세 설명

### 1. 형태표 — 6 소유자 × 4 성수 = 24형

| 소유자 | 남단 (le ___) | 여단 (la ___) | 남복 (les ___) | 여복 (les ___) |
|--------|---------------|---------------|----------------|----------------|
| 1sg (나) | *le mien* | *la mienne* | *les miens* | *les miennes* |
| 2sg (너) | *le tien* | *la tienne* | *les tiens* | *les tiennes* |
| 3sg (그/그녀) | *le sien* | *la sienne* | *les siens* | *les siennes* |
| 1pl (우리) | ***le nôtre*** | ***la nôtre*** | ***les nôtres*** | ***les nôtres*** |
| 2pl (당신/너희) | ***le vôtre*** | ***la vôtre*** | ***les vôtres*** | ***les vôtres*** |
| 3pl (그들) | *le leur* | *la leur* | *les leurs* | *les leurs* |

#### 주의점 1 — 단/복수 소유자에 따른 패턴 차이

| 소유자 유형 | 패턴 | 예 |
|------------|------|------|
| 단수 소유자 (1sg/2sg/3sg) | 4형 모두 분화 | *le mien / la mienne / les miens / les miennes* |
| 1pl/2pl | 성 구분 없음, 수만 구분 (2형) | *le nôtre / la nôtre / les nôtres* |
| 3pl | 성 구분 없음, 수만 구분 (2형) | *le leur / la leur / les leurs* |

복수 소유자 (*nous, vous, ils/elles*)는 **여성형이 따로 없다**. *la nôtre*에서 *la*만 여성 표지.

#### 주의점 2 — ^ (시르콩플렉스) 표지가 결정적

***nôtre / vôtre*** (대명사, ^ 있음) vs ***notre / votre*** (형용사, ^ 없음) 구별이 핵심.

| 형태 | 품사 | 사용 | 예 |
|------|------|------|------|
| *notre* (^ 없음) | 소유**형용사** | + 명사 | *notre maison* (우리 집) |
| ***le nôtre*** (^ 있음) | 소유**대명사** | 명사 자리 대체 | *Cette maison est **la nôtre**.* (이 집은 우리 것이다) |
| *votre* (^ 없음) | 소유**형용사** | + 명사 | *votre voiture* (당신 차) |
| ***le vôtre*** (^ 있음) | 소유**대명사** | 명사 자리 대체 | *C'est **la vôtre**.* (당신 것이다) |

발음 차이도 있다:
- *notre* /nɔtʁ/ (열린 o), *votre* /vɔtʁ/
- *nôtre* /notʁ/ (닫힌 o), *vôtre* /votʁ/

악센트가 발음을 바꾼다. 쓸 때 ^ 누락하면 다른 단어가 된다.

### 2. 일치 규칙 — 가리키는 명사의 성수

소유 대명사는 **소유자의 성수가 아니라 가리키는 명사의 성수**에 일치한다. 영어 *his/her* 발상과 정반대.

| 상황 | 영어 | 프랑스어 | 일치 기준 |
|------|------|----------|----------|
| Marie의 차 (la voiture) | *her car* | *sa voiture* / *la sienne* | 차 (la, 여단) |
| Marie의 책 (le livre) | *her book* | *son livre* / *le sien* | 책 (le, 남단) |
| Pierre의 차 (la voiture) | *his car* | *sa voiture* / *la sienne* | 차 (la, 여단) |
| Pierre의 책 (le livre) | *his book* | *son livre* / *le sien* | 책 (le, 남단) |

소유자가 남자(Pierre)든 여자(Marie)든 상관없이, **차** = *la sienne*, **책** = *le sien*. 한국어 화자가 처음에 매우 헷갈리는 부분이지만 프랑스어 논리는 일관된다.

### 3. 정관사 축약 — à + le/les, de + le/les

소유 대명사는 정관사를 포함하므로, *à* 또는 *de* 뒤에서 정관사 축약 규칙이 그대로 적용된다.

| 전치사 | 축약 |
|--------|------|
| *à + le mien* | ***au mien*** |
| *à + la mienne* | *à la mienne* (축약 없음) |
| *à + les miens* | ***aux miens*** |
| *à + les miennes* | ***aux miennes*** |
| *de + le mien* | ***du mien*** |
| *de + la mienne* | *de la mienne* (축약 없음) |
| *de + les miens* | ***des miens*** |
| *de + les miennes* | ***des miennes*** |

예문:
- *Je pense **au mien**.* (내 것을 생각한다 — *au* = à + le)
- *Je parle ***du mien***.* (내 것에 대해 말한다 — *du* = de + le)
- *Il s'occupe ***des siennes***.* (그는 그의 것들[여복]을 돌본다 — *des* = de + les)

### 4. 사용 환경 — 명사 반복 회피

소유 대명사는 **이미 화제에 등장한 명사를 다시 가리킬 때** 사용. 같은 명사를 두 번 쓰는 어색함을 피한다.

| 명사 반복 (어색) | 소유 대명사 사용 (자연) |
|----------------|---------------------|
| *Mon livre est ici, ton livre est là.* | *Mon livre est ici, **le tien** est là.* |
| *Sa voiture est rouge, ma voiture est bleue.* | *Sa voiture est rouge, **la mienne** est bleue.* |
| *Nos enfants jouent avec leurs enfants.* | *Nos enfants jouent avec **les leurs**.* |

대비/비교 문맥에서 특히 자주 사용된다.

### 5. 특수 표현 — 우정/소속을 나타내는 관용구

소유 대명사는 일부 관용구에서 사람의 친구/가족 관계를 함축한다.

| 표현 | 의미 |
|------|------|
| ***les miens*** | 내 가족 / 내 사람들 |
| ***les tiens*** | 네 가족 |
| ***les siens*** | 그/그녀의 가족 |
| ***les nôtres*** | 우리 가족 / 우리 편 사람들 |
| ***À la tienne!*** / ***À la vôtre!*** | 건배! (= à ta santé / à votre santé의 단축형) |
| ***Y mettre du sien*** | 자기 노력을 보태다 |

## 비교: 한국어 / 영어 / 다른 로망스어

### 한국어와의 비교

한국어 "~ 것"이 프랑스어 소유 대명사에 대응한다.

| 한국어 | 프랑스어 |
|--------|---------|
| 내 것 (남단 책) | *le mien* |
| 내 것 (여단 차) | *la mienne* |
| 내 것들 (남복 책들) | *les miens* |
| 내 것들 (여복 차들) | *les miennes* |
| 네 것 | *le tien / la tienne* |
| 그/그녀의 것 | *le sien / la sienne* (성별과 무관) |
| 우리 것 | *le nôtre / la nôtre* |
| 그들의 것 | *le leur / la leur* |

핵심 차이: 한국어 "그의 것 / 그녀의 것"은 소유자 성별로 분화하지만, 프랑스어는 **소유 대상의 성수**로만 분화한다. *Marie의 책* = *le sien*, *Pierre의 책* = *le sien* (동일).

또한 한국어 "내 것"은 사물 한 종류면 단일 형태지만, 프랑스어는 같은 "내 것"이라도 가리키는 명사가 *livre*(남)이냐 *voiture*(여)이냐에 따라 *le mien / la mienne*으로 분기한다.

### 영어와의 비교

| 영어 | 프랑스어 | 차이 |
|------|---------|------|
| *mine* | *le mien / la mienne / les miens / les miennes* | 영어 1형 → 프랑스어 4형 |
| *yours (sg)* | *le tien / la tienne / les tiens / les tiennes* | 동일 |
| *his / hers* | *le sien / la sienne / les siens / les siennes* | 영어는 소유자 성, 프랑스어는 대상 성 |
| *ours* | *le nôtre / la nôtre / les nôtres* | (^ 표지) |
| *yours (pl)* | *le vôtre / la vôtre / les vôtres* | (^ 표지) |
| *theirs* | *le leur / la leur / les leurs* | (성 구분 없음) |

영어와의 결정적 차이:
1. **성수 일치**: 영어는 *mine* 단일 형태, 프랑스어는 4형 (*le mien / la mienne / les miens / les miennes*).
2. **소유자 성별 무시**: 영어 *his/hers*는 소유자 성별 표시. 프랑스어 *le sien*은 대상 성 표시뿐. *Marie의 책*도 *Pierre의 책*도 모두 *le sien*.
3. **정관사 동반**: 영어 *mine*은 단독, 프랑스어는 항상 *le/la/les* 동반.
4. **영어 *my own***: 강조형 (*my own car*)은 프랑스어 *ma propre voiture*로 별도 표현. *la mienne*과 다름.

### 스페인어/이탈리아어와의 비교

| 프랑스어 | 스페인어 | 이탈리아어 |
|----------|---------|------------|
| *le mien / la mienne* | *el mío / la mía* | *il mio / la mia* |
| *les miens / les miennes* | *los míos / las mías* | *i miei / le mie* |
| *le tien / la tienne* | *el tuyo / la tuya* | *il tuo / la tua* |
| *le sien / la sienne* | *el suyo / la suya* | *il suo / la sua* |
| *le nôtre / la nôtre* | *el nuestro / la nuestra* | *il nostro / la nostra* |
| *le leur / la leur* | *el suyo / la suya* | *il loro / la loro* |

세 로망스어 모두 **정관사 + 소유형**의 구조가 동일. 라틴어 *meus, tuus, suus, noster, vester*의 어간이 그대로 보존됐다. 이탈리아어 *il mio, il tuo*는 프랑스어 *le mien, le tien*과 1:1 평행.

차이점: **스페인어는 소유 대명사가 형용사와 형태가 같다** (*el mío* = 대명사 / 명사 뒤 강조형 형용사). 이탈리아어 *il loro*는 3pl에서 성/수 모두 불변. 프랑스어 *le leur / la leur / les leurs*는 수만 분화.

## 한국어 화자 주의점

1. ***nôtre / vôtre* (^) vs *notre / votre* (^ 없음) — 시르콩플렉스 누락은 다른 단어**: 두 글자 차이가 품사를 바꾼다.
   - *notre maison* (^ 없음) = 우리 집 (형용사 + 명사)
   - ***la nôtre*** (^ 있음) = 우리 것 (대명사 단독)
   - 발음도 다름: *notre* /nɔtʁ/ (열린 o), *nôtre* /notʁ/ (닫힌 o)
   쓸 때 절대 ^ 누락 금지. 대명사 쓰는데 *notre* 쓰면 문법 오류가 아니라 단어 오류.

2. **소유자 성별이 아니라 대상의 성수에 일치 — 영어 *his/hers* 발상 버리기**: 한국어/영어 화자가 가장 자주 실수하는 부분.
   - *Marie의 책* → *le sien* (livre 남단)
   - *Marie의 차* → *la sienne* (voiture 여단)
   - *Pierre의 책* → *le sien* (동일, 소유자 성별 무관)
   - *Pierre의 차* → *la sienne* (동일)
   "Marie니까 *la sienne*", "Pierre니까 *le sien*"이라는 발상은 **틀린다**. 항상 *가리키는 명사*가 무엇인지가 결정 기준.

3. **복수 소유자(1pl/2pl/3pl)는 여성형이 없다 — 형태 단순화**: 단수 소유자(1sg/2sg/3sg)는 4형(*le mien / la mienne / les miens / les miennes*)이지만, 복수 소유자는 2형만 (*le nôtre / la nôtre / les nôtres*).
   - 1pl: *le nôtre, la nôtre, les nôtres* (단/복만 구분)
   - 2pl: *le vôtre, la vôtre, les vôtres*
   - 3pl: *le leur, la leur, les leurs*
   *la nôtre*에서 여성은 정관사 *la*가 표시할 뿐, *nôtre* 자체는 남/여 동형. 단수 소유자처럼 *nôtres / nôtresse* 같은 구별 없음.

4. **정관사 축약(à + le → au, de + le → du)이 그대로 적용**: 소유 대명사는 정관사를 포함하므로 *à* 또는 *de* 뒤에서 축약된다.
   - *à + le mien* → ***au mien***
   - *à + les siens* → ***aux siens***
   - *de + le tien* → ***du tien***
   - *de + les leurs* → ***des leurs***
   *à le mien* (X), *de le tien* (X)는 비문. 정관사 축약 규칙(2장)을 그대로 적용해야 한다.

5. ***le sien* vs *son* — 같은 "그의/그녀의"인데 사용 환경 다름**: 둘 다 3sg 소유를 표시하지만 품사가 다르다.
   - *son livre* (소유형용사 + 명사) — 명사를 수식
   - *le sien* (소유 대명사, 단독) — 명사 자체를 대체
   - 두 형태를 한 문장에 같이 쓸 수 없음: *son le sien* (X) 비문.
   - 사용: *son livre*는 "그의 책"이라고 처음 도입할 때, *le sien*은 이미 *livre*가 화제에 등장한 뒤 반복 회피 시.

6. ***À la tienne! / À la vôtre!* — 관용 건배 표현**: 직역하면 "너의 것에 / 당신의 것에"이지만 실제는 "너의 건강을 위해 / 당신의 건강을 위해" (*à ta santé / à votre santé*의 단축형). *santé*(여)가 함축됐기 때문에 *la* 여성형. 한국어 "건배!" 또는 "위하여!"에 대응. 회식·축하 자리에서 반드시 등장하는 표현이다.

## 어원과 역사

소유 대명사는 라틴어의 강세형 소유사(*meus, tuus, suus, noster, vester*)에서 유래한다.

- ***mien*** < 고프랑스 *mien* < 라틴 *meum* (남단 대격). *mon*도 같은 어원이지만 약세 위치에서 단축됨. *mien*은 강세 위치 형태가 살아남은 결과.
- ***tien*** < 라틴 *tuum* > 고프랑스 *tuen* > *tien*. 모음 변화 *u* → *ie*.
- ***sien*** < 라틴 *suum* > 고프랑스 *suen* > *sien*. 형식상 *tien*과 평행 진화.
- ***nôtre / vôtre*** < 라틴 *nostrum / vestrum*의 강세형. **시르콩플렉스 ^는 사라진 s의 흔적**이다. 중세 프랑스어에서 *nostre, vostre*로 쓰였다가 17세기에 *s* 탈락하면서 ^로 표시. 같은 패턴: *forêt* < *forest*, *hôpital* < *hospital*, *île* < *isle*. *notre / votre* (형용사, 약세 위치)는 *s*가 더 일찍 탈락해 ^ 없음.
- ***leur*** < 라틴 *illorum* (그들의, 대명사 *ille*의 복수 속격). 다른 소유형과 어원 자체가 다르다 (대명사 속격에서 유래). 그래서 *leur*은 형용사/대명사 모두 같은 형태 (시르콩플렉스 없음).

**시르콩플렉스 = 사라진 s**라는 통찰은 *nôtre / vôtre*의 ^를 외울 때 결정적이다. 라틴 *nostrum / vestrum* → 중세 *nostre / vostre* → 현대 *nôtre / vôtre*. *s* 탈락 + ^로 표시 + 발음이 닫힌 o로 변화 (/nɔ/ → /no/).

흥미로운 점: **약세/강세 형태의 분화**는 라틴어에서 프랑스어로 넘어오면서 발생한 보편 패턴이다. *mon/mien*은 같은 *meum*에서 갈라진 쌍둥이다. 약세 위치(형용사)는 *mon*으로 단축, 강세 위치(대명사)는 *mien*으로 보존. 이 분화 패턴은 인칭대명사 *me/moi*, *te/toi*, *se/soi*에서도 동일하게 나타난다.

## 핵심 정리

```
소유 대명사 24형 마스터 표:

         │ 남단         │ 여단          │ 남복         │ 여복
─────────┼──────────────┼──────────────┼──────────────┼─────────────
 1sg     │ le mien     │ la mienne    │ les miens    │ les miennes
 2sg     │ le tien     │ la tienne    │ les tiens    │ les tiennes
 3sg     │ le sien     │ la sienne    │ les siens    │ les siennes
 1pl     │ le nôtre^   │ la nôtre^    │ les nôtres^  │ les nôtres^
 2pl     │ le vôtre^   │ la vôtre^    │ les vôtres^  │ les vôtres^
 3pl     │ le leur     │ la leur      │ les leurs    │ les leurs

^ = 시르콩플렉스 표지 (notre/votre 형용사와 구별)
```

핵심 규칙 3가지:

| 규칙 | 내용 |
|------|------|
| **일치 기준** | 가리키는 명사의 성수 (소유자 성별 X) |
| **시르콩플렉스** | *nôtre, vôtre* (대명사) ≠ *notre, votre* (형용사) |
| **정관사 축약** | *au mien, du sien, aux siens, des leurs* (à/de + le/les) |

핵심 대조 쌍:

| 대조 | 차이 |
|------|------|
| **소유형용사 vs 소유대명사** | *mon livre* (+ 명사) vs *le mien* (단독) |
| ***notre* vs *nôtre*** | 형용사 (^ 없음) vs 대명사 (^ 있음) — 단어 자체가 다름 |
| **단수 vs 복수 소유자** | 1sg/2sg/3sg = 4형 분화 / 1pl/2pl/3pl = 2형만 |
| **영어 *his/hers* vs 프랑스어 *le sien***  | 영어 = 소유자 성 / 프랑스어 = 대상 성 |

사용 환경:

```
이미 등장한 명사 반복 회피 → 소유 대명사
대비/비교 문맥 → 자연스러움 (Mon livre est ici, le tien est là)
관용 표현 → À la tienne! (건배), les miens (내 가족)
```

## 다음 단계

소유 대명사는 2장 소유형용사(*mon, ton, son, notre, votre, leur*)의 자연스러운 확장이다. 형용사가 명사를 수식하는 반면, 대명사는 명사 자체를 대체한다. 두 형태를 자유롭게 전환할 수 있어야 명사 반복을 피하는 자연스러운 프랑스어 작문이 가능해진다.

다음 학습은 **지시 대명사** (S10, *celui / celle / cela / ça*)로 이어진다. 지시 대명사도 소유 대명사와 마찬가지로 **명사 반복 회피** 기능을 하지만, "그/저것"이라는 지시 의미가 추가된다. 두 대명사 시스템을 함께 익히면 작문 표현력이 크게 확장된다.

S11(부정대명사 긍정 측, *tout / chacun / plusieurs*)과 함께 본 자료의 세 대명사 묶음(S9/S10/S11)은 14장 추상 대명사 및 담화 연결의 기초가 된다. 이미 학습한 1~12장의 명사·동사·관계대명사 시스템 위에 대명사 체계를 완성하는 단계로 이해하면 좋다.

특히 *à la tienne!* / *à la vôtre!* 같은 관용 건배 표현은 일상 회화에서 자주 등장하므로 따로 외워두면 회화의 자연스러움이 올라간다. 작문 연습(`/project:write`)에서도 두 명사를 비교하는 문장(*mon X est..., le tien est...*)이 등장하면 의식적으로 소유 대명사로 표현해 보는 훈련을 권장한다.
