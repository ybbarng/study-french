# S10. 지시 대명사 (Pronoms démonstratifs)

> *Le livre de Pierre est ici, **celui** de Marie est là* — *celui*가 무엇을 대신하는가? *J'aime **ce que** tu fais* vs *J'aime **celui que** tu fais* — *ce que*와 *celui que*는 어떻게 다른가? *Ceci est important*, *cela m'intéresse*, *ça va* — 세 형태(*ceci / cela / ça*)는 무엇이 같고 다른가? 본 자료는 프랑스어 지시 대명사 시스템을 두 축(**가변 / 불변**)으로 정리한다.

## 핵심 개념

**공리**: 프랑스어 지시 대명사는 두 종류로 양분된다.

| 종류 | 형태 | 가리키는 대상 |
|------|------|----------|
| **가변 (variable)** | *celui / celle / ceux / celles* | 구체적 명사 (성수 일치) |
| **불변 (invariable)** | *ce / ceci / cela / ça* | 추상적 개념, 문장, 상황 |

핵심 사용:
- **가변** — 이미 알려진 명사를 대체하면서 한정구(*de + 명사*, 관계절, *-ci/-là* 위치 표지)와 결합
- **불변** — 명사 없이 개념/사실/상황을 가리킴 (영어 *this / that / it*에 대응)

소유 대명사(S9)가 "소유 관계"로 명사를 대체했다면, 지시 대명사는 "지시 관계 + 한정"으로 명사를 대체한다.

## 상세 설명

### 1. 가변 지시 대명사 — celui / celle / ceux / celles

#### 형태표 (성수 4형)

| 성수 | 형태 | 발음 (IPA) |
|------|------|-----------|
| 남단 | ***celui*** | /sə.lɥi/ |
| 여단 | ***celle*** | /sɛl/ |
| 남복 | ***ceux*** | /sø/ |
| 여복 | ***celles*** | /sɛl/ |

여단 *celle*과 여복 *celles*는 발음 동일 (/sɛl/). 철자만 구별.

#### 사용 환경 — 단독으로 쓸 수 없음

가변 지시 대명사는 **단독으로 사용 불가**. 반드시 다음 세 가지 중 하나가 뒤따라야 한다.

| 후행 요소 | 의미 | 예 |
|----------|------|------|
| ***de + 명사*** | ~의 것 (소유/소속) | *celui de Pierre* (피에르의 것) |
| ***qui / que / dont + 관계절*** | ~한 것/사람 | *celui qui parle* (말하는 사람) |
| ***-ci / -là*** (접미사) | 이것 / 저것 (위치 대비) | *celui-ci / celui-là* |

단독 *celui*만 쓰면 비문: *J'aime celui.* (X). 한정 요소 없으면 어떤 명사를 가리키는지 명확하지 않다.

#### 사용 1 — celui de + 명사 (소유/소속)

명사를 반복하지 않고 **소유자/출처**만 바꾸어 표현.

| 명사 반복 (어색) | celui de 사용 (자연) |
|----------------|---------------------|
| *Le livre de Pierre et le livre de Marie* | *Le livre de Pierre et **celui** de Marie* |
| *La voiture de mon père et la voiture de ma mère* | *La voiture de mon père et **celle** de ma mère* |
| *Les idées de Sartre et les idées de Camus* | *Les idées de Sartre et **celles** de Camus* |

영어 *Pierre's*, *the one of Marie*에 해당. *celui de*는 한국어 "~의 것"과 매우 직관적으로 매핑된다.

#### 사용 2 — celui qui / que / dont + 관계절

명사를 관계절로 한정. 8장(관계대명사 기초) + 11장(관계대명사 심화)에서 익힌 *qui / que / dont* 그대로 적용.

| 관계대명사 | 역할 | 예 |
|-----------|------|------|
| *qui* | 관계절의 주어 | ***celui qui*** parle (말하는 사람) |
| *que* | 관계절의 직접목적어 | ***celui que*** je vois (내가 보는 사람) |
| *dont* | *de* 보어 (~에 대해) | ***celui dont*** je parle (내가 말하는 사람) |
| *avec qui* | 전치사 + 관계대명사 | ***celui avec qui*** je travaille (내가 함께 일하는 사람) |

예문:
- ***Celui qui*** veut, peut. (원하는 자, 할 수 있다 — 격언)
- ***Ceux que*** tu aimes (네가 사랑하는 사람들)
- ***Celles dont*** je t'ai parlé (내가 너에게 말한 그것들[여복])

사람/사물 모두 가능. 문맥에서 가리키는 명사가 명확해야 한다.

#### 사용 3 — celui-ci / celui-là (이것 / 저것)

위치/시간 대비를 위해 *-ci* (가까운 것, "이~") 또는 *-là* (먼 것, "저~") 접미사 사용. 4형 모두 결합 가능.

| 형태 | 의미 |
|------|------|
| *celui-ci / celui-là* | 남단 — 이것 / 저것 |
| *celle-ci / celle-là* | 여단 — 이것 / 저것 |
| *ceux-ci / ceux-là* | 남복 — 이것들 / 저것들 |
| *celles-ci / celles-là* | 여복 — 이것들 / 저것들 |

사용 환경:
1. **두 사물 대비**: *J'ai deux livres. **Celui-ci** est en français, **celui-là** est en anglais.* (이건 프랑스어, 저건 영어)
2. **문어체 "후자/전자"**: ***Celui-ci*** = 후자 (방금 언급), ***celui-là*** = 전자 (먼저 언급).
   - *Pierre et Marie sont venus. **Celui-ci** (= Marie, 방금 언급, 가까움) est partie tôt.*
   - 실제 사용에서 한국어 화자에게 헷갈리는 부분 (한국어와 정반대 직관일 수 있음).

### 2. 불변 지시 대명사 — ce / ceci / cela / ça

#### 형태 4종

| 형태 | 사용 환경 | 의미 |
|------|----------|------|
| ***ce*** | + être / + 관계절 (qui/que/dont) | 그것 / 그/그녀 / 그것이 / 그것이 ~한 것 |
| ***ceci*** | 단독 (격식) | 이것 |
| ***cela*** | 단독 (격식) | 그것 / 저것 |
| ***ça*** | 단독 (구어) | 그것 / 저것 (만능) |

#### 사용 1 — ce + 관계절 (ce qui / ce que / ce dont)

선행사 없이 추상적 "~한 것"을 가리킬 때 사용. 14장(추상 대명사)의 핵심 구조 예고.

| 관계대명사 | 역할 | 예 |
|-----------|------|------|
| ***ce qui*** | 관계절 주어 | ***Ce qui*** m'intéresse, c'est la musique. (나를 흥미롭게 하는 것은 음악이다) |
| ***ce que*** | 관계절 직접목적어 | *Je sais ***ce que*** tu veux.* (네가 원하는 것을 안다) |
| ***ce dont*** | *de* 보어 | *Voici ***ce dont*** j'ai besoin.* (여기 내가 필요한 것이 있다) |
| ***ce à quoi*** | 전치사 보어 | *C'est ***ce à quoi*** je pense.* (그것이 내가 생각하는 것이다) |

핵심: ***ce* + 관계절**은 **추상 개념** (생각, 사건, 사실)을 가리킨다. **명사가 명시되지 않은 "그것"**.

대조: *celui qui parle* (말하는 **사람**, 명사 함축) vs *ce qui m'intéresse* (나를 흥미롭게 하는 **것/일/사실**, 추상).

#### 사용 2 — c'est / ce sont (정체 확인 구문)

***c'est + 명사/형용사*** = "그것은 ~이다". 14-2에서 본격 학습. 핵심 발음/철자 패턴만 정리.

| 표현 | 사용 |
|------|------|
| *C'est un livre.* | 그것은 책이다. |
| *C'est intéressant.* | 흥미롭다. |
| ***Ce sont*** *des livres.* | 그것들은 책들이다. (복수 명사 앞, 격식) |
| ***C'est*** *des livres.* | (구어, *ce sont* 대신 자주 사용) |

*ce*는 *est* 앞에서 항상 ***c'***로 축약 (모음 충돌 회피).

#### 사용 3 — ceci / cela / ça (단독 사용)

세 형태 모두 "이것/그것/저것"을 가리키지만 격식과 위치 의미가 다르다.

| 형태 | 격식 | 위치 의미 | 사용 빈도 |
|------|------|----------|----------|
| ***ceci*** | 격식 / 문어 | 이것 (가까운 / 앞으로 말할 것) | 드뭄 |
| ***cela*** | 격식 / 문어 | 그것 (먼 / 방금 말한 것) | 중간 |
| ***ça*** | 구어 / 일상 | 만능 (이것/그것 모두) | 매우 빈번 |

비교:
- *Ceci* /sə.si/ ↔ *cela* /sə.la/ — 한 쌍의 대비 (이것 vs 저것). 문어체.
- *ça* /sa/ — *cela*의 구어 단축형. 일상 회화에서 *cela* 대신 거의 항상 *ça* 사용.

예문 비교:
| 문맥 | 표현 |
|------|------|
| 격식 문서 / 문학 | *Cela m'intéresse beaucoup.* |
| 일상 회화 | ***Ça*** *m'intéresse beaucoup.* |
| 격식 대비 (이것/그것) | ***Ceci*** *est ancien, **cela** est moderne.* |
| 격언 / 정형 표현 | ***Cela dit****, je suis d'accord.* (그렇게 말했지만) |

#### 사용 4 — ça 관용 표현 (구어 핵심)

*ça*는 프랑스 일상 회화의 최고 빈출 단어. 관용 표현 다수.

| 표현 | 의미 |
|------|------|
| ***Ça va?*** | 잘 지내? (인사 기본) |
| ***Ça va.*** | 잘 지내. / 괜찮아. |
| ***Ça marche.*** | 됐어. / 좋아. |
| ***Ça suffit!*** | 그만해! |
| ***Ça me plaît.*** | 마음에 들어. |
| ***Ça dépend.*** | 경우에 따라. |
| ***Comme ça*** | 이렇게 / 그냥 |
| ***C'est ça!*** | 바로 그거야! |

### 3. ce qui/que/dont vs celui qui/que/dont — 결정적 대조

가변/불변의 핵심 사용 차이.

| 표현 | 가리키는 대상 | 의미 |
|------|-------------|------|
| ***celui qui*** *parle* | 사람/사물 (구체) | 말하는 **사람** |
| ***celle qui*** *brille* | 여성 사람/사물 (구체) | 빛나는 **여자/것[여]** |
| ***ce qui*** *brille* | 추상 (것/사실/사물 일반) | 빛나는 **것** (구체적 무엇인지 미정) |
| ***ce que*** *tu fais* | 추상 (행위/일) | 네가 하는 **것** (행위 자체) |

격언 비교:
- ***Celui qui*** *veut, peut.* = 원하는 **사람**은 할 수 있다. (사람 = celui)
- ***Ce qui*** *brille n'est pas or.* = 빛나는 **것**이 모두 금은 아니다. (사물 일반 = ce qui)

선택 원칙:
- 구체적 명사가 문맥에 있고 그것을 가리키면 ***celui/celle/ceux/celles*** + 관계절
- 명사 없이 추상적 "것"을 표현하면 ***ce*** + 관계절

### 4. 가변 vs 불변 종합 비교

| 측면 | 가변 (celui...) | 불변 (ce / cela / ça) |
|------|----------------|---------------------|
| 성수 일치 | O (4형) | X (단일 형태) |
| 단독 사용 | X (반드시 한정 요소) | O (ceci/cela/ça) |
| 가리키는 대상 | 구체적 명사 | 추상 개념/문장/상황 |
| 한정 방식 | de + 명사 / 관계절 / -ci/-là | 단독 또는 + 관계절 |
| 영어 대응 | *the one of / the one who* | *this / that / it / what* |

## 비교: 한국어 / 영어 / 다른 로망스어

### 한국어와의 비교

한국어 지시어 체계와 프랑스어 매핑.

| 한국어 | 프랑스어 | 비고 |
|--------|---------|------|
| 이것 (가까운) | *ceci* / *celui-ci* / *ça* | *celui-ci*는 명사 함축, *ceci*는 추상 |
| 저것 (먼) | *cela* / *celui-là* / *ça* | *celui-là*는 명사 함축, *cela*는 추상 |
| ~의 것 | *celui de + N* | "피에르의 것" = *celui de Pierre* |
| ~한 사람 / ~한 것 (구체) | *celui qui / que* | 구체적 명사 가리킴 |
| ~한 것 (추상) | *ce qui / ce que* | 추상 개념 |
| 그/그것 (만능) | *ça* (구어) / *cela* (문어) | 일상은 거의 *ça* |

핵심 차이: 한국어 "것"은 구체/추상 구분 없이 단일 형태지만, 프랑스어는 **구체적 명사 가리킴 = celui계 / 추상 = ce계**로 분기한다.

| 한국어 | 프랑스어 (두 선택) | 차이 |
|--------|------------------|------|
| "네가 산 것이 좋다" | *J'aime **celui** que tu as acheté.* (구체) | 어떤 명사를 산 것인지 문맥에 있음 |
| "네가 산 것이 좋다" | *J'aime **ce que** tu as acheté.* (추상) | 무엇을 샀는지 미정, "네가 산 모든 것" |

문맥에서 어떤 명사인지 분명하면 *celui*, 미정/추상이면 *ce*.

### 영어와의 비교

| 영어 | 프랑스어 | 차이 |
|------|---------|------|
| *the one (of)* | *celui de / celle de* | 영어 *one*은 성 구별 없음, 프랑스어는 4형 |
| *the one who/that* | *celui qui / celui que* | 동일 구조 |
| *this one / that one* | *celui-ci / celui-là* | 영어 *one* + 프랑스어 *-ci/-là* |
| *what* (선행사 없는 관계절) | *ce qui / ce que* | 영어 *what*이 한 단어로 통합 |
| *this / that* (단독) | *ceci / cela / ça* | 영어 = 가까움/먼 2형 / 프랑스어 = 2~3형 + 격식 |
| *it / this* (정체 확인) | *c'est / ce sont* | 영어 *it is* + 프랑스어 *c'est* |

영어 *what*과 프랑스어 *ce qui / ce que*의 대응이 핵심:
- *I know **what** you want.* = *Je sais ***ce que*** tu veux.*
- ***What*** *interests me is music.* = ***Ce qui*** *m'intéresse, c'est la musique.*

영어 *what*은 주어/목적어 구분 없지만, 프랑스어는 *ce qui*(주어) / *ce que*(목적어)로 분기.

### 스페인어/이탈리아어와의 비교

| 프랑스어 | 스페인어 | 이탈리아어 |
|----------|---------|------------|
| *celui de Pierre* | *el de Pedro* | *quello di Piero* |
| *celle de Marie* | *la de María* | *quella di Maria* |
| *ceux qui parlent* | *los que hablan* | *quelli che parlano* |
| *celui-ci / celui-là* | *éste / aquél* | *questo / quello* |
| *ce qui / ce que* | *lo que* | *ciò che / quel che* |
| *ça / cela* | *eso / aquello* | *ciò* |

스페인어/이탈리아어는 **정관사 자체가 지시 대명사 역할**을 한다 (*el de Pedro*). 프랑스어는 별도 형태(*celui de*)를 발전시킨 차이.

이탈리아어 *quello / quella / quelli / quelle*는 프랑스어 *celui / celle / ceux / celles*와 1:1 평행. 같은 라틴어 *eccu(m) illu(m)* 어원에서 유래.

## 한국어 화자 주의점

1. ***celui*는 단독으로 절대 사용 불가 — 한정 요소 필수**: 한국어 "그것"은 단독으로 쓸 수 있지만, 프랑스어 *celui*는 반드시 후행 요소(*de + N*, 관계절, *-ci/-là*) 동반.
   - *J'aime celui.* (X — 비문)
   - *J'aime ***celui de Pierre***.* (O — 피에르 것)
   - *J'aime ***celui que*** tu as choisi.* (O — 네가 고른 것)
   - *J'aime ***celui-ci***.* (O — 이것)
   단독 "그것"을 표현하려면 *ça* (구어) 또는 *cela* (문어) 사용.

2. ***celui (가변) vs *ce* (불변) — 구체 명사 vs 추상 개념**: 둘 다 "~한 것"으로 번역되지만 사용 환경이 다르다.
   - ***celui qui*** *parle* = 말하는 **사람** (사람/사물 명사 함축)
   - ***ce qui*** *m'intéresse* = 나를 흥미롭게 하는 **것** (추상, 무엇인지 미정)
   - 문맥에 구체적 명사가 있으면 *celui* (그 명사 가리킴)
   - 명사 없이 "어떤 것/일/사실" 추상 표현이면 *ce*
   한국어 "것"은 양쪽 모두 가능하므로 직역으로 안 됨. **명사가 가리키는 게 명확한가?**가 판단 기준.

3. ***ça*는 일상 회화 만능, *cela*는 문어/격식**: 한국어 "그거"의 회화/격식 차이와 평행.
   - 일상 회화에서 *cela*는 거의 안 씀. *ça*가 압도적.
   - 시험/작문에서는 격식에 따라 *cela* 사용 권장.
   - *ceci*는 거의 사용 안 됨 (*ceci... cela*의 대비 또는 격식 문서에서만).
   - "잘 지내?" = *Ça va?* (***Cela va?*** 거의 안 씀, 매우 부자연).

4. ***c'est* vs *il est* — 정체 확인 vs 묘사**: 한국어 "그것은 ~이다"가 두 가지 프랑스어로 분기.
   - ***C'est*** *un livre.* (그것은 책이다 — 정체 확인, *un livre*가 새 정보)
   - ***Il est*** *grand.* (그는 키가 크다 — 묘사, 주어가 이미 알려진 사람/사물)
   - *c'est + 명사/형용사* (정체/평가) vs *il/elle est + 형용사* (묘사)
   - 직업: *Il est médecin.* (그는 의사다 — 직업 묘사) vs *C'est un médecin.* (그는 의사다 — 정체 소개)
   14-2에서 본격 학습. 일단 *c'est + 명사/형용사 평가*만 안전하게 사용.

5. ***celui-ci* (후자) vs *celui-là* (전자) — 한국어 직관과 정반대일 수 있음**: 문어체 "후자/전자" 표현은 한국어 화자가 가장 자주 헷갈리는 부분.
   - ***Celui-ci*** = 후자 (방금 언급, 가까움 → *-ci*)
   - ***Celui-là*** = 전자 (먼저 언급, 멀어짐 → *-là*)
   - 예: *Pierre et Marie sont venus. **Celui-ci** est partie tôt.* — 여기서 *celui-ci*는 마지막 언급된 Marie. 하지만 *celui-ci* 남단 → Pierre 가리킴 (성에 일치). 실제로는 *celle-ci*가 맞음.
   - 정확한 예: *Pierre et Marie sont venus. **Celui-ci** est resté.* = Marie(방금 언급, 남단 pas) → 사실 성 일치 문제로 사용 제한적.
   격언/관용 외에는 직접 사용 피하고 명시적 이름 반복 권장.

6. ***ce qui / ce que / ce dont* — 관계대명사 선택은 동사 지배**: *ce* + 관계대명사 선택은 관계절 내 동사의 지배 구조로 결정.
   - 동사의 **주어** 자리: *ce qui* (*Ce qui m'intéresse...* — *qui*가 *intéresse*의 주어)
   - 동사의 **직접목적어** 자리: *ce que* (*Ce que tu fais* — *que*가 *fais*의 직접목적어)
   - *de + 명사* 자리 (parler de, avoir besoin de): *ce dont* (*Ce dont tu parles* — parler **de**)
   - 그 외 전치사 + 명사 (à, sur 등): *ce à quoi, ce sur quoi* (*Ce à quoi je pense* — penser **à**)
   8장/11장 관계대명사 규칙 그대로 적용. *ce*만 추가.

## 어원과 역사

지시 대명사는 라틴어 *ecce ille* (보라, 저것)에서 유래.

- ***celui / celle / ceux / celles*** < 고프랑스 *cestui / cesti / ceuls / celes* < 라틴 *ecce + ille* (저것을 보라) → *eccillum > celui*. 어원적으로 "그+이/것"의 강조형.
- ***ce*** < 라틴 *ecce hoc* (이것을 보라) > 고프랑스 *ço* > *ce*. 약세 위치에서 단축됨.
- ***ceci*** = *ce* + *ici* (여기) = "여기 이것". 17세기 형성.
- ***cela*** = *ce* + *là* (저기) = "저기 그것". 17세기 형성.
- ***ça*** = *cela*의 구어 단축형. *e* 탈락 + *l* 약화. 18~19세기 일상 회화에서 정착.

흥미로운 패턴: ***ci / là* 접미사 = 위치 표지**가 지시 대명사 시스템 전체에 일관된다.
- *celui-ci* / *celui-là* (가변, 명사 가리킴)
- *ceci* / *cela* (불변, 단독)
- 같은 *-ci* (가까움) / *-là* (멈) 대비 구조

라틴어 *ecce* (보라!)가 프랑스어/이탈리아어/스페인어 지시 대명사 전체의 출발점이라는 점이 중요. 이탈리아어 *questo* < *ecce istu*, *quello* < *ecce illu*. 같은 *ecce* 어근에서 세 로망스어가 다른 방향으로 발달했다.

또 흥미로운 점: ***c'est*의 *ce* < *ecce hoc***. 즉 *c'est*는 어원적으로 "이것을 보라, 이것이 있다" → "그것은 ~이다"로 의미 추상화. 프랑스어가 *c'est*를 만능 정체 확인 구문으로 발전시킨 것은 라틴어 *ecce hoc est*에서 출발한 자연스러운 진화.

## 핵심 정리

```
지시 대명사 시스템 구조:

┌─────────────────────────────────────────────────────┐
│                  지시 대명사                          │
├─────────────────┬───────────────────────────────────┤
│   가변          │   불변                              │
│   (명사 대체)    │   (추상/사실/상황)                   │
├─────────────────┼───────────────────────────────────┤
│ celui / celle  │ ce + être / ce + 관계절             │
│ ceux / celles  │ ceci (가까움 / 격식)                 │
│                │ cela (먼 / 격식)                     │
│                │ ça (구어 만능)                       │
├─────────────────┼───────────────────────────────────┤
│ + de N         │ c'est / ce sont                    │
│ + qui/que/dont │ ce qui / ce que / ce dont           │
│ + -ci / -là    │ (단독 ceci/cela/ça)                 │

```

핵심 대조 쌍:

| 대조 | 차이 |
|------|------|
| ***celui qui*** vs ***ce qui*** | 구체 명사 (사람/사물) vs 추상 (것/사실) |
| ***celui de*** vs ***le sien*** (S9) | 한정구(de + N) vs 소유 표시 |
| ***celui-ci*** vs ***celui-là*** | 가까움/후자 vs 멈/전자 |
| ***ceci*** vs ***cela*** vs ***ça*** | 격식 가까움 / 격식 멈 / 구어 만능 |


## 다음 단계

지시 대명사는 S9 소유 대명사(*le mien*)와 함께 **명사 반복 회피**의 두 축을 이룬다. 두 시스템을 자유롭게 전환할 수 있어야 자연스러운 작문이 가능하다.

본 자료의 **ce qui / ce que / ce dont**은 14장 추상 대명사의 핵심 구조를 예고한다. 14-1에서 본격적으로 다룰 내용이지만, 격언/일상 표현 (*Ce qui brille n'est pas or*)에서 자주 등장하므로 미리 익혀두면 14장 진입이 매끄러워진다.

**c'est / ce sont** 정체 확인 구문은 14-2 (*c'est ... que/qui* 강조 구문)의 기초가 된다. 본 자료는 기본 사용만 다루고, 강조 구문의 본격 학습은 14-2에서 진행한다.

다음 학습은 **부정대명사 긍정 측** (S11, *tout / chacun / plusieurs / quelqu'un*)이다. S4(부정 측 *personne / rien / aucun*)와 짝을 이루어 프랑스어 부정대명사 체계 전체를 완성한다. 본 자료의 *ce qui / ce que* 구조는 *tout ce qui / tout ce que* (모든 것) 표현으로 S11에서 확장된다.

또한 일상 회화에서 *ça*가 차지하는 비중을 의식적으로 늘리는 훈련을 권장한다. *Ça va? / Ça marche! / Ça suffit!* 같은 관용 표현을 자유롭게 구사하는 것이 회화 자연스러움의 큰 부분이다.
