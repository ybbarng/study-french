## 0. 한 줄 결론

> **독학으로 막막하던 프랑스어 문법을, 한 단계도 건너뛰지 않고 체계적으로 완주했다.** 단순 "진도 완료"가 아니라, 규칙을 스스로 도출하고 시스템을 비판적으로 검증하며 내재화까지 끌고 간, 보기 드물게 능동적인 학습이었다.

## 2. 학습자 평가

### 2-1. 강점 ⭐

**① 메타인지 — 규칙을 스스로 도출한다 (가장 큰 강점)**
- 현재형 "boot pattern"을 직접 발견하고 접속법 이중 어간까지 연결
- 대조 표현 5형태를 영어와 1:1 매핑 도출 (mais=but / alors que=whereas / cependant=however / en revanche=on the other hand)
- "동사의 nous 현재형이 허브 — 현재분사·반과거·접속법이 다 거기서 나온다" 스스로 정리
- être 동사를 "왕래발착(세상에 옴/떠남)"으로 의미화하여 14개 암기 회피
- → 단순 암기가 아니라 **시스템을 재구성**하며 배운다. 이게 정착 속도와 깊이를 만들었다.

**② 언어학적·어원학적 깊이를 즐긴다 (개발자 사고)**
- "분사구문엔 mood 비트가 없다", "대과거는 과거의 과거를 백레퍼런스" 같은 비유로 추상 규칙을 자기 것으로 만듦
- 어원(campagne vs compagne, -uire→-uit) 토론을 동력으로 활용
- → 추상적 규칙일수록 오히려 강했다.

**③ 비판적 검증 — 시스템을 더 좋게 만든다**
- 출제 범위 위반(par erreur·alors que 미학습)을 정확히 지적 → 출제 원칙 확립
- 채점 과실(Q15/Q16을 "비문 아닌데 오답 처리") 재검토 요청 → 채점 정책 정비
- 용어 오류("vivre는 boot 아니라 단복 2분할") 교정
- → **가르치는 사람을 긴장시키는 학습자.** 본인 실력뿐 아니라 시스템 품질까지 끌어올렸다.

**④ 성실성·지속성**
- 100일간 거의 매일, 이사 기간에도 복귀
- 시험 미통과 시 회피 없이 재시험·드릴로 정면 돌파

**⑤ 정직함과 점진주의**
- "운이 좋았던 것 같다", "자신이 없다", "봐주지 마"처럼 자기 상태를 정직하게 보고
- 완벽주의보다 지속 가능성 우선 (분석 마비를 스스로 경계)

### 2-2. 약점 / 보완점

**① 자유 산출에서의 흔들림 (가장 중요)**
- 빈칸·판별 문제는 거의 만점인데, **자유 작문에서 동사 복합형태**가 흔들림 (être/avoir 조동사, 불규칙 과거분사, 가정법 시제)
- 분열문 인칭 일치(qui as/ai/avons)가 정착했다가 재발
- → 진단: "규칙 인지"는 완성됐으나 "산출 자동화"가 한 박자 늦다. **빈도(산출량) 부족**이 원인 — Q3 작문 단계가 정확히 이걸 메운다.

**② 며칠 공백 시 활용 흔들림**
- 6/18 복귀 시 lire(lit)·dormir(dort) 같은 기본 현재 활용이 잠깐 흔들림 (워밍업 5문제로 회복)
- → 규칙은 견고하나 굴절 자동화는 꾸준한 사용으로만 유지된다.

**③ 디테일 반복 실수**
- 엘리지옹(qu'elle, d'amis, j'avais) 자유 작문에서 누락 반복
- 철자(campagne≠compagne), 정서법 악센트
- 영어 단어 혼용(dangereux, voyager) — 직업·형용사에서 빈번
- → 치명적이진 않으나(의사소통 무방해) 꾸준한 모니터링 대상.

**④ 실력 대비 낮은 자신감**
- 19/20을 맞고도 "운이 좋았다", 모의 직전 "자신이 없다"
- → 데이터(꾸준한 95%+)가 보여주는 실력을 본인이 충분히 신뢰하지 못함. 산출 경험이 쌓이면 자연 해소될 영역.

### 2-3. 학습 스타일 요약

> **시스템 사고형 + 패턴 발견형.** 규칙을 통째로 받기보다 구조를 재구성하고, 비유(특히 개발/언어학)로 내재화하며, 검증을 통해 확신을 얻는다. 암기보다 일관 규칙을 선호하고, 깊이 파는 것을 즐기되 그것이 산출 부족으로 이어지지 않도록 경계가 필요하다.

## 3. 학습 인프라(시스템) 평가

### 3-1. 잘 작동한 것 ✅

| 시스템 | 효과 |
|--------|------|
| **간격 반복** (1→3→7→14→30일) | 망각 곡선 대응, 정착률 ↑. 5/22→5/24 +24%p가 대표 사례 |
| **4-tier 약점 관리** (T1 적극/T2 단기/T3 SRS/T4 참조) | "모든 오류 학습 ≠ 모든 오류 카드화" — 부담 조절하며 약점 32개 회복 |
| **누적 워밍업** | 매 세션 약점 재출제로 휘발 방지. 새 주제에 이전 구조 섞기 |
| **복습 우선 게이트** | 진도보다 내재화 우선 — 밀린 복습 시 복습 전용 전환 |
| **드릴 시스템** (26동사×7형태) | 굴절 자동화. boot/2분할 패턴 체득 |
| **난이도 체계** (L1~L5) + 오답 순환 | 기계적 적용→통합 응용까지 단계적 측정, 틀린 문제 다음 시험 이월 |
| **갈음 통과 정책** | 모의로 본시험 갈음 — 유연하면서 검증은 유지 |

### 3-2. 개선이 필요했던 것 / 문제점 ⚠️

**① 출제 범위 위반 (튜터 과실, 반복)**
- 안 가르친 표현을 시험에 냄: `par erreur`(관용구), `alors que/tandis que`(대조 접속사), 워밍업의 `manquer/le train`(미등록 어휘)
- → 학습자가 매번 지적해서 바로잡음. "출제 전 드릴·어휘 풀 확인" 원칙을 뒤늦게 확립 (개선 완료).

**② 채점 일관성 미흡**
- 오타 채점 정책(정서법·문맥 명백 실수 면제)을 늦게 확립
- Q15/Q16을 "비문이 아닌데 오답"으로 과하게 채점 → 학습자 이의로 정정
- → 채점 정책 문서화로 개선.

**③ 재료 제공 방식**
- 작문 재료에 전치사 결합을 안 줘서 어휘 지식을 추측하게 만든 사례(voyager dans le monde)
- → "재료는 collocation 완성형으로" 원칙 확립.

**④ 기록 형식 비일관**
- 시험 기록 헤더가 월마다 제각각(4월 불릿 / 5월 블록 / 6월 또 다름)
- → 6월을 5월형으로 통일, 향후 표준화 여지.

**⑤ 도구 오작동**
- task 추적 도구 오호출, /end 스킬 의도치 않은 실행 등 (학습 내용엔 무영향)

> **요약**: 시스템의 핵심 골격(간격 반복·4-tier·드릴·게이트)은 매우 효과적이었다. 약점은 주로 **튜터(Claude)의 출제·채점 운영**에 있었고, 학습자의 비판적 피드백 덕분에 그때그때 교정되며 오히려 시스템이 단단해졌다.

## 4. 인상적이었던 순간들

- **자가 도출의 연쇄**: boot pattern → 접속법 이중 어간 / 영어 매핑으로 대조 5형태 정리 / "nous가 허브" — 가르친 것 이상을 스스로 연결
- **5/22→5/24 대반전**: 모의 74%에서 약점 9개를 하루 만에 전부 회복해 본시험 98%
- **비판이 시스템을 고친 순간들**: 미학습 출제·과한 채점을 학습자가 잡아내며 출제·채점 원칙이 정립됨
- **정직한 자기 인식**: 통과해놓고 "운이 좋았다"·"봐주지 마" — 실력을 부풀리지 않는 태도

## 5. 교훈 / 다음 단계 제언

1. **문법은 수단이지 목적이 아니다.** 95%↔100%의 1%가 실전을 가르지 않는다. 진짜 실력은 다음 단계(산출)에서 자란다.
2. **분석 ↔ 산출 균형.** 인지·분석은 이미 강하다. 이제 무게중심을 **산출(작문·말하기)**로 옮길 때. 자유 산출 빈도가 늘면 동사 복합형태 흔들림·자신감 부족이 자연 해소된다.
3. **자동화는 빈도로만 유지된다.** 며칠 공백에도 흔들리지 않으려면 짧게라도 매일 굴리는 게 낫다 (워밍업의 가치).
4. **검증하는 학습자가 가장 빨리 큰다.** 받아 적기보다 따지고 도출하는 태도를 Q3에서도 유지할 것.

## 6. 총평

> 100일 전, "해리포터를 읽었지만 문법이 부족해서 체계적으로 다지고 싶다"는 막연한 갈증에서 출발했다. 그 갈증은 이제 **A1부터 C1+ 문어체까지 한 번 훑고, 단계별 평가 3개와 모의까지 통과한 토대**로 바뀌었다.
>
> 무엇보다 인상적인 것은 점수가 아니라 **태도**다. 규칙을 스스로 재구성하고, 시스템의 허점을 잡아내고, 자기 상태를 정직하게 보고하며, 완벽주의에 빠지지 않고 앞으로 나아가기로 결정하는 것 — 이건 가르쳐서 되는 게 아니다.
>
> 남은 약점(자유 산출 흔들림·자신감)은 결함이 아니라 **다음 단계의 입구**다. 입력(input)은 충분히 쌓았다. 이제 출력(output)으로 그것을 "말과 글이 되게" 할 차례다. 100일이 만든 토대 위에서, 비로소 프랑스어가 자기 표현이 되어 나올 것이다.
