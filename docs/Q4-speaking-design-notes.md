## 배경

- **Q4 (10~12월)**: 말하기 본격 — 버즈챌린지 양적 기준 + B1~B2 수준 도달 (DELF 응시는 선택)
- **9월**: 말하기 시스템 인프라 구축 (`francais-tres-facile` 확장 + speaking/ 디렉토리)
- **5~9월**: 사전 토론으로 점진적 다듬기

## 버즈챌린지 평가 기준 (확정)

> 월 RFI shadowing ≥ 20편 + 월말 자유 발화 평가 ≥ 70점
> (또는 단순 양적: 월 발화 녹음 누적 ≥ 30분)

## 5/1 토론 — 발음 학습 설계 ⭐

### 우선순위 결론 (학습 과학 + DELF 채점 기준 일치)

> **이해 가능성(Comprehensibility) > 자연스러움(Fluency) > 발음 정확성(Accuracy)**

근거:
- 응용 언어학 정설 (Munro & Derwing 2009 등)
- 외국어 학습자 목표 = "이해 가능한 외국인 억양", 아닌 "원어민 발음"
- 인지 과학: 음소 변별력은 만 6~12세 이전 결정 — 성인은 한계
- ROI: 자연스러움(chunk + intonation)은 1년 가능 / 원어민 발음은 5년+ 또는 영영 불가능

### DELF B1 구술 채점 가중치 (대략)

| 항목 | 가중 | 학습 우선 |
|------|------|----------|
| 과제 적합성 (내용) | ⭐⭐⭐⭐⭐ | ✅ 자연스러움 |
| 어휘 다양성 | ⭐⭐⭐⭐ | ✅ 자연스러움 |
| 유창성/응집성 | ⭐⭐⭐⭐ | ✅ 자연스러움 |
| 문법 정확성 | ⭐⭐⭐ | ✅ Q2 토대 |
| 발음 (이해 가능성) | ⭐⭐⭐ | ⚠️ 핵심 음소만 |
| 음성학적 정확성 (원어민 수준) | ⭐ | ❌ 무시 가능 |

### 발음 학습 2단계 (사용자 제안)

#### 1단계: 문자-음소 변환 훈련 ⭐⭐⭐

**왜 효율적인가**:
- 프랑스어 표기-발음 일관성 ~85% (영어 ~50%보다 훨씬 높음)
- **핵심 ~30개 패턴** 외우면 90%+ 단어 발음 자동 도출
- DELF 준비 학생들의 표준 첫 단계

**핵심 패턴 (예시)**:

| 표기 | 음소 | 예 |
|------|------|-----|
| **oi** | /wa/ | moi, voir |
| **ou** | /u/ | nous, où |
| **u** (단독) | /y/ | tu, su |
| **eau / au** | /o/ | beau, eau, jaune |
| **ai / ei** | /ɛ/ | mais, peine |
| **eu / œu** | /ø/ /œ/ | peu, sœur |
| **an / am / en / em** | /ɑ̃/ | enfant, temps |
| **on / om** | /ɔ̃/ | bon, nom |
| **in / im / ain / ein** | /ɛ̃/ | vin, pain, plein |
| **ille** | /ij/ | fille, billet |
| **gn** | /ɲ/ | montagne |
| **ph** | /f/ | photo |
| **ch** | /ʃ/ | chat |
| **qu** | /k/ | qui, quand |
| **c+e/i** vs **c+a/o/u** | /s/ vs /k/ | ciel vs cas |
| **g+e/i** vs **g+a/o/u** | /ʒ/ vs /g/ | gendre vs gare |
| **묵음 자음** | (sound 없음) | h, t/d/s 단어 끝, e muet |

→ 약 **30~40개 패턴**이면 거의 모든 일상 단어 발음 자동 도출.

#### 2단계: Liaison ⭐⭐⭐⭐ (자연스러움 핵심)

**한국어 화자가 가장 어색하게 들리는 지점**. liaison 빠뜨리면 즉시 "외국인 발음" 인식.

| 종류 | 예 | 규칙 |
|------|-----|------|
| **의무 liaison** ⭐ | les_amis, mon_ami, ils_ont, très_intéressant | 관사+명사, 소유형용사+명사, 대명사+동사, 부사+형용사 |
| **금지 liaison** ⚠️ | et / aspirated h / 단수 명사+형용사 | et_안 함, les#héros, étudiant#intelligent |
| **선택적 liaison** | je suis_américain | 격식체에서만 |

→ **약 7~10개 규칙**으로 90% 커버.

### 보완 의견 (5/1 추가)

**추가로 의식할 항목 1~2개**:

| 항목 | 이유 |
|------|------|
| **단어 끝 강세 (악센트 위치)** | 프랑스어는 단어 끝 음절 강세. 한국어는 의미 강세 → 한국어식 강세는 불자연스러움 |
| **억양 (intonation)** | 의문문 (상승), 평서문 (하강), 강조 — 단어 발음 맞아도 억양 어색하면 외국인 |

→ 사용자 안 (문자-음소 + liaison) + 위 2개 = **발음 학습 완성도 ↑**.

### 무시 가능한 항목

- 원어민 수준 /R/, /y/, 비음 정확도
- 음소 변별 미세 차이
- 모음 길이 정밀 조정
- 지방 억양 (parisien vs sud)

→ shadowing으로 자연 흡수만 충분. 의식 학습 X.

## 5/1 토론 — 발화 학습 3단계 (사용자 제안) ⭐

### 단계별 설계

| 단계 | 활동 | 자유도 | 인지 부하 | DELF 환산 |
|------|------|--------|----------|-----------|
| **초반** | shadowing + 한→불 발화 번역 | 0~⭐ | 낮음 | A1~A2 |
| **중반** | 질문/대답 대화 | ⭐⭐ | 중 | A2~B1 |
| **후반** | 자유 발표 | ⭐⭐⭐⭐ | 높음 | B1+ |

### 평가 — SLA 정설과 일치

- Krashen의 i+1 (한 단계씩 자유도 확장)
- Vygotsky의 ZPD (점진적 비계 제거)
- Q3 작문에서 채택한 controlled-to-free와 같은 원리 → 일관성 ⭐

### 단계별 핵심 활동

#### 초반 (10~11월)

**A. Shadowing**:
- 원본 음성 따라 말하기
- 발음 + 억양 + 속도 + 듣기 동시
- 자료: RFI Français Facile (A2, 121편 보유 — `francais-tres-facile` DB)

**B. 한→불 발화 번역** (Q3 작문에서 검증된 방법의 발화 버전)
- 한국어 문장 → 즉시 프랑스어 발화
- 시간 한정 (5~10초 내) → 즉흥성 강제
- 직역 함정 직접 노출 + 자동화 강제
- Q3에서 만든 자기 코퍼스 활용 가능

#### 중반 (11~12월)

**시나리오 역할극** (`/speak`):
- Claude가 식당 직원 / 행인 / 카페 주문 등 역할
- 사용자 즉흥 응답
- 발음 + 자연스러움 + 응집성 평가
- DELF B1 구술 part 2와 유사

#### 후반 (12월)

**자유 발표**:
- 주어진 주제로 3분 모놀로그
- DELF B1 구술 part 3과 일치
- 자기 의견 + 근거 + 결론 구조
- 12월 후반 DELF 모의 시뮬레이션 2~3회

## Register 분리 전략 (5/2 추가) ⭐

Q3 학습한 chunk가 Q4에 자동 이어짐 + 구어 전용 chunk 추가 학습.

- **문어 chunk** (Q3 메인) ~250개
- **구어 chunk** (Q4 메인) ~250개 — *du coup, ouais, t'inquiète, en gros, tu vois* 등
- **공유 chunk** ~400개 — Q3에서 학습 → Q4 자동 베이스

→ Q4 새로 학습할 chunk = **구어 250개만** (부담 ↓)

상세: **`docs/chunk-register-strategy.md`** 참조 (향후 발전 가능성 포함)

### 한국어 화자 register 함정 (Q4 특화)

- 회화에 격식체 과다 (*par conséquent* 친구에게 → 어색)
- 구어 chunk 자체 모름 (*du coup, en gros, t'inquiète*) → "교과서 프랑스어" 인식
- 인사 register 혼동 (*Salut!* vs *Bonjour Madame*)

→ Q4 학습 핵심 = "교과서 프랑스어" 탈출 = 구어 chunk 250개 + 자연스러운 발음.
