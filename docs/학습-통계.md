## 7. 흥미로운 파생 지표

- **출석률 67%** — 매일 강제 없이도 사흘 중 이틀
- **시험 28회 / 학습 68일** — 학습의 40%가 "측정과 함께" 진행
- **재시험 10회** — 미통과를 다음으로 미루지 않고 즉시 재도전
- **약점 50개 해소** — 단순 진도가 아니라 "틀린 것을 끝까지 추적"
- **기록 11,000줄** — 학습 과정 전체가 검색 가능한 자산으로 남음
