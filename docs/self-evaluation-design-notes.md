# 자가 평가 설계 노트

> Q3 작문 + Q4 말하기 양쪽에 적용되는 자가 평가 방법론. 6~9월 추가 토론으로 디테일 결정.

## 자가 평가의 목적

1. **자기 약점 진단** — 학습 우선순위 결정
2. **Self-revision** — 작문/발화 후 본인 검토 (시험에서 본인 검토 능력)
3. **진척률 측정** — 외부 평가(시험) 없이 자체 측정
4. **메타인지 능력** — 자기 능력 정확히 인식 (B2+ 핵심)
5. **자가 학습 능력** — 평생 학습 / 다국어 확장 시 결정적

→ 사용자가 DELF 응시 안 할 수도 있음 (5/2 명시) → **자가 평가가 더 중요**.

## 우리 시스템 적용 시점

| 시점 | 자가 평가 강도 | 활동 |
|------|--------------|------|
| Q3 베이스라인 (6월) | ⭐⭐⭐⭐⭐ | chunk 풀 자가 분류 (능동/인지/모름) |
| Q3 Self-revision (매일) | ⭐⭐⭐ | 작문 첨삭 전 1분 검토 |
| Q3 매월 평가 | ⭐⭐ | DELF 점수 + 자가 점수 |
| Q4 발화 평가 | ⭐⭐⭐⭐ | 시나리오 능숙도 0~5 |
| 시험 응시 X 시 | ⭐⭐⭐⭐⭐ | 자체 진척 측정 |

## 미결 사항 (시점별)

### 6월 결정

#### 1. Self-revision 형식
- 체크리스트 vs 자유 검토 vs 하이브리드
- 시간: 1분 / 2분 / 5분
- 검토 항목: 시제 / 관사 / 응집 / chunk 사용 등

#### 2. 점수 척도 (rubric)
- 점수 체계: 0~5 / 0~10 / DELF 25점 6항목
- 활동별 통일 vs 다름
- 각 단계 정의 명확화

#### 3. 데이터 누적 / 시간 추적
- 저장 위치 (`writing/evaluations/YYYY-MM-DD.md`?)
- 자가-Claude 점수 차이 추적
- 시각화 방법

### 6~7월 결정

#### 4. 자가 vs Claude 점수 통합 방식

| 옵션 | 설명 |
|------|------|
| A. 평균 | (자가 + Claude) / 2 |
| B. 보수적 | min(자가, Claude) |
| C. 가중치 변동 | 시간에 따라 자가 비중 ↑ |
| D. 둘 다 기록 + 일치율 추적 | 메타 평가 가능 |

### 8~9월 결정 (Q4 직전)

#### 5. Q4 시나리오 5점 척도 정의
- 5/1 임시 정의 → 객관 메트릭 추가 (발화 속도, 망설임 빈도)

#### 6. 녹음 셀프 리뷰
- 즉시 평가 vs 시간차 평가
- 녹음 듣기 시간 포함 여부

#### 7. 자가 평가 정확성 향상 메커니즘
- 모범 답안 비교 / 시간차 검토 / 메타 평가

## 자가 평가의 한계 (인식)

- 관대 경향 (본인 글이 더 좋아 보임)
- 본인이 모르는 약점 (unknown unknowns)
- 컨디션에 따른 변동
- 메타인지 능력 자체가 학습 대상

→ **3주체 종합 평가** 권장: Claude (텍스트 분석) + 자가 (자연스러움/유창성) + 객관 도구 (Whisper diff).

## 한 줄 요약

> **자가 평가 = 학습자 독립성 키우는 도구**.
> 시험 응시 안 할 시 진척 측정의 메인.
> 단, 자가만 의존 X — Claude + 객관 도구와 종합.
