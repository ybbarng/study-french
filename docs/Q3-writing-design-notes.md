> 6월 인프라 구축 시 참고. Q3(7~9월) 쓰기 훈련 본격 시작 전에 누적되는 토론/시범 결과 기록.

## 배경

- **Q2 (4~6월)**: 문법 공부 (현재 진행)
- **Q3 (7~9월)**: 쓰기 본격 — 버즈챌린지 양적 기준 + B1~B2 수준 도달 (DELF 응시는 선택)
- **6월**: 쓰기 시스템 인프라 구축 (writing/ 디렉토리, /write 강화)
- **5~6월**: 사전 토론으로 점진적 다듬기

## 시범 결과 — 2026-05-01

사용자 제안 ("키워드/문장 prompts → 사용자가 문장 만들기 + 점진적 확장")을 기반으로 4-tier 구조 설계 + 1단계만 시범 진행.

### 4-tier 구조 (검증된 흐름)

| Tier | 활동 | 시간 | 효과 |
|------|------|------|------|
| 1 | 키워드 5개 → 5문장 (사용자식 워밍업) | 5분 | 함정 진단 + 의식화 |
| 2 | 한국어 3문장 → 프랑스어 번역 | 5~10분 | 직역 함정 직접 노출 |
| 3 | 모델 문장 → 슬롯 채우기 (모방) | 5~10분 | 자연스러운 패턴 흡수 |
| 4 | 짧은 자유 작문 + 첨삭 | 10~15분 | 통합 산출 + 자연스러움 첨삭 |
| 주 1회 | 역번역 (좋은 문장 → 한국어 → 프랑스어 재작성) | 15분 | 메타 인지 폭발 |

**총 약 30~40분/일** (사용자 가용 시간 평일 1시간에 맞음)

### 1단계 시범 결과 (5/1)

**시나리오**: "오랜만에 친구를 만나 카페에서 이야기"
**키워드 5개**: retrouver, terrasse, raconter, depuis, fou rire

5문장 산출에서 **13가지 한국어 화자 함정 패턴 노출** — 진단 도구로 매우 효과적임 확인.

### 발견된 사용자 함정 패턴 (13개)

| # | 함정 패턴 | 회피 방법 | 비고 |
|---|----------|----------|------|
| 1 | donner qch (수령자 빠짐) | donner qch à qqn | 한국어 "주다" 직역 |
| 2 | à la terrasse | **en terrasse** (관사 없는 chunk) | ⭐ 큰 수확 |
| 3 | pendant + 동사원형 | pendant + 명사 / en + gérondif | 강한 함정 |
| 4 | sa + 모음 명사 | son + 모음 명사 | 발음 hiatus 회피 |
| 5 | raconter qch (수령자 빠짐) | raconter qch à qqn | 패턴 1과 동일 |
| 6 | l'histoire de... (풀어쓰기) | 명사 직접 목적어 | "X의 이야기" → "X" |
| 7 | téa, hour (영어 단어 혼용) | thé, heure | 메모리 등록 패턴 재발 |
| 8 | sortir (음식이 나오다) | arriver / être servi | 동사 선택 |
| 9 | de nos ordre (영어식 직역) | depuis qu'on a commandé | 영어 거쳐 학습한 흔적 |
| 10 | depuis vs il y a 혼동 | depuis=지속, il y a=시점 | 시간 표현 핵심 |
| 11 | character (영어) | personnage | 7번 패턴 재발 |
| **12** | **donner (호의 맥락)** ⭐⭐ | **offrir** | **한 세션 두 번 등장** |
| 13 | qui est écrit (관계절 구조) | avec X écrit dessus | 관계절 어색 |

### 가장 큰 자연스러움 수확 3가지

1. **en terrasse** (chunk 통째 암기) — 카페/식당 맥락 정형
2. ***Ça fait + 기간 + que + 절*** — "~한 지 ~이 됐다", 회화 정형 표현 핵심
3. **donner → offrir** (호의/선물 맥락) — 사용자 가장 약한 패턴 (한 세션 두 번 등장)

## 자연스러움 학습의 5가지 핵심 메커니즘

(Q3 인프라에서 활용)

### 1. Collocation (단어 결합 chunk)
프랑스어는 단어보다 단어 짝이 굳어 있음. 한국어 직역하면 다 어색.
- prendre une décision (결정을 내리다)
- faire la queue (줄을 서다)
- prendre une photo (사진을 찍다)
- passer un examen (시험을 보다)

### 2. Formules (정형 표현)
글의 뼈대 표현. DELF 작문/구술 채점관이 직접 카운트.
- "Pour ma part, je pense que..." (의견)
- "Cela dit, ..." (양보)
- "En revanche, ..." (대조)
- "En somme, ..." (요약)
- "Il me semble que..." (인상)
- "Ce qui m'intéresse, c'est..." (강조)

### 3. Reverse translation (역번역)
좋은 프랑스어 문장 → 한국어 옮김 (며칠 보관) → 한국어만 보고 프랑스어 재작성 → 원문 비교.
**메타 인지 폭발**. 자기 한국어 사고 패턴이 어디서 어색해지는지 자가 발견.

### 4. Model imitation (구조 모방)
좋은 단락 → 구조만 빌려와서 자기 내용 작성.
> 모델: *Bien qu'il soit fatigué, il continue à travailler. Cela montre qu'il est très déterminé.*
> 모방: *Bien que ___, ___. Cela montre que ___.*

자연스러운 패턴 내재화. **HP 자산 활용 가능** (이미 비교 독서 했음).

### 5. 한국어 화자 함정 카드 (contrastive cliché)
한국어 화자가 자주 잘못 쓰는 패턴 의식화. 누적 자료화.

## 사용자 한국어 화자 전형 약점 (메모리 등록 + 5/1 시범 보강)

이전 메모리 5개:
1. 관사 누락/과다
2. 소유 표현 (de 구조 vs 한국어 X의 Y)
3. 시제 일치 부족
4. 연결 표현 부족 (donc, car, alors que 등)
5. 어순 직역

5/1 시범 추가 발견:
6. **수령자 명시 누락** (donner/raconter 등 → à qqn 빠뜨림)

11. **관계절 어색 구조** (qui est écrit → avec X écrit dessus 등)

## 부담 조절 권장 ⭐⭐⭐ (5/1 토론 핵심)

### 문제 인식 (사용자 우려, 정확함)

5/1 시범 1단계만 했음에도 13개 함정 노출 → **모두 체득은 불가능**.
- 학습 과학: 한 번에 새 의식 항목 한계 = **5±2개 (Miller)**
- 노출 ≠ 체득
- 누적되면 따라가지 못함

### 해결 — "오늘의 1~2 함정" 4-Tier 시스템

핵심 원칙: **모든 오류 카드화 ≠ 모든 오류 의식 학습**

| Tier | 분류 | 처리 | 부담 |
|------|------|------|------|
| **T1 (적극 학습)** | 오늘의 핵심 1~2개 | 의식적 반복 + 다음날 워밍업 출제 | 낮음 |
| **T2 (단기 복습)** | 최근 1주 학습한 5~10개 | 워밍업 회전 출제 | 중 |
| **T3 (간격 반복)** | 정착 후 SRS (1→3→7→14→30일) | 자동 스케줄 | 낮음 |
| **T4 (참조 보관)** | 단발성/저우선순위 | 카드만 보관, 능동 학습 X | 0 |

→ 매일 작문에서 N개 나와도 **T1으로 올릴 건 1~2개만 선택**. 나머지는 카드만 누적.

### T1 우선순위 결정 기준 (ROI)

| 우선순위 | 기준 |
|---------|------|
| ⭐⭐⭐ T1 후보 | 같은 세션 2번+ 등장 / 매일 쓰이는 정형 chunk / DELF 채점표 직격 |
| ⭐⭐ T2 후보 | 자주 쓰이지만 한 번에 외우기 부담 / 시간 지나면 자연 정착 가능 |
| ⭐ T4 (참조만) | 단발성 어휘 오류 / 향후 학습으로 자동 해소 예정 |

### 5/1 시범 13개 함정의 Tier 분류 (예시)

| 패턴 | Tier | 이유 |
|------|------|------|
| donner → offrir (호의) | ⭐⭐⭐ T1 | 한 세션 두 번 등장 + 매일 쓰임 |
| en terrasse 등 chunk | ⭐⭐⭐ T1 | 매일 정형 |
| Ça fait + 기간 + que | ⭐⭐ T2 | 유용하지만 한 번에 외우기 부담 |
| 수령자 명시 (à qqn) | ⭐⭐ T2 | 패턴 인식 시간 필요 |
| depuis vs il y a | ⭐⭐ T2 | 시간 지나면 자연 정착 |
| character (영어 혼용) | ⭐ T4 | 단발성 |
| qui est écrit (관계절 구조) | ⭐ T4 | 11장 학습 시 자동 해소 예정 |

→ 5/1 시범 결과 **T1으로 올릴 건 2개만**: donner→offrir + en terrasse chunk.

### 분량 자체 조절

| 옵션 | 작문량 | 노출 함정 | 의식 학습 (T1) | ROI |
|------|--------|----------|---------------|-----|
| 풀 | 키워드 5+자유 5문장 | ~10~15 | 1~2 | △ |
| **표준** ⭐ | **키워드 3+자유 3문장** | **~5~8** | **1~2** | **○** |
| 미니 | 키워드 2 | ~2~3 | 1 | △ (양 부족) |

→ **표준 옵션** (총 30분/일). 평일 가용 1시간 안에서 작문+첨삭+다음날 워밍업까지 소화.

### "한 주 동안 익히기"의 효율 (사용자 우려 해소)

사용자 직관: "월요일 새 문장, 한 주 익힘 = 너무 오래" — 일리 있지만, 실제로는 **누적 + 회전**으로 해소:

- 월: T1 새 1~2개
- 화: 월 T1 출제 (워밍업) + T1 새 1개
- 수: 월/화 T1 출제 + T1 새 1개 (정착되면 T2로 이동)
- 목: T2 회전 + T1 새 1개
- ...
- 정착된 항목은 T3로 이동, 빈자리에 새 항목

→ 같은 항목을 일주일 내내 학습 X. **매일 1~2개씩 들어오고 나가는 흐름**.

### 권장 일정 (수정안)

- **평일** (30~40분): 키워드 3+자유 3문장 작문 (15분) + 첨삭 (10분) + 함정 카드 정리 (5분) + 다음날 T1 선정 (5분)
- **워밍업** (다음날 5분): 어제 T1 출제 + T2 회전 1개
- **주말** (1시간): 모방 작문 + 자유 작문 (긴 글, 4-tier 통합)
- **주 1회**: 역번역 1문단 (별도)

## 월말 평가 시스템 ⭐⭐⭐ (5/1 토론)

### 사용자 우려 (정확함)

1. "주제 다르고 길이 다른 글로 매월 평가하면 점수 향상이 능력 향상을 정확히 반영하나?"
2. "7월 첫 평가 — 무조건 통과? 기준 있어야 하나? 얼마?"

### 신뢰도 문제 — 표준화 프로토콜로 해결

**4가지 변수 고정 → 능력만 변동**:

| 항목 | 고정 |
|------|------|
| 글 형식 | 의견글 (B1 PI 표준) — 매월 동일 |
| 길이 | 160~180 단어 (B1 PI 표준) — 매월 동일 |
| 시간 한정 | 45분 (B1 PI 표준) — 매월 동일 |
| 채점표 | DELF B1 PI 공식 채점표 (25점 만점) — 매번 동일 |
| 채점 가이드 | `writing/rubric.md` 매번 참조 |
| 주제 | (변동) B1 일상 주제 풀에서 무작위 |

→ 변하는 것은 **주제뿐**. 나머지는 시험 환경 = 고정.

### DELF B1 PI 공식 채점표 (25점 만점)

| 항목 | 점수 |
|------|------|
| 과제 적합성 (respect de la consigne) | 4점 |
| 응집성/정합성 (cohérence et cohésion) | 4점 |
| 어휘 (lexique) | 6점 |
| 문법 (morphosyntaxe) | 6점 |
| 음운/표기 (phonologie/orthographe) | 3점 |
| 글쓰기 정확성 (sociolinguistique) | 2점 |

→ 항목별 점수 따로 기록 → 어디서 향상/퇴보했는지 시각화.

### 채점 일관성 보강

## Q3 메인 목표 = chunk 체득 (5/1 사용자 결정) ⭐⭐⭐

### 결정

> **궁극 목표**: B1 쓰기 (B2 발전 가능 수준)
> **실전 훈련 베이스**: chunk 체득 (문법/시제/표기 등은 곁들여 첨삭)
> **이유**: 단일 메트릭 + 명확한 진척률 + 사용자 가장 큰 약점(인지-능동 갭) 직격

### B1 쓰기 능력 10차원 점검

| # | 차원 | chunk 베이스로 커버? |
|---|------|---------------------|
| 1 | Lexique (어휘) | ⭕ 완벽 직격 |
| 2 | Morphosyntaxe (문법) | ⭕ Q2 토대 + 첨삭 |
| 3 | Cohérence (응집성) | ⭕ connecteurs chunk |
| 4 | **Consigne (글 구조/장르)** | ⚠️⚠️ **누락** — 모델 모방 필요 |
| 5 | Orthographe (표기) | ⭕ 첨삭 자동 |
| 6 | Sociolinguistique (격식) | ⭕ chunk 카테고리에 격식 정보 |
| 7 | Fluency (작문 속도) | ⭕ chunk 자동화 |
| 8 | Accuracy under pressure | ⭕ 월말 평가 45분 한정 |
| 9 | Self-revision (자가 수정) | ⚠️ 낮 — /write 본인 검토 단계 추가 |
| 10 | **Critical thinking / 논증력** | ⚠️⚠️ **누락** — 가장 본질적, chunk로 안 됨 |

→ **8/10 자동 커버**. 진짜 누락 = **글 구조 + 논증력** 2가지.

### 사용자 시제 예시 검증

> 사용자: "시제 훈련은 반복하면 충분히 체득"

✅ 정확. 시제는 Q2 토대 + chunk 사용 시 자동 + 첨삭 자동. 같은 논리로 관사/대명사/형용사 일치/어순도 자연 체득.

### 누락 보완 — 모델 모방 비중 명시

**누락 1: 글 구조 / 장르 인식**
- 의견글 vs 편지 vs 묘사 vs 이메일 = 각자 다른 구조
- 해결: 모델 모방 단계에서 **글 구조 + chunk + 논증 흐름 동시 학습**
- 6월 인프라: 장르별 모델 텍스트 10~15편 (`writing/models/`)

**누락 2: Critical thinking / 논증력** (더 본질적)

약 vs 강 논증 비교:
> 약: *Je pense que les voitures électriques sont bonnes parce qu'elles ne polluent pas.*
> 강: *Bien que les voitures électriques semblent écologiques, leur production de batteries pose un problème environnemental majeur. Cela dit, elles restent une étape vers une mobilité plus durable.*

- 두 번째 = chunk(*bien que, cela dit*) + 논증 흐름(긍정→반박→종합)
- 해결: **논증 패턴 5~10개 별도 학습**
  - 패턴 1: 양보 + 종합 (*Bien que..., il faut admettre que...*)
  - 패턴 2: 원인 + 결과 + 평가
  - 패턴 3: 비교 (X와 Y의 장단점 → 결론)
  - 패턴 4: 가정 (Si... alors..., dans ce cas...)
  - 패턴 5: 시간적 발전 (Autrefois... aujourd'hui... à l'avenir...)
- 6월 인프라: `writing/argumentation-patterns.md`

### 4-tier 시간 분배 (수정안 — 모델 모방 비중 명시)

|------|------|------|------|

## 어휘 활성화 시스템 ⭐⭐⭐ (5/1 토론 추가)

### 진단

사용자 어휘 상태 (메모리 + 5/1 시범 분석):

| 영역 | 수준 | 근거 |
|------|------|------|
| **인지 어휘 (수동)** | **B1~B2** | 단어장 4,600개 + HP 7권 + 꼬마 니콜라 집중 시 이해 |
| **능동 어휘 (생산)** | **A2** | 5/1 시범에서 영어 단어 혼용 (téa/hour/character), 단순 표현 |

→ **인지-생산 갭이 큼**. "활성화(activation)" 단계 부족.

### 두 전략

| 전략 | 효과 | ROI |
|------|------|-----|
| A. Input 양 늘리기 | 인지 어휘 ↑ | ⭐⭐ (이미 충분) |
| **B. 어휘 활성화** (수동→능동) | 생산 어휘 ↑ | ⭐⭐⭐⭐⭐ (가장 큰 약점) |

→ **B가 핵심**. A는 보조.

### 활성화 4가지 방법 (ROI 순)

#### 1. ⭐⭐⭐⭐⭐ Output 강제 노출

**원리**: 학습한 chunk를 다음 작문/대화에 **의무 사용**.

Q3 4-tier 작문에 통합:
- 매일 작문 시 **그날의 chunk 1~2개 의무 사용 슬롯**
- 워밍업 키워드에 새 chunk 끼워넣기

#### 2. ⭐⭐⭐⭐⭐ 카드 시스템 (SRS)

`writing/vocabulary-cards.md` (함정 카드와 동일 구조 4-tier):

| Tier | 분류 | 처리 |
|------|------|------|
| T1 | 이번 주 적극 학습 | 5~10 chunk, 매일 사용 강제 |
| T2 | 1주 내 복습 | 20~30 chunk, 워밍업 출제 |
| T3 | SRS 정착 | 1→3→7→14→30일 |
| T4 | 인지만 OK | 카드만 보관 |

→ **함정 카드 + 어휘 카드 = 같은 시스템**으로 통합 관리.

#### 3. ⭐⭐⭐⭐ Reading Smart (꼬마 니콜라 활용)

**현재 패턴**: "집중하면 이해" → 인지만 강화
**바꾸면**: 읽으면서 **chunk 5~10개씩 추출 → 카드화**

| 활동 | 효과 |
|------|------|
| 단순 이해 독서 | 인지 ↑ (수동) |
| **Chunk 추출 독서** | **인지 + 능동 ↑↑** |

**예시 (꼬마 니콜라 한 페이지)**:
- *se mettre à + 동사원형* (~하기 시작하다)
- *avoir l'air de + 동사원형* (~인 것 같다)
- *tout d'un coup* (갑자기)
- *faire semblant de + 동사원형* (~하는 척하다)

→ 한 책 = 100~300 chunk 잠재. **읽기를 "input"에서 "input + 추출"로 전환하면 효과 4배**.

#### 4. ⭐⭐⭐⭐ 주제별 빌드업 (Topical)

**DELF B1 시험 주제 영역 ~10개**:
가족 / 직장 / 여가 / 환경 / 교육 / 건강 / 미디어 / 여행 / 음식 / 기술

각 주제당 30~50 chunk 미리 빌드업:
- 직장: *prendre rendez-vous, faire la pause, télétravail, entretien...*
- 환경: *réchauffement climatique, recycler, transports en commun...*

→ 시험 답안 작성 시 **즉시 활용 풀**. DELF 점수 직격.

### 핵심 메시지

> **사용자에게 부족한 건 어휘량(input)이 아니라 활성화(output 전환)**.
> Q3 = "input → output" 전환 분기. 어휘 학습도 같은 원리.
> 꼬마 니콜라/유튜브를 "이해"에서 "**chunk 채굴**"로 패러다임 전환.

### 가장 빨리 효과 보는 한 가지 액션

> **꼬마 니콜라 다음 챕터부터 5 chunk 카드화**. 그게 다음 작문에 들어가야 함.
