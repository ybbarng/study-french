## ⭐ 핵심 인사이트 7가지

| # | 영역 | 핵심 |
|---|------|------|
| 1 | 발음 규칙 (사용자 요청 적중) | 자음 종결 → 모음 열림, 모음 종결 → 닫힘. 악센트 외울 필요 X |
| 2 | 두 절대 규칙 | 동일 모음 반복 금지 + 모음 충돌 회피 → 발음/축약/생략 모두 도출 |
| 3 | 24강 동사 매핑 수정판 | 직설/조건/접속 한 평면 통합 도식 |
| 4 | 29강 접속법 "정보의 양" | 동사 암기 X → 정보 흐름 방향 (밖→나 / 나→밖) |
| 5 | 34~37강 관사 정석 | 관사가 명사를 명사로 만든다. de 변형 진짜 이유 |
| 6 | 38강 꼬마니콜라 통암기 | 메타 시퀀싱 — 압축 문법 진입 전 리듬 내재화 |
| 7 | 5대 통합 원칙 | 왕 중심, 정보의 양, 비현실성 거리, 명사화, 회화/문어 분기 |
