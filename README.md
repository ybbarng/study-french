# Grammaire Française

Claude Code와 대화형으로 프랑스어 문법을 학습하는 프로젝트입니다.

## 목표
1. A1부터 C2(문어체 포함)까지 프랑스어 문법을 체계적으로 완주
2. 프랑스어 문장을 자유롭게 구사할 수 있는 수준 도달

## 학습 방식
- **전체 프랑스어로 수업** — 원어민 선생님과 대화하듯 진행
- 정형화된 퀴즈가 아닌 **대화형 수업 + 피드백**
- **간격 반복** (에빙하우스 망각 곡선 기반) — 1일 → 3일 → 7일 → 14일 → 30일
- 슬래시 커맨드로 세션 관리

│   ├── tests/             #   시험 기록 (문제, 제출 답, 채점 결과)
