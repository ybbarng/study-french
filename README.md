# Grammaire Française

Claude Code와 대화형으로 프랑스어 문법을 학습하는 프로젝트입니다.

## 목표
1. A1부터 C2(문어체 포함)까지 프랑스어 문법을 체계적으로 완주
2. 프랑스어 문장을 자유롭게 구사할 수 있는 수준 도달

## 학습 방식
- **전체 프랑스어로 수업** — 원어민 선생님과 대화하듯 진행
- 정형화된 퀴즈가 아닌 **대화형 수업 + 피드백**
- **간격 반복** (에빙하우스 망각 곡선 기반) — 1일 → 3일 → 7일 → 14일 → 30일
- 슬래시 커맨드로 세션 관리

## 커리큘럼 (15장 57항목)

| 단계 | 범위 | 핵심 | 묶음 |
|------|------|------|------|
| 초급 (A1-A2) | 1\~5장, 19항목 | 문장을 "정의하고 생성" | A, B, C |
| 중급 (B1-B2) | 6\~10장, 23항목 | 문장을 "연결·변형" | D, E, F, G |
| 고급 (C1+) | 11\~15장, 15항목 | 문장을 "설계·압축·조정" | H, I, J |

상세 진도는 [progress.md](progress.md) 참고.

## 커맨드

| 커맨드 | 용도 |
|--------|------|
| `/study` | 학습 세션 시작 (페이스 확인 → 복습 워밍업 → 수업) |
| `/review` | 취약점 복습 |
| `/write` | 작문 연습 |
| `/drill` | 동사 굴절 드릴 (집중 반복 학습) |
| `/test` | 모의시험 (수시) |
| `/study-status` | 페이스 점검 + 전체 일정 출력 |
| `/end` | 세션 종료 + 기록 + 커밋 |
