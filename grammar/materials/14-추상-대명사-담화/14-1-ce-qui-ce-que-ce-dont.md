# 14-1. ce qui / ce que / ce dont

> 특정 명사가 아니라 '내용 전체'를 가리키는 관계대명사는 어떻게 작동하는가?

## 핵심 개념

ce qui / ce que / ce dont는 특정 명사가 아니라 **'앞에서 말한 내용 전체(사건, 사실, 생각)'를 지시하는 관계대명사**다.

- qui / que / dont -> **명사 수식**
- **ce qui / ce que / ce dont -> 내용 지시**

이 차이를 놓치면 14장 전체가 흔들린다.

## 상세 설명

### 1. ce의 정체

> ce = 지시 대상이 '명사'가 아니라 '추상 내용'임을 표시하는 형식 요소

특성:
- 성/수 없음
- 항상 단수 취급
- 뒤에 오는 관계대명사가 역할을 결정

ce 자체는 혼자 서는 대명사가 아니다. 항상 다른 구조(ce + être, ce qui/que/dont)와 함께 사용된다.

### 2. ce qui -- 주어 역할

정의: ce qui = '그것이 ~하는 것' -> 관계절 안에서 **주어**

> Il a oublié son passeport, **ce qui** est grave.
>
> (그가 여권을 잊은 것, 그것이 심각하다)

> **Ce qui** m'étonne, c'est sa réaction.
>
> (나를 놀라게 하는 것은 그의 반응이다)

구조: 지시 대상 = 앞 문장 전체, qui = 관계절 주어. **"무엇이?"**에 답한다.

### 3. ce que -- 직접목적어 역할

정의: ce que = '그것을 ~하는 것' -> 관계절 안에서 **COD(직접목적어)**

> Je ne comprends pas **ce que** tu veux dire.
>
> (네가 말하려는 것을 이해 못하겠다)

> **Ce que** j'ai appris ici est précieux.
>
> (여기서 배운 것이 소중하다)

구조: comprendre / apprendre -> **무엇을?** 그 '무엇'을 내용 전체로 받음.

### 4. ce dont -- de 보어 역할

정의: ce dont = '그것에 대해 / 그것을 ~하는 것' -> **de를 요구하는 보어**

11장의 dont 논리가 그대로 적용된다.

> **Ce dont** nous avons besoin, c'est du temps.
>
> (우리가 필요한 것은 시간이다 -- avoir besoin **de**)

> **Ce dont** il a peur, c'est l'échec.
>
> (그가 두려워하는 것은 실패이다 -- avoir peur **de**)

dont 선택은 여전히 **'구조적 de'** 기준이다.

### 5. 결정 트리

ce-관계대명사 선택은 다음으로 고정된다:

```
지시 대상 = 내용 전체

관계절 안에서 역할은?

- 주어 -> ce qui
- COD(직접목적어) -> ce que
- de 보어 -> ce dont
```

의미 판단이 아니라 **문장 구조 판단**이다.

### 6. 자주 나는 오답 패턴

**(1) 주어 자리에 ce que 사용:**

> (틀림) **ce que** est grave -> 주어 자리이므로 **ce qui** est grave

**(2) de 보어 자리에 ce que 사용:**

> (틀림) **ce que** j'ai besoin -> avoir besoin **de**이므로 **ce dont** j'ai besoin

**(3) COD 자리에 ce dont 사용:**

> (틀림) **ce dont** je comprends -> comprendre는 de 요구 없음 -> **ce que** je comprends

## 비교: 한국어 / 영어 / 다른 로망스어

### 영어

영어의 "what"이 ce qui/ce que 모두를 포괄한다:

> **What** I want is peace. (ce que)
>
> **What** matters is sincerity. (ce qui)

영어에는 ce dont에 해당하는 단일 형태가 없다: "what I need"(ce dont j'ai besoin)에서 전치사가 흡수된다.

### 한국어

한국어에서는 "~하는 것"으로 통일된다:

> "중요한 **것**", "내가 원하는 **것**", "내가 필요한 **것**"

한국어에는 관계절 내부 역할에 따라 형태가 바뀌는 체계가 없으므로, 프랑스어의 삼중 구분(ce qui/ce que/ce dont)은 의식적 훈련이 필요하다.

### 스페인어

스페인어의 lo que가 ce qui/ce que에 해당한다. de 보어의 경우 lo de que / lo que 구조를 사용한다.

## 한국어 화자 주의점

1. **"~하는 것" 만능 번역의 위험**: 한국어의 "~하는 것"이 ce qui/ce que/ce dont 모두에 대응하므로, 관계절 내부의 문법적 역할을 반드시 확인해야 한다.

2. **de 보어 판별**: ce dont를 써야 할 자리에 ce que를 쓰는 오류가 가장 흔하다. 동사/형용사가 de를 요구하는지 항상 확인한다.

3. **독립 사용 vs 문장 연결**: ce qui/ce que는 문장 내부에서 명사절로 사용될 수도 있고(Ce que je veux...), 앞 문장 전체를 받을 수도 있다(..., ce qui est grave).

## 핵심 정리

| 형태 | 관계절 내부 역할 | 예 |
|---|---|---|
| ce qui | 주어 | Ce qui est important, c'est... |
| ce que | 직접목적어 (COD) | Ce que je veux, c'est... |
| ce dont | de 보어 | Ce dont j'ai besoin, c'est... |

**판별 공식:**
1. "무엇**이** ~하는가?" -> ce **qui**
2. "무엇**을** ~하는가?" -> ce **que**
3. "무엇**에 대해/으로부터** ~하는가?" (de 보어) -> ce **dont**

## 연습 문제

**1. 빈칸을 채우시오: ___ m'inquiète, c'est son silence.**

<details>
<summary>정답 및 해설</summary>

**Ce qui**

m'inquiéter(걱정시키다)의 주어가 필요. "무엇이 나를 걱정시키는가?" -> ce qui.
</details>

**2. 빈칸을 채우시오: Je ne sais pas ___ il veut.**

<details>
<summary>정답 및 해설</summary>

**ce que** (또는 ce qu')

vouloir + 직접목적어. "그가 무엇을 원하는가?" -> ce que. 모음 앞에서 ce qu'il.
</details>

**3. 빈칸을 채우시오: Voilà ___ j'ai peur.**

<details>
<summary>정답 및 해설</summary>

**ce dont**

avoir peur **de** quelque chose. de 보어 -> ce dont.
</details>

**4. 다음 오류를 수정하시오: Ce que est intéressant dans ce livre, c'est le style.**

<details>
<summary>정답 및 해설</summary>

**Ce qui** est intéressant dans ce livre, c'est le style.

"무엇이 흥미로운가?"이므로 주어 역할 -> ce qui. ce que는 직접목적어 자리에만 사용.
</details>

**5. 다음 문장에서 ce qui / ce que / ce dont를 구분하고 이유를 설명하시오.**

(a) Ce ___ il se souvient, c'est le paysage.
(b) Ce ___ il a dit était faux.
(c) Ce ___ se passe ici est inacceptable.

<details>
<summary>정답 및 해설</summary>

(a) **ce dont** -- se souvenir **de** quelque chose. de 보어.

(b) **ce qu'** -- dire quelque chose (COD). 직접목적어.

(c) **ce qui** -- se passer의 주어. "무엇이 일어나는가?"
</details>
