# 14-2. ce + être

> 내용을 '정의, 평가, 분류'하는 프랑스어의 기본 틀은 어떻게 작동하는가?

## 핵심 개념

ce + être 구조는 앞에서 언급된 **'내용 전체'를 명확히 정의, 평가, 분류**하기 위해 사용된다.

- ce = **추상 내용**
- être = **연결자(copule)**
- 뒤 요소 = **그 내용에 대한 정의/평가**

이 구조는 **정보 초점화(focus)**의 기본 장치다.

## 상세 설명

### 1. ce의 역할 (14-1 복습 + 확장)

ce는 사람/사물이 아니라 '사건/사실/말한 내용 전체'를 가리킨다.

특성:
- 성/수 없음, 항상 단수
- ça로 바꿔도 의미가 크게 변하지 않음 (구어체)
- **혼자서 문장 주어가 될 수 없다** (ce + être / ce qui/que/dont 구조 필수)

### 2. 기본 구조: ce qui/ce que + c'est...

가장 전형적인 패턴:

> **Ce qui** est important, **c'est** la cohérence.
>
> (중요한 것은 일관성이다)

> **Ce que** je veux dire, **c'est** ceci.
>
> (내가 말하려는 것은 이것이다)

> **Ce dont** j'ai peur, **c'est** l'échec.
>
> (내가 두려워하는 것은 실패이다)

구조:
- 앞절: 문제 제기 (내용 도입)
- c'est 이하: 정의 / 핵심 정보

프랑스어는 이렇게 **정보를 두 단계로 나눈다**.

### 3. 왜 항상 c'est인가

**(1) 재도입 구조:**

> (틀림) Ce qui est important, est la cohérence.
>
> (맞음) Ce qui est important, **c'est** la cohérence.

"Ce qui est important"가 이미 주어이므로, 뒤는 동격 설명이다. 프랑스어는 이때 **c'est를 사용해 재도입**한다. 이를 **분열문(cleft structure)**의 일종으로 본다.

**(2) être의 copule(연결자) 기능:**

être는 여기서 '존재'가 아니라 **'정의 연결자'**다. 뒤에 명사, 부정사, 절 모두 연결 가능하다.

### 4. c'est vs il est (결정적 구분)

| 구조 | 쓰임 |
|---|---|
| **c'est + 명사 / 대명사 / 절** | 정의, 동격 |
| **il est + 형용사** | 성질 서술 |

비교:

> **C'est** important. (일반적 강조)
>
> **Il est** important **de** partir tôt. (부정사 주어의 서술)

> **C'est** un problème. (정의)
>
> (틀림) **Il est** un problème. (명사 앞에 il est 불가)

핵심 규칙: **명사 앞에는 c'est**, 형용사 + de/que 구조에서는 il est.

구어체에서는 il est 대신 c'est를 더 넓게 사용하는 경향이 있다.

### 5. 고급 패턴: ce + être + que...

강조 구문(10-3)과도 연결된다:

> **C'est ce que** je voulais dire. (그것이 내가 말하려던 것이다)
>
> **C'est ce qui** me dérange. (그것이 나를 괴롭히는 것이다)
>
> **C'est ce dont** il est question. (그것이 문제가 되는 것이다)

ce + 관계구조 + être -> 담화 중심 설정

### 6. 시제와 부정도 자유롭다

시제 변화:

> Ce qui comptait, **c'était** le résultat. (중요했던 것은 결과였다)
>
> Ce que je veux, **ce sera** la clarté. (내가 원하는 것은 명확함일 것이다)

부정:

> **Ce n'est pas** ce que j'attendais. (이것은 내가 기대한 것이 아니다)
>
> **Ce n'est pas** ce qui pose problème. (이것이 문제가 되는 것은 아니다)

### 7. ce와 cela의 차이 (핵심)

ce와 cela는 같은 "그것"이 아니다.

| 표현 | 문법적 지위 | 쓰임 |
|---|---|---|
| **ce** | 형식 요소 | ce + être / 관계대명사 구조에서만 |
| **cela** | 완전한 지시대명사 | 문장 주어로 독립 사용 가능 |
| **ça** | 구어체 cela | 문장 주어로 독립 사용 가능 |

> (틀림) **Ce** a compliqué la situation. (ce는 être 없이 단독 주어 불가)
>
> (맞음) **Cela** a compliqué la situation. (cela는 독립 주어)

ce를 쓰려면 구조가 바뀌어야 한다:

> **Ce qui** a compliqué la situation, **c'est** son départ.
>
> (상황을 복잡하게 한 것은 그의 출발이다)

담화에서의 역할 분리:
- **14-2의 영역 -> ce** (정의/평가)
- **14-3의 영역 -> cela** (문장 간 재지시)

## 비교: 한국어 / 영어 / 다른 로망스어

### 영어

영어의 "It is... that..." 분열문이 유사하다:

> **It is** sincerity **that** matters. (c'est la sincérité qui compte)

하지만 프랑스어의 ce + être는 영어보다 훨씬 광범위하게 사용되며, 담화 구조의 핵심 장치다.

### 한국어

한국어에서는 "~하는 것은 ...이다"로 표현한다:

> "중요한 **것은** 신뢰**이다**."

한국어의 "것" + "이다"가 프랑스어의 ce + être에 대응하지만, 한국어에는 c'est vs il est의 구분이 없으므로 이 차이를 의식적으로 학습해야 한다.

## 한국어 화자 주의점

1. **c'est vs il est 혼동**: 한국어에서는 둘 다 "~이다"로 번역되지만, 프랑스어에서는 명사 앞에 il est를 쓸 수 없다.

2. **ce 단독 사용 오류**: ce는 혼자 주어가 될 수 없다. être나 관계대명사 없이 "Ce complique..."와 같은 구조는 불가능하다. 독립 주어가 필요하면 cela를 사용한다.

3. **분열문 구조 연습**: "Ce qui..., c'est..." / "Ce que..., c'est..." 패턴은 에세이와 논술에서 핵심적이므로, 반복 연습이 필요하다.

## 핵심 정리

| 구조 | 기능 | 예 |
|---|---|---|
| Ce qui/que/dont..., c'est... | 분열문 (정보 초점화) | Ce qui compte, c'est l'effort. |
| C'est + 명사 | 정의/동격 | C'est un problème. |
| Il est + 형용사 + de/que | 성질 서술 | Il est important de partir. |
| Cela + 동사 | 독립 주어로 재지시 | Cela complique tout. |

| ce | cela |
|---|---|
| 형식 요소 | 완전 대명사 |
| être/관계대명사와 함께 | 독립 주어 가능 |
| 정의/평가 구조 | 문장 간 연결 |

## 연습 문제

**1. c'est와 il est 중 올바른 것을 고르시오.**

(a) ___ un problème difficile.
(b) ___ difficile de résoudre ce problème.

<details>
<summary>정답 및 해설</summary>

(a) **C'est** un problème difficile. (명사 앞 -> c'est)

(b) **Il est** difficile de résoudre ce problème. (형용사 + de + 부정사 -> il est)
</details>

**2. ce와 cela 중 올바른 것을 고르시오: ___ a changé ma vie.**

<details>
<summary>정답 및 해설</summary>

**Cela** a changé ma vie.

être 없이 독립 주어로 사용 -> cela. (ce a changé는 불가능)
</details>

**3. 다음 문장을 분열문으로 바꾸시오: La patience est importante.**

<details>
<summary>정답 및 해설</summary>

**Ce qui est important, c'est la patience.**

정보를 두 단계로 나눈 분열문 구조.
</details>

**4. 다음 오류를 수정하시오: Ce qui est important est la cohérence.**

<details>
<summary>정답 및 해설</summary>

Ce qui est important, **c'est** la cohérence.

프랑스어에서는 분열문의 뒤쪽에 반드시 c'est를 사용하여 재도입한다.
</details>

**5. 다음 두 문장의 차이를 설명하시오.**

(a) C'est grave.
(b) Il est grave de mentir.

<details>
<summary>정답 및 해설</summary>

(a) "그것은 심각하다" -- 앞에서 언급된 내용을 받아 평가. c'est + 형용사(일반 강조).

(b) "거짓말하는 것은 심각하다" -- 부정사(mentir)를 주어로 하는 성질 서술. il est + 형용사 + de + 부정사.
</details>

## 다음 단계

ce + être로 내용을 정의/평가하는 방법을 확인했다. 이제 **14-3. 담화 연결**에서 이 정의를 다음 문장으로 어떻게 이어가는지를 다룬다. ce 계열(ce qui/ce que)과 cela는 담화의 윤활유로서 프랑스어 글쓰기의 수준을 결정한다.
