# 4-3. être / avoir

> 프랑스어에서 이 두 동사는 왜 "특별 동사"가 아니라 문법의 기반인가?

## 핵심 개념

être(~이다, 있다)와 avoir(가지다)는 의미 동사이면서 동시에 프랑스어 문법을 작동시키는 구조 동사다. être는 상태와 성질을 문장으로 성립시키는 계사(copula)이고, avoir는 소유를 나타내는 의미 동사다. 두 동사 모두 복합시제에서 조동사 역할을 하며, 프랑스어 시제 체계 전체의 뼈대를 형성한다.

## 상세 설명

### 1. 의미 동사로서의 être

être는 **계사(copula, 繫辭)**라는 특별한 문법적 지위를 갖는다. 계사란 주어와 서술어(명사, 형용사)를 연결만 해 주는 동사로, 자체적으로 동작이나 행위를 나타내지 않는다.

#### être의 핵심 기능

| 기능 | 예문 | 설명 |
|---|---|---|
| 신분/정체 | Il est médecin. | 그는 의사다. |
| 성질/상태 | Je suis fatigué. | 나는 피곤하다. |
| 소재 | Le livre est sur la table. | 책은 탁자 위에 있다. |
| 국적/출신 | Elle est coréenne. | 그녀는 한국인이다. |
| 시간 | Il est 8 heures. | 8시다. |
| 날씨(일부) | C'est beau. | 아름답다. |

#### être 없이는 상태 문장이 불가능하다

프랑스어는 être 없이 주어와 형용사/명사를 직접 연결할 수 없다.

- (틀림) Moi fatigué.
- (틀림) Lui étudiant.
- (맞음) **Je suis** fatigué. (나는 피곤하다)
- (맞음) **Il est** étudiant. (그는 학생이다)

프랑스어는 서술을 항상 동사로 실현하며, être가 그 역할을 전담한다.

#### c'est vs il est

être와 관련된 중요한 구분:

| 구조 | 용법 | 예시 |
|---|---|---|
| **c'est** + 명사 | 사물/사람을 지칭 | C'est un livre. (그것은 책이다) |
| **il/elle est** + 형용사 | 이미 알려진 주어의 속성 | Il est intéressant. (그것은 흥미롭다) |
| **c'est** + 형용사 | 상황/사실에 대한 평가 | C'est intéressant. (흥미롭다) |
| **il/elle est** + 직업명 (무관사) | 직업 | Il est médecin. (그는 의사다) |
| **c'est** + un(e) + 직업명 | 직업 (강조/수식 포함) | C'est un bon médecin. (좋은 의사다) |

### 2. 의미 동사로서의 avoir

avoir는 소유와 보유를 나타내는 동사다.

#### avoir의 핵심 기능

| 기능 | 예문 | 설명 |
|---|---|---|
| 소유 | J'ai un livre. | 나는 책이 있다. |
| 나이 | J'ai 30 ans. | 나는 30살이다. |
| 감각/상태 | J'ai faim. | 나는 배고프다. |
| 특징 | Elle a les yeux bleus. | 그녀는 파란 눈을 가지고 있다. |

#### avoir를 사용한 관용 표현 (영어/한국어와 다른 구조)

한국어에서 "~하다"나 "~이다"로 표현하는 것을 프랑스어에서는 avoir로 표현하는 경우가 많다.

| 프랑스어 | 직역 | 실제 의미 |
|---|---|---|
| avoir faim | 배고픔을 갖다 | 배고프다 |
| avoir soif | 갈증을 갖다 | 목마르다 |
| avoir chaud | 더위를 갖다 | 덥다 (체감) |
| avoir froid | 추위를 갖다 | 춥다 (체감) |
| avoir peur | 두려움을 갖다 | 무섭다 |
| avoir sommeil | 졸림을 갖다 | 졸리다 |
| avoir raison | 이치를 갖다 | 맞다 |
| avoir tort | 잘못을 갖다 | 틀리다 |
| avoir besoin de | 필요를 갖다 | ~이 필요하다 |
| avoir envie de | 욕구를 갖다 | ~하고 싶다 |
| avoir l'air | 모습을 갖다 | ~처럼 보이다 |
| avoir lieu | 장소를 갖다 | 일어나다 (사건) |
| avoir ... ans | ...살을 갖다 | ...살이다 |

주의: 이 표현들에서 avoir 뒤에 관사가 없는 경우가 많다 (avoir faim, avoir peur). 영어 be 구문(I am hungry)과 혼동하지 않도록 주의한다.

- (틀림) Je suis faim. / Je suis 30 ans.
- (맞음) J'**ai** faim. / J'**ai** 30 ans.

### 3. 조동사로서의 être와 avoir

être와 avoir의 가장 중요한 두 번째 역할은 조동사(auxiliaire)다. 다른 동사의 복합시제를 만들어준다.

#### 복합시제의 기본 구조

```
조동사 (être / avoir의 활용형) + 과거분사(participe passé)
```

- J'**ai** mangé. (나는 먹었다 -- avoir + 과거분사)
- Elle **est** arrivée. (그녀는 도착했다 -- être + 과거분사)

이 구조는 7장에서 다루는 시제 논리 전체의 뼈대다.

#### 대부분의 동사는 avoir를 조동사로 쓴다

이것이 기본값이다.

- J'**ai** parlé. (나는 말했다)
- Il **a** fini. (그는 끝냈다)
- Nous **avons** compris. (우리는 이해했다)
- Tu **as** mangé. (너는 먹었다)

#### être를 조동사로 쓰는 동사: 두 가지 범주

**범주 1: 이동/상태 변화 동사**

이 동사들은 "Dr. & Mrs. Vandertramp" 또는 "La maison d'être"라는 기억 보조 장치로 외울 수 있다.

| 동사 | 뜻 | 반대 동사 |
|---|---|---|
| aller | 가다 | venir (오다) |
| venir | 오다 | aller (가다) |
| arriver | 도착하다 | partir (떠나다) |
| partir | 떠나다 | arriver (도착하다) |
| entrer | 들어가다 | sortir (나가다) |
| sortir | 나가다 | entrer (들어가다) |
| monter | 올라가다 | descendre (내려가다) |
| descendre | 내려가다 | monter (올라가다) |
| naître | 태어나다 | mourir (죽다) |
| mourir | 죽다 | naître (태어나다) |
| rester | 머무르다 | -- |
| tomber | 떨어지다 | -- |
| retourner | 돌아가다 | -- |
| passer | 지나가다 | -- |
| devenir | ~이 되다 | -- |
| revenir | 돌아오다 | -- |

예문:
- Elle **est** arrivée hier. (그녀는 어제 도착했다)
- Nous **sommes** partis à 8h. (우리는 8시에 떠났다)
- Ils **sont** nés en France. (그들은 프랑스에서 태어났다)

**범주 2: 대명동사 (verbes pronominaux)**

se로 시작하는 동사는 모두 être를 조동사로 쓴다.

- se lever (일어나다): Elle **s'est** levée. (그녀는 일어났다)
- se souvenir (기억하다): Je **me suis** souvenu. (나는 기억해냈다)
- s'arrêter (멈추다): Il **s'est** arrêté. (그는 멈췄다)

대명동사의 상세한 내용은 6장에서 다룬다.

#### être 조동사의 결과: 과거분사 일치

être를 조동사로 쓰면 과거분사가 주어의 성수에 일치한다. 이것은 3-1에서 배운 형용사 성수 일치와 같은 원리다.

| 주어 | 예문 | 과거분사 형태 |
|---|---|---|
| il (남성 단수) | Il est arrivé. | arrivé |
| elle (여성 단수) | Elle est arrivé**e**. | arrivée |
| ils (남성 복수) | Ils sont arrivé**s**. | arrivés |
| elles (여성 복수) | Elles sont arrivé**es**. | arrivées |

avoir를 조동사로 쓰면 기본적으로 과거분사가 변하지 않는다.

- Elle **a** mangé. (그녀는 먹었다 -- mangé는 변하지 않음)

(avoir 조동사에서의 과거분사 일치 예외는 7장에서 다룬다.)

### 4. 왜 이 두 동사가 특별한가: 언어학적 관점

"있다/이다"(être/be)와 "가지다"(avoir/have)가 거의 모든 언어에서 문법적으로 특권적 지위를 갖는 이유는, 인간이 세계를 문장으로 만들기 위해 가장 먼저 필요로 하는 관계이기 때문이다.

1. "그것이 무엇인가 / 어떤 상태인가?" -> 존재, 상태 -> être / be
2. "누가 무엇을 가지고 있는가?" -> 소유, 보유 -> avoir / have

이 두 관계 없이는 사물 정의, 상태 설명, 사건 설정이 불가능하다.

#### 인도유럽어 전체의 패턴

| 언어 | 'be' | 'have' |
|---|---|---|
| 프랑스어 | être | avoir |
| 영어 | be | have |
| 독일어 | sein | haben |
| 이탈리아어 | essere | avere |
| 스페인어 | ser / estar | tener (+ haber) |

#### 왜 가장 불규칙한가

가장 많이 쓰이는 단어일수록 음운 변화에 가장 많이 노출되고, 그래서 가장 불규칙해진다. être와 avoir가 극도로 불규칙한 것은 사용 빈도가 극히 높기 때문이다.

#### 계사와 소유 동사의 차이

- **être = 계사(copula)**: 주어와 서술어를 연결만 하는 동사. 자체 의미가 매우 약하다.
- **avoir = 소유 동사**: 소유라는 관계 의미를 갖는 동사. 의미가 완전히 비워지지 않았다.
- **공통점**: 둘 다 조동사로 확장되었다.

### 5. être와 avoir의 현재시제 전체 활용

#### être (be)

| 주어 | 활용형 | 발음 |
|---|---|---|
| je | suis | /sɥi/ |
| tu | es | /ɛ/ |
| il/elle/on | est | /ɛ/ |
| nous | sommes | /sɔm/ |
| vous | êtes | /ɛt/ |
| ils/elles | sont | /sɔ̃/ |

#### avoir (have)

| 주어 | 활용형 | 발음 |
|---|---|---|
| j' | ai | /ɛ/ |
| tu | as | /a/ |
| il/elle/on | a | /a/ |
| nous | avons | /a.vɔ̃/ |
| vous | avez | /a.ve/ |
| ils/elles | ont | /ɔ̃/ |

주의사항:
- j'ai에서 je는 모음 앞이므로 j'로 축약된다.
- il a에서 연음이 일어나지 않는다 (/il a/).
- ils ont에서 연음이 일어난다 (/il.z‿ɔ̃/).

### 6. il y a 구문

avoir를 사용한 중요한 비인칭 구문이다.

**il y a = ~가 있다 (존재를 나타냄)**

- Il y a un livre sur la table. (탁자 위에 책이 있다)
- Il y a beaucoup de monde. (사람이 많다)
- Il n'y a pas de problème. (문제없다)

il y a는 "there is / there are"에 해당하며, 뒤에 오는 명사의 수에 관계없이 형태가 변하지 않는다.

## 비교: 한국어 / 영어 / 다른 로망스어

### 한국어와의 비교

한국어에서는 "이다"와 "있다"가 être의 기능을 나누어 담당하고, avoir에 대응하는 단일 동사가 없다.

| 프랑스어 | 한국어 |
|---|---|
| Il **est** médecin. | 그는 의사**이다**. |
| Il **est** fatigué. | 그는 피곤**하다**. |
| Le livre **est** sur la table. | 책은 탁자 위에 **있다**. |
| J'**ai** un livre. | 나는 책**이 있다**. |
| J'**ai** faim. | 나는 배**가 고프다**. |
| J'**ai** 30 ans. | 나는 30**살이다**. |

한국어 화자가 혼동하기 쉬운 부분:
- "배고프다" -> être(~이다)가 아닌 avoir(가지다) 구문.
- "30살이다" -> être가 아닌 avoir 구문.
- 한국어 "~하다/~이다"만으로는 être와 avoir를 구분할 수 없으므로, 프랑스어 고유의 분류를 외워야 한다.

### 영어와의 비교

영어 be와 have는 프랑스어 être/avoir와 대부분 대응하지만, 차이도 있다.

| 영어 | 프랑스어 | 차이점 |
|---|---|---|
| I **am** hungry. | J'**ai** faim. | 영어는 be, 프랑스어는 avoir |
| I **am** 30 years old. | J'**ai** 30 ans. | 영어는 be, 프랑스어는 avoir |
| I **am** cold. | J'**ai** froid. | 영어는 be, 프랑스어는 avoir |
| I **have** eaten. | J'**ai** mangé. | 동일 |
| She **has** arrived. | Elle **est** arrivée. | 영어는 have, 프랑스어는 être |

### 스페인어

스페인어에서는 "~이다"가 두 동사로 나뉜다:
- **ser**: 본질적/영구적 성질 (Soy médico. 나는 의사다.)
- **estar**: 일시적 상태/위치 (Estoy cansado. 나는 피곤하다.)
- 프랑스어 être는 ser와 estar를 모두 포괄한다.

조동사 체계: 스페인어는 haber(avoir의 대응)만 조동사로 쓰고, ser(être의 대응)는 조동사로 쓰지 않는다. 프랑스어와의 중요한 차이점이다.

### 이탈리아어

프랑스어와 가장 유사한 체계.
- essere(être) + avere(avoir) 둘 다 조동사로 사용.
- 이동 동사는 essere로: Sono arrivato. (Elle est arrivée와 동일 구조)
- 과거분사 일치도 동일하게 작동.

### 독일어

- sein(be) + haben(have) 둘 다 조동사.
- 이동 동사는 sein으로: Er ist angekommen. (그는 도착했다)
- 프랑스어와 유사한 조동사 선택 체계.

## 한국어 화자 주의점

### 1. avoir 구문을 être로 쓰는 실수

한국어에서 "~이다/~하다"로 표현되는 것을 프랑스어에서는 avoir로 표현하는 경우가 많다.

- (틀림) Je **suis** faim. -> J'**ai** faim.
- (틀림) Je **suis** 25 ans. -> J'**ai** 25 ans.
- (틀림) Je **suis** froid. -> J'**ai** froid.

교정: avoir 관용 표현(avoir faim, avoir soif, avoir chaud, avoir froid, avoir peur, avoir ... ans 등)을 덩어리로 외운다.

### 2. être를 빠뜨리는 실수

한국어에서는 "나 피곤"으로 충분하므로, 프랑스어에서도 동사를 빠뜨린다.

- (틀림) Moi fatigué.
- (맞음) Je **suis** fatigué.

교정: 프랑스어에서 형용사나 명사로 상태를 말할 때는 반드시 être가 필요하다.

### 3. 조동사 선택을 잊는 실수

복합시제에서 어떤 동사에 être를 쓰고 어떤 동사에 avoir를 쓰는지 혼동한다.

기본 기준:
- 대부분 -> avoir
- 이동/상태 변화 동사(aller, venir, arriver, partir 등) -> être
- 대명동사(se lever 등) -> être

### 4. être 조동사에서 과거분사 일치를 잊는 실수

- (틀림) Elle est arrivé.
- (맞음) Elle est arrivé**e**.

être 조동사 -> 과거분사가 주어에 일치. 이것을 자동화해야 한다.

## 어원과 역사

### être의 어원

être는 인도유럽어에서 가장 복잡한 보충법(supplétisme) 동사 중 하나다. 세 개의 라틴어 어근이 합쳐졌다:

1. **esse** (라틴어 부정사) -> être (프랑스어 부정사)
2. **sum, es, est** (라틴어 현재시제) -> suis, es, est (프랑스어 현재시제)
3. **stare** (서 있다, 머무르다) -> été (과거분사)
4. **essere** (속라틴어) -> 일부 형태에 영향

영어 be도 유사하게 세 어근이 합쳐졌다: be/been (게르만), am/is (인도유럽 *es-), was/were (인도유럽 *wes-).

### avoir의 어원

라틴어 habere(가지다, 보유하다)에서 직접 유래했다.

- habere -> avoir
- habeo -> ai
- habes -> as
- habet -> a
- habemus -> avons

h가 탈락하고 모음이 변화했지만, 기본 구조는 보존되었다.

### 조동사로의 확장

라틴어에서는 habere가 조동사가 아니었다. 속라틴어에서 "habeo + 과거분사"라는 구문이 발달하면서 조동사로 확장되었다. 이것이 프랑스어, 이탈리아어, 스페인어의 복합시제 체계로 이어졌다.

être가 이동 동사의 조동사로 선택된 이유는, 이동의 결과 상태("가서 지금 있다")를 강조하는 의미적 친화성 때문이다: "Elle est arrivée" = 문자적으로 "그녀는 도착한 상태에 있다".

### être 현재시제 활용표

| 주어 | 활용형 | 발음 |
|---|---|---|
| je | suis | /sɥi/ |
| tu | es | /ɛ/ |
| il/elle/on | est | /ɛ/ |
| nous | sommes | /sɔm/ |
| vous | êtes | /ɛt/ |
| ils/elles | sont | /sɔ̃/ |

### avoir 현재시제 활용표

| 주어 | 활용형 | 발음 |
|---|---|---|
| j' | ai | /ɛ/ |
| tu | as | /a/ |
| il/elle/on | a | /a/ |
| nous | avons | /a.vɔ̃/ |
| vous | avez | /a.ve/ |
| ils/elles | ont | /ɔ̃/ |

### être vs avoir 역할 비교

| 역할 | être | avoir |
|---|---|---|
| 의미 동사 | 상태, 신분, 위치 | 소유, 나이, 감각 |
| 문법적 지위 | 계사 (copula) | 소유 동사 |
| 조동사 | 이동/상태 변화 동사, 대명동사 | 대부분의 동사 |
| 과거분사 일치 | 주어에 일치 | 기본적으로 불일치 |

### avoir 관용 표현 핵심 목록

| 표현 | 의미 |
|---|---|
| avoir faim / soif | 배고프다 / 목마르다 |
| avoir chaud / froid | 덥다 / 춥다 |
| avoir peur | 무섭다 |
| avoir raison / tort | 맞다 / 틀리다 |
| avoir besoin de | ~이 필요하다 |
| avoir ... ans | ...살이다 |
| il y a | ~가 있다 |

## 연습 문제

### 문제 1: être / avoir 선택

빈칸에 être 또는 avoir의 올바른 형태를 넣으시오.

1. Je ________ coréen.
2. Tu ________ quel âge ?
3. Elle ________ fatiguée.
4. Nous ________ trois enfants.
5. Ils ________ à Paris.

**답과 해설:**

1. Je **suis** coréen. -- 국적: être 사용.
2. Tu **as** quel âge ? -- 나이: avoir 사용.
3. Elle **est** fatiguée. -- 상태: être 사용.
4. Nous **avons** trois enfants. -- 소유: avoir 사용.
5. Ils **sont** à Paris. -- 소재: être 사용.

### 문제 2: avoir 관용 표현

한국어를 프랑스어로 번역하시오.

1. 나는 배고프다.
2. 너는 맞다.
3. 그녀는 무섭다.
4. 우리는 도움이 필요하다.
5. 탁자 위에 책이 있다.

**답과 해설:**

1. **J'ai faim.** -- avoir faim 관용 표현.
2. **Tu as raison.** -- avoir raison 관용 표현.
3. **Elle a peur.** -- avoir peur 관용 표현.
4. **Nous avons besoin d'aide.** -- avoir besoin de 관용 표현.
5. **Il y a un livre sur la table.** -- il y a 구문.

### 문제 3: 조동사 선택

다음 동사의 복합과거(passé composé)에서 être / avoir 중 어느 것을 조동사로 쓰는지 고르시오.

1. manger (먹다)
2. arriver (도착하다)
3. finir (끝내다)
4. partir (떠나다)
5. se lever (일어나다)

**답과 해설:**

1. **avoir** -- 대부분의 동사, 기본값. J'ai mangé.
2. **être** -- 이동 동사. Elle est arrivée.
3. **avoir** -- 기본값. Il a fini.
4. **être** -- 이동 동사. Il est parti.
5. **être** -- 대명동사. Elle s'est levée.

### 문제 4: 오류 교정

1. Je suis 30 ans.
2. Elle a arrivée hier.
3. Moi fatigué.
4. Il est un chien.
5. Elle est arrivé.

**답과 해설:**

1. J'**ai** 30 ans. -- 나이는 avoir.
2. Elle **est** arrivée hier. -- arriver는 être 조동사.
3. **Je suis** fatigué. -- être 필수, 주어 대명사 필수.
4. Il **a** un chien. -- "개가 있다"는 소유(avoir). "그는 개다"가 아님.
5. Elle est arrivé**e**. -- être 조동사에서 과거분사는 주어(여성)에 일치.
