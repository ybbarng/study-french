# S5. 간접 의문문 종합 (Interrogation indirecte)

> 직접 의문 *"Que fais-tu?"* 를 *"Il demande ce que je fais"* 로 옮길 때, 의문사는 어떤 규칙으로 변환되는가?

## 핵심 개념

**공리**: 간접 의문문은 직접 질문(*"Tu viens?"*)을 **종속절**로 바꿔 다른 문장의 보어로 만든 구조다. 이 변환에는 세 가지 통사 조정이 일어난다.

1. **연결사 선택**: 직접 의문 유형에 따라 *si* / *ce que* / *ce qui* / *qui* / 기타 의문사가 선택된다.
2. **어순 정상화**: 직접 의문의 도치(*Que fais-tu?*) → 평서문 어순(*ce que je fais*).
3. **시제 일치**: 주절 동사가 과거이면 종속절 시제도 일치 규칙에 따라 조정 (참조: 10-2 간접화법).

10-2에서 이미 *si* 변환을 다뤘으나, 본 자료는 **의문사 변환의 전체 체계**를 정리한다.

| 직접 의문 유형 | 간접 의문의 연결사 |
|--------------|-------------------|
| Yes/No 질문 | ***si*** |
| *que / quoi* (사물) | ***ce que / ce qui*** |
| *qui* (사람) | ***qui*** (그대로) |
| *où / quand / comment / pourquoi / combien* | **그대로** |

## 상세 설명

### 1. Yes/No 질문 → si

#### 변환 규칙

직접 의문이 의문사 없이 도치, 억양, 또는 *est-ce que*로 형성된 Yes/No 질문이면 → 간접 의문에서는 ***si*** 도입.

#### 세 가지 직접 의문 형태 모두 *si*로 흡수

| 직접 의문 | 간접 의문 |
|----------|-----------|
| *Tu viens?* (억양) | *Il demande **si** je viens.* |
| *Est-ce que tu viens?* (est-ce que) | *Il demande **si** je viens.* |
| *Viens-tu?* (도치) | *Il demande **si** je viens.* |

세 형태 모두 같은 간접 의문이 된다. *est-ce que*는 사라지고, 도치는 평서문 어순으로 복원된다.

#### 예문

| 직접 | 간접 | IPA (간접) |
|------|------|-----------|
| *As-tu faim?* | *Il me demande **si** j'ai faim.* | /il mə də.mɑ̃d si ʒe fɛ̃/ |
| *Est-ce qu'elle arrive demain?* | *Je me demande **si** elle arrive demain.* | /ʒə mə də.mɑ̃d si ɛl a.ʁiv də.mɛ̃/ |
| *Êtes-vous prêts?* | *Il veut savoir **si** nous sommes prêts.* | /il vø sa.vwaʁ si nu sɔm pʁɛ/ |

#### *si* vs *s'il / s'ils*

*si*는 *il, ils* 앞에서 ***s'***로 축약된다 (모음 충돌 회피).

| 형태 | 예문 |
|------|------|
| *si je / tu / elle / on / nous / vous / elles* | *Il demande **si** elle vient.* |
| ***s'il / s'ils*** | *Il demande **s'il** vient.* / *Il demande **s'ils** viennent.* |

*si elle*는 축약하지 *않는다* (이미 모음 충돌 없음 -- *si* /si/ + *elle* /ɛl/).

#### 조건 *si*와의 구별

*si*는 동음이의로 **조건절**("만약 ~라면")로도 쓰인다. 문맥 동사로 구별한다.

| 환경 | 의미 |
|------|------|
| **savoir, demander, se demander, ignorer 등** + *si* | 간접 의문 ("~인지") |
| 다른 동사 또는 단독 절 시작 + *si* | 조건 ("만약 ~라면") |

| 예문 | 분석 |
|------|------|
| *Je **me demande** **si** il pleut.* | 간접 의문 (비가 오는지 궁금) |
| ***Si** il pleut, je reste.* | 조건 (비가 오면 머무른다) |

### 2. que / quoi → ce que / ce qui

#### 사물 의문사 변환

직접 의문에서 사물을 묻는 *que* (또는 강조형 *quoi*)는 간접 의문에서 ***ce que*** (목적어) 또는 ***ce qui*** (주어)로 변환된다.

#### ce que (의문사가 목적어일 때)

직접 의문 *Que* + 도치 동사 (사물이 동사의 **목적어**) → ***ce que*** + 평서문 어순.

| 직접 | 간접 |
|------|------|
| *Que fais-tu?* | *Il demande **ce que** je fais.* |
| *Que veut-il?* | *Je me demande **ce qu'**il veut.* |
| *Qu'est-ce que tu manges?* | *Il demande **ce que** je mange.* |

*Qu'est-ce que*는 *que*의 강조 형태 + 도치 회피 장치이므로, 간접 의문에서도 동일하게 *ce que*로 흡수된다.

#### ce qui (의문사가 주어일 때)

사물이 동사의 **주어**일 때는 직접 의문 *Qu'est-ce qui* → 간접 의문 ***ce qui***.

| 직접 | 간접 |
|------|------|
| *Qu'est-ce qui se passe?* | *Il demande **ce qui** se passe.* |
| *Qu'est-ce qui te plaît?* | *Je veux savoir **ce qui** te plaît.* |
| *Qu'est-ce qui est arrivé?* | *Il demande **ce qui** est arrivé.* |

#### 직접 의문에서 사람 *qui* 주어와의 대조

| 주어 유형 | 직접 의문 | 간접 의문 |
|----------|-----------|----------|
| **사물 주어** | *Qu'est-ce qui se passe?* | *... ce qui se passe* |
| **사람 주어** | *Qui est venu?* | *... qui est venu* (그대로) |

사물은 *ce qui* / *ce que*로 변환되지만, 사람은 *qui* 그대로다. 이유는 다음 절 참조.

#### 전치사 + quoi → 전치사 + quoi (또는 ce 도입)

전치사 + *quoi* 구조는 두 가지 처리가 가능하다.

| 직접 | 간접 (두 형태 가능) |
|------|---------------------|
| *À quoi penses-tu?* | *Il demande **à quoi** je pense.* / *Il demande **ce à quoi** je pense.* (격식적) |
| *De quoi parlent-ils?* | *Je veux savoir **de quoi** ils parlent.* / *... **ce dont** ils parlent.* (격식적) |

일상적으로는 전치사 + *quoi*가 그대로 간접 의문에 옮겨진다. 격식적 문체에서는 *ce* 도입 (관계대명사 구조와 평행).

#### 정리: 사물 의문사 변환표

| 직접 의문에서의 *que/quoi* | 간접 의문 |
|---------------------------|----------|
| *que* (목적어) | ***ce que*** |
| *qu'est-ce que* (목적어) | ***ce que*** |
| *qu'est-ce qui* (주어) | ***ce qui*** |
| *à/de/sur/... quoi* (전치사) | 전치사 + *quoi* (또는 *ce* 도입) |

### 3. qui (사람) → qui (그대로 유지)

#### 변환 규칙

사람을 묻는 *qui*는 **주어든 목적어든 그대로 유지**된다. *ce*가 붙지 않는다.

#### 주어 자리

| 직접 | 간접 |
|------|------|
| *Qui est venu?* | *Il demande **qui** est venu.* |
| *Qui a téléphoné?* | *Je veux savoir **qui** a téléphoné.* |
| *Qui est-ce qui veut sortir?* (강조) | *Il demande **qui** veut sortir.* |

직접 의문 *qui est-ce qui*의 강조 부분은 사라지고 *qui* 단일 형태로 흡수.

#### 목적어 자리

| 직접 | 간접 |
|------|------|
| *Qui vois-tu?* | *Il demande **qui** je vois.* |
| *Qui as-tu invité?* | *Je veux savoir **qui** tu as invité.* |
| *Qui est-ce que tu cherches?* | *Il demande **qui** je cherche.* |

#### 전치사 + qui

전치사 + *qui*는 그대로 옮겨진다.

| 직접 | 간접 |
|------|------|
| *À qui parles-tu?* | *Il demande **à qui** je parle.* |
| *Avec qui sors-tu?* | *Je veux savoir **avec qui** tu sors.* |
| *De qui parlez-vous?* | *Il demande **de qui** nous parlons.* |

#### 사물 vs 사람 대조 정리

| | 주어 | 목적어 |
|-|------|--------|
| **사물** | *ce qui* | *ce que* |
| **사람** | *qui* | *qui* |

사람 의문사는 변환 없이 그대로, 사물 의문사는 *ce* 도입. 이유: 사람은 본래부터 명사적 실체로 인식되어 *qui*가 단독으로 명사 자리를 채울 수 있지만, 사물 *que*는 약한 의문사여서 *ce*("그것")라는 명사적 받침이 필요하다.

### 4. où / quand / comment / pourquoi / combien (그대로)

#### 변환 규칙

이 다섯 의문사(부사 또는 양 표현)는 **그대로 유지**된다. 변환 없음.

| 직접 | 간접 |
|------|------|
| *Où vas-tu?* | *Il demande **où** je vais.* |
| *Quand pars-tu?* | *Je veux savoir **quand** tu pars.* |
| *Comment ça va?* | *Il demande **comment** ça va.* |
| *Pourquoi pleures-tu?* | *Il veut savoir **pourquoi** je pleure.* |
| *Combien ça coûte?* | *Il demande **combien** ça coûte.* |

#### 어순 조정만 필요

도치(*Quand pars-tu?*)는 평서문 어순(*quand tu pars*)으로 복원된다. 의문사 자체는 변하지 않는다.

| 직접 (도치) | 간접 (평서문 어순) |
|------------|---------------------|
| *Où **est-il**?* | *Il demande où **il est**.* |
| *Quand **arrive-t-elle**?* | *Il demande quand **elle arrive**.* |
| *Pourquoi **a-t-il** dit cela?* | *Je veux savoir pourquoi **il a** dit cela.* |

#### combien의 변형

*combien de + 명사*도 그대로.

| 직접 | 간접 |
|------|------|
| *Combien de livres as-tu?* | *Il demande **combien de livres** j'ai.* |

### 5. 어순 정상화

#### 핵심 원칙

직접 의문의 모든 도치는 간접 의문에서 **평서문 어순(주어 + 동사)**으로 복원된다.

| 도치 유형 | 직접 의문 | 간접 의문 (평서문 어순) |
|----------|-----------|------------------------|
| 단순 도치 (대명사 주어) | *Quand pars-tu?* | *... quand **tu pars**.* |
| 복합 도치 (명사 주어) | *Quand ton frère part-il?* | *... quand **ton frère part**.* |
| *est-ce que* 사용 | *Est-ce qu'il vient?* | *... **si il vient** / **s'il vient**.* |
| 억양만 | *Tu viens?* | *... **si tu viens**.* |

#### 명사 주어의 복합 도치는 단순화된다

직접 의문에서 명사 주어를 도치할 때는 *주어 + 동사 + 대명사* 형식("복합 도치", 참조: 10장 문장 변형)이 쓰이지만, 간접 의문에서는 평서문 어순으로 단순화된다.

| 직접 (복합 도치) | 간접 |
|----------------|------|
| *Que **veut Pierre**?* | *Il demande **ce que Pierre veut**.* |
| *Où **vont vos amis**?* | *Je veux savoir **où vos amis vont**.* |
| *Pourquoi **a-t-elle pleuré**?* | *Il demande **pourquoi elle a pleuré**.* |

#### 예외: *que* 다음 명사 주어 (격식적 도치 잔존)

격식적 문체에서는 *ce que* 뒤에 명사 주어 도치가 가끔 유지된다.

| 일상적 (어순 정상화) | 격식적 (도치 유지) |
|--------------------|--------------------|
| *Il demande **ce que pense Pierre**.* | *Il demande **ce que Pierre pense**.* |

두 형태 모두 가능하나, 일반적으로 평서문 어순이 더 흔하다.

### 6. 시제 일치 (주절이 과거일 때)

#### 일치 규칙 요약 (참조: 10-2 간접화법)

주절 동사가 과거(passé composé / imparfait / plus-que-parfait)이면 종속절 시제도 다음과 같이 조정된다.

| 직접 의문의 시제 | 간접 의문의 시제 (주절 과거 시) |
|----------------|--------------------------------|
| présent | imparfait |
| passé composé | plus-que-parfait |
| futur simple | conditionnel présent |
| futur antérieur | conditionnel passé |
| imparfait | imparfait (그대로) |
| plus-que-parfait | plus-que-parfait (그대로) |

#### 예문

| 직접 의문 (현재 시점) | 간접 의문 (주절 과거) |
|--------------------|---------------------|
| *Il a demandé: "**Tu viens**?"* | *Il a demandé si je **venais**.* (présent → imparfait) |
| *Il a demandé: "**Qu'as-tu fait**?"* | *Il a demandé ce que j'**avais fait**.* (passé composé → plus-que-parfait) |
| *Il a demandé: "**Quand viendras-tu**?"* | *Il a demandé quand je **viendrais**.* (futur → conditionnel présent) |

#### 주절이 현재/미래면 일치 없음

주절이 현재 또는 미래면 종속절은 직접 의문의 시제를 그대로 유지한다.

| 주절 시제 | 예문 |
|----------|------|
| 현재 (*il demande*) | *Il demande si je **viens**.* (현재 그대로) |
| 미래 (*il demandera*) | *Il demandera si je **viens**.* (현재 그대로) |
| 과거 (*il a demandé*) | *Il a demandé si je **venais**.* (imparfait로 이동) |

### 7. 간접 의문문이 가능한 주절 동사

#### 의문 보어를 받는 동사 목록

간접 의문문은 의문/지식/정보 관련 동사 다음에 온다.

| 동사 | 의미 |
|------|------|
| *demander* | 묻다 |
| *se demander* | 자문하다 |
| *savoir* | 알다 |
| *vouloir savoir* | 알고 싶다 |
| *ignorer* | 모르다 |
| *dire* | 말하다 (간접 의문 보어) |
| *expliquer* | 설명하다 |
| *comprendre* | 이해하다 |
| *voir* | (확인하다, "보다" → 알다) |
| *deviner* | 짐작하다 |
| *oublier* | 잊다 |

#### 예문

| 예문 | IPA |
|------|-----|
| *Je **me demande** où il est.* | /ʒə mə də.mɑ̃d u il ɛ/ |
| *Je **ne sais pas** ce qu'elle veut.* | /ʒə nə sɛ pa sə kɛl vø/ |
| *J'**ignore** si tu viendras.* | /ʒi.ɲɔʁ si ty vjɛ̃.dʁa/ |
| *Explique-moi pourquoi tu pleures.* | /ɛks.plik mwa puʁ.kwa ty plœʁ/ |

#### *demander*의 두 가지 보어

*demander*는 직접 목적어(요구)와 간접 의문(질문) 두 가지를 받을 수 있다.

| 보어 유형 | 예문 | 의미 |
|----------|------|------|
| 직접 목적어 (사물) | *Il demande **un café**.* | 커피를 요구한다. |
| 간접 의문 (절) | *Il demande **si je veux un café**.* | 내가 커피를 원하는지 묻는다. |

## 비교: 한국어 / 영어 / 다른 로망스어

### 한국어와의 비교

한국어 간접 의문은 어미 ***-는지***(현재) / ***-(으)ㄴ지***(과거) / ***-(으)ㄹ지***(미래)로 통일적으로 처리된다. 의문사가 있으면 그대로 유지하고, Yes/No 질문은 어미만으로 표현.

| 한국어 | 프랑스어 |
|--------|----------|
| 그가 오**는지** 묻는다 | *Il demande **si** il vient.* |
| 무엇을 하**는지** 묻는다 | *Il demande **ce que** je fais.* |
| 누가 오**는지** 묻는다 | *Il demande **qui** vient.* |
| 어디 가**는지** 묻는다 | *Il demande **où** je vais.* |

한국어는 어미가 의문 보어 기능을 단일하게 표시하므로 모든 유형을 같은 방식으로 처리하지만, 프랑스어는 의문사별로 변환 규칙이 다르다는 점이 학습자에게 부담이다.

### 영어와의 비교

영어는 프랑스어와 매우 유사한 체계를 가진다.

| 직접 의문 | 영어 간접 | 프랑스어 간접 |
|----------|----------|----------------|
| *Are you coming?* | *He asks **if/whether** I'm coming.* | *Il demande **si** je viens.* |
| *What are you doing?* | *He asks **what** I'm doing.* | *Il demande **ce que** je fais.* |
| *What's happening?* | *He asks **what**'s happening.* | *Il demande **ce qui** se passe.* |
| *Who came?* | *He asks **who** came.* | *Il demande **qui** est venu.* |
| *Where do you live?* | *He asks **where** I live.* | *Il demande **où** j'habite.* |

**핵심 차이**:
- 영어 *what*은 주어/목적어 자리 구별 없이 동일하게 *what*. 프랑스어는 ***ce qui*** (주어) vs ***ce que*** (목적어) 구별 필요.
- 영어는 *whether*와 *if* 둘 다 가능 (격식적 vs 일상). 프랑스어는 *si*만.
- 영어는 도치 회피만 신경 쓰면 됨 (*do/does* 제거). 프랑스어는 *est-ce que* 제거 + 도치 복원 + 의문사 변환 모두 필요.

### 스페인어/이탈리아어와의 비교

| 프랑스어 | 스페인어 | 이탈리아어 |
|----------|---------|------------|
| *Il demande **si** je viens.* | *Pregunta **si** vengo.* | *Chiede **se** vengo.* |
| *Il demande **ce que** je fais.* | *Pregunta **qué** hago.* | *Chiede **cosa** faccio.* / *... **che cosa** faccio.* |
| *Il demande **ce qui** se passe.* | *Pregunta **qué** pasa.* | *Chiede **cosa** succede.* |
| *Il demande **qui** est venu.* | *Pregunta **quién** vino.* | *Chiede **chi** è venuto.* |

**핵심 차이**: 스페인어와 이탈리아어는 사물 의문사를 *qué/cosa* 단일 형태로 처리한다 (주어/목적어 구별 없이). 프랑스어만 *ce qui / ce que* 구별이 발달했다. 이는 프랑스어 사물 의문사 *que*가 단독으로 명사 자리를 못 채우는 약한 형태이기 때문에 *ce* 받침이 필요한 통사적 특수성에서 비롯된다.

## 한국어 화자 주의점

1. **사물 *que*는 *ce que / ce qui*로 변환되지만, 사람 *qui*는 그대로**: 한국어는 둘 다 "무엇/누구"가 그대로 옮겨지므로 이 비대칭이 직관적이지 않다. 외워야 하는 핵심: **사물에는 *ce* 받침이 필요, 사람에는 불필요**. *Il demande **qui** est venu* (누가 왔는지), *Il demande **ce qui** est arrivé* (무슨 일이 일어났는지). 사람 *qui*에 *ce*를 붙이지 말 것 (*ce qui est venu*는 사물에만).

2. **사물 의문사의 주어/목적어 구별 (*ce qui* vs *ce que*)**: 직접 의문에서 *qu'est-ce qui* (주어) vs *qu'est-ce que* (목적어)의 구별이 간접 의문에서도 유지된다.
   - 주어: *Qu'est-ce qui fait ce bruit?* → *Il demande **ce qui** fait ce bruit.* (사물이 동작의 주체)
   - 목적어: *Qu'est-ce que tu manges?* → *Il demande **ce que** tu manges.* (사물이 동작의 대상)
   판별법: 종속절에서 의문사가 동사의 주어 자리에 있으면 *ce qui*, 목적어 자리에 있으면 *ce que*.

3. **도치를 반드시 풀어야 함**: 직접 의문 *"Que fais-tu?"*의 도치 *fais-tu*를 그대로 옮겨 ***Il demande ce que fais-tu*** (비문) 라고 쓰는 실수가 매우 흔하다. 간접 의문에서는 **항상 평서문 어순**: *Il demande ce que **tu fais**.* 마찬가지로 *est-ce que*는 사라진다: *Il demande **est-ce que tu viens*** (비문) → *Il demande **si tu viens***.

4. ***si*가 간접 의문인지 조건절인지 문맥 동사로 판별**: 같은 *si*가 두 가지 기능을 한다.
   - 간접 의문: *Je me demande **si** il vient.* (오는지 궁금) → *demander, savoir, ignorer* 등이 앞에 있으면 간접 의문.
   - 조건: *Je viendrai **si** tu m'invites.* (네가 부르면 갈게) → 다른 동사 또는 절 시작 위치면 조건.
   한국어로 옮길 때 간접 의문은 "~인지", 조건은 "~라면"으로 명확히 구별된다.

5. **주절이 과거이면 시제 일치 적용 (10-2 복습)**: 한국어 "그가 왔**는지** 물었다"는 시제 변화가 적지만, 프랑스어는 일치 규칙으로 *présent → imparfait*, *passé composé → plus-que-parfait*, *futur → conditionnel*. *Il a demandé si je **venais*** (왔는지 물었다, 현재 → imparfait), *Il a demandé ce que j'**avais fait*** (무엇을 했는지 물었다, passé composé → plus-que-parfait). 주절이 현재/미래면 일치 없음.

6. ***s'il / s'ils* 축약은 필수, *si elle*는 축약 안 함**: *si* 다음에 *il, ils*가 오면 반드시 *s'il, s'ils*. 다른 인칭 대명사 (*elle, on, je* 등) 앞에서는 축약하지 않는다 (*si elle*, *si on*, *si je* 그대로). 이는 모음 충돌 회피 규칙으로, *si*의 /i/와 *il*의 /i/가 충돌하기 때문이다.

## 어원과 역사

프랑스어 간접 의문 체계는 라틴어의 두 가지 구조에서 진화했다.

- **라틴어**:
  - Yes/No 간접 의문: *num* / *an* / *-ne* + 접속법. *Quaero **num** veniat* ("그가 오는지 묻는다").
  - 의문사 간접 의문: *quid* / *quis* / *ubi* 등 직접 의문사 + 접속법. *Quaero **quid** facias*.
- **속라틴어**:
  - *num/an* 소실 → 조건절의 *si*가 의문 도입에도 흡수 (의문과 가정의 의미적 친연성). 이미 후기 라틴어에서 *Quaero si venit* 같은 용법 발달.
  - 의문사는 라틴 형태 직계 (quid > que, quis > qui, ubi > où, quando > quand, quomodo > comment).
- **고프랑스어 ~ 현대 프랑스어**:
  - *quid* (사물 의문사)가 단독으로 명사 자리를 채우기에 음운적으로 너무 짧아짐 → ***ce*** ("그것", < 라틴 *ecce hoc*) 받침이 추가되어 ***ce que / ce qui*** 형성. 16~17세기에 안정화.
  - *quis* (사람 의문사)는 형태가 더 견고해서 *qui* 단독으로 유지.
  - 시제 일치 규칙은 17세기 고전 프랑스어에서 명문화. 라틴어 시제 일치(consecutio temporum)의 영향.
  - 접속법 사용 (라틴어 잔재)은 점차 직설법으로 대체됨. 현대 프랑스어 간접 의문은 직설법이 표준.
- ***pourquoi***의 형성: *pour* (~을 위해) + *quoi* (무엇) = "무엇을 위해" → "왜". 다른 의문사들이 단일 어원인 것과 달리 합성어.

이 역사는 왜 사물에만 *ce*가 붙고 사람에는 안 붙는지의 근본 이유 -- ***que***의 음운적 빈약함 -- 를 설명한다.

## 핵심 정리

```
직접 의문 → 간접 의문 변환 3단계:
  1. 연결사 선택 (의문 유형에 따라)
  2. 어순 정상화 (도치 제거, est-ce que 제거)
  3. 시제 일치 (주절이 과거이면)
```

연결사 변환표:

| 직접 의문 형태 | 간접 의문 연결사 | 예 |
|--------------|-----------------|-----|
| Yes/No (도치/est-ce que/억양) | ***si*** (s'il, s'ils 축약) | *si je viens* |
| *que* / *qu'est-ce que* (사물 목적어) | ***ce que*** | *ce que je fais* |
| *qu'est-ce qui* (사물 주어) | ***ce qui*** | *ce qui se passe* |
| *qui* / *qui est-ce qui* (사람 주어) | ***qui*** (그대로) | *qui est venu* |
| *qui* / *qui est-ce que* (사람 목적어) | ***qui*** (그대로) | *qui je vois* |
| *où / quand / comment / pourquoi / combien* | **그대로** | *où je vais* |
| 전치사 + qui/quoi | 전치사 + qui/quoi (그대로) | *à qui je parle* |

사물 vs 사람 대칭표:

| | 주어 | 목적어 |
|-|------|--------|
| **사물** | *ce qui* | *ce que* |
| **사람** | *qui* | *qui* |

시제 일치 (주절 과거 시):

| 직접 시제 | 간접 시제 |
|----------|-----------|
| présent | imparfait |
| passé composé | plus-que-parfait |
| futur | conditionnel présent |
| futur antérieur | conditionnel passé |
| imparfait, plus-que-parfait | (그대로) |

주절 동사 풀: *demander, se demander, savoir, vouloir savoir, ignorer, dire, expliquer, comprendre, deviner, oublier* 등 의문/지식/정보 동사.

## 연습 문제

**1. 다음 직접 의문문을 간접 의문문으로 바꾸시오 (주절: *Il demande*).**

a) *Tu viens?*
b) *Que fais-tu?*
c) *Qu'est-ce qui se passe?*
d) *Qui est venu?*
e) *Où vas-tu?*

<details>
<summary>정답 및 해설</summary>

a) *Il demande **si** tu viens.* — Yes/No → *si*.

b) *Il demande **ce que** tu fais.* — 사물 목적어 *que* → *ce que*. 도치 *fais-tu* → 평서문 *tu fais*.

c) *Il demande **ce qui** se passe.* — 사물 주어 *qu'est-ce qui* → *ce qui*.

d) *Il demande **qui** est venu.* — 사람 주어 *qui* (그대로).

e) *Il demande **où** tu vas.* — *où* (그대로). 도치 *vas-tu* → *tu vas*.

</details>

**2. 다음 직접 의문문을 간접 의문문으로 바꾸시오 (주절: *Je veux savoir*).**

a) *Est-ce qu'il dort?*
b) *Qu'est-ce que tu veux?*
c) *Qui as-tu invité?*
d) *Pourquoi pleure-t-elle?*
e) *Combien ça coûte?*

<details>
<summary>정답 및 해설</summary>

a) *Je veux savoir **s'il** dort.* — *si* + *il* → *s'il* 축약.

b) *Je veux savoir **ce que** tu veux.* — *qu'est-ce que* (목적어) → *ce que*.

c) *Je veux savoir **qui** tu as invité.* — 사람 목적어 *qui* (그대로). 도치 *as-tu invité* → *tu as invité*.

d) *Je veux savoir **pourquoi** elle pleure.* — *pourquoi* (그대로). 복합 도치 *pleure-t-elle* → *elle pleure*.

e) *Je veux savoir **combien** ça coûte.* — *combien* (그대로).

</details>

**3. 다음 비문을 찾고 올바르게 고치시오.**

a) *Il demande **ce que** est arrivé.*
b) *Je me demande **est-ce que** tu viens.*
c) *Il demande **ce qui** tu fais.*
d) *Je veux savoir **où vas-tu**.*
e) *Il a demandé **si** je **viens** hier.*

<details>
<summary>정답 및 해설</summary>

a) **비문 (주어/목적어 혼동)**. *arriver*가 자동사이고 의문사가 주어 자리이므로 *ce qui*. → *Il demande **ce qui** est arrivé.*

b) **비문 (*est-ce que* 잔류)**. 간접 의문에서는 *est-ce que*가 사라지고 *si*로 대체. → *Je me demande **si** tu viens.*

c) **비문 (주어/목적어 혼동)**. *faire*의 목적어이므로 *ce que*. → *Il demande **ce que** tu fais.*

d) **비문 (도치 잔류)**. 간접 의문은 평서문 어순. → *Je veux savoir **où tu vas**.*

e) **비문 (시제 일치 누락)**. 주절 *a demandé* (과거) + 종속절 *hier* (과거) → présent → imparfait. → *Il a demandé si je **venais** la veille.* (직접 의문이 과거 시점에서 발화되었으므로 *hier* → *la veille*도 시점 이동. 시제만 고치면: *Il a demandé si je **venais** hier* -- 단, *hier*가 발화 시점 기준이라면 OK).

</details>

**4. 다음 한국어 문장을 프랑스어 간접 의문문으로 옮기시오.**

a) 그가 오는지 궁금하다. (*Je me demande*)
b) 그녀가 무엇을 좋아하는지 알고 싶다. (*Je veux savoir*)
c) 그는 누가 그 말을 했는지 물었다. (*Il a demandé*, 시제 일치)
d) 어디서 그를 만났는지 기억나지 않는다. (*Je ne me souviens pas*)
e) 왜 그렇게 늦었는지 설명해 줘. (*Explique-moi*)

<details>
<summary>정답 및 해설</summary>

a) *Je me demande **s'il** vient.* — Yes/No + *si* + *il* 축약.

b) *Je veux savoir **ce qu'**elle aime.* — 사물 목적어 *ce que* (모음 앞 *ce qu'*).

c) *Il a demandé **qui** avait dit cela.* — 사람 주어 *qui*. 시제 일치: passé composé → plus-que-parfait (*a dit* → *avait dit*).

d) *Je ne me souviens pas **où** je l'ai rencontré.* — *où* (그대로). 주절 현재이므로 시제 일치 없음.

e) *Explique-moi **pourquoi** tu es arrivé si tard.* — *pourquoi* (그대로). 주절 명령문이지만 발화 시점 기준이므로 일치 없음.

</details>

**5. 다음 두 문장의 의미 차이를 설명하시오.**

a) *Je me demande **si** il vient.* vs ***Si** il vient, je reste.*
b) *Il demande **ce qui** fait ce bruit.* vs *Il demande **ce que** tu fais.*
c) *Il a demandé si je **viens**.* vs *Il a demandé si je **venais**.*

<details>
<summary>정답 및 해설</summary>

a)
- *Je me demande **si** il vient.* (= *s'il vient*) = 간접 의문 "그가 오는지 궁금하다". *se demander* (자문하다)가 *si*를 의문 도입사로 만든다.
- ***Si** il vient, je reste.* (= *S'il vient*) = 조건 "만약 그가 오면 머무른다". 절 시작 위치 + 결과절(*je reste*)이 따라옴 → 조건.

같은 *si*지만 주절 동사와 통사 환경이 기능을 결정한다.

b)
- *Il demande **ce qui** fait ce bruit.* = 그는 무엇이 이 소음을 내는지 묻는다. 의문사가 *faire*의 **주어** ("무엇이 ~을 한다") → *ce qui*.
- *Il demande **ce que** tu fais.* = 그는 네가 무엇을 하는지 묻는다. 의문사가 *faire*의 **목적어** ("너는 무엇을 한다") → *ce que*.

판별법: 종속절에서 명시적 주어가 있으면 의문사는 목적어 → *ce que*. 명시적 주어가 없으면(의문사가 곧 주어이면) → *ce qui*.

c)
- *Il a demandé si je **viens**.* — 시제 일치 누락 (또는 화자가 시제 일치를 의도적으로 무시). 표준 격식에서는 비문 또는 어색.
- *Il a demandé si je **venais**.* — 시제 일치 적용 (주절 과거 → 종속절 imparfait). 표준 격식.

격식적 글쓰기에서는 두 번째가 옳다. 단, 구어/현대 일상에서는 첫 번째도 들리며, 직접 의문의 시점이 여전히 유효하다고 화자가 판단하면 일치를 면제할 수 있다.

</details>

## 다음 단계

간접 의문문 종합은 직접 의문(10장)과 간접화법(10-2)을 잇는 통합 주제다. 같은 의문사 변환 규칙이 *demander*뿐 아니라 *savoir, ignorer, expliquer* 등 모든 지식/의문 동사 보어에 적용되므로, 본 자료의 변환표를 자동화 수준으로 익히면 복합 문장 구사력이 크게 확장된다.

다음 학습은 **단순과거 인지** (S6, 해리포터 등 문학 독해용)와 **수량/시간 전치사**(S7, S8)로 이어진다. 또한 간접 의문에서 시제 일치는 **간접화법 전체**(10-2 진술문 변환, 10-3 명령문 변환)와 동일한 원리를 공유하므로, 본 자료에서 시제 일치를 한 번 더 정착시키면 14장 담화 연결에서 자유 작문이 수월해진다. 추가로 *ce qui / ce que*는 관계대명사(8장, 11장)와 형태가 같아 **의문/관계 통합 인식**으로 발전시킬 수 있다.
