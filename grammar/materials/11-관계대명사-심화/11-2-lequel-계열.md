# 11-2. lequel 계열

> 전치사 뒤에서 qui/que가 사용 불가능할 때, 무엇이 그 자리를 대신하는가?

## 핵심 개념

전치사 뒤에서는 qui/que를 사용할 수 없으며, 그 자리를 대신하기 위해 **lequel 계열**이 사용된다. lequel은 스타일 선택이 아니라 **통사적 필연**이다.

- qui/que: 독립적 관계대명사 (주어/직접목적어 전용)
- lequel: 전치사 뒤에서만 작동하는 관계대명사

## 상세 설명

### 1. qui/que의 구조적 한계

8장에서 확인한 바에 따르면:
- qui: 관계절 안에서 **주어**
- que: 관계절 안에서 **직접목적어**

둘 다 동사의 핵심 자리(주어/목적어)만 담당한다. 전치사 뒤 자리는 동사 핵심 자리가 아니라 **전치사 보어 자리**이므로, qui/que의 관할 밖이다.

따라서 다음은 불가능하다:

> (틀림) la table **sur que** je pose le livre
>
> (틀림) la chaise **à qui** je m'assieds (사물에 qui 불가)

**참고**: 전치사 + qui는 선행사가 **사람**인 경우에만 가능하다: la personne **à qui** je parle. 사물이 선행사이면 반드시 lequel 계열을 써야 한다.

### 2. lequel의 본질과 형태

lequel = 전치사 보어 자리를 담당하는 관계대명사. 정관사(le/la/les) + quel 계열로 구성되며, 선행사의 성과 수에 일치한다.

| 성/수 | 형태 | 구성 |
|---|---|---|
| 남성 단수 | lequel | le + quel |
| 여성 단수 | laquelle | la + quelle |
| 남성 복수 | lesquels | les + quels |
| 여성 복수 | lesquelles | les + quelles |

### 3. 실제 구조 분석

> la table **sur laquelle** je pose le livre

분해: sur + **la table** -> sur + **laquelle**. 관계대명사가 명사 자리를 완전히 대체한다.

추가 예문:
- le problème **avec lequel** je lutte (lutter avec)
- la raison **pour laquelle** il est parti (partir pour)
- les règles **selon lesquelles** on travaille (travailler selon)

공통점: 전치사 + 관계대명사 = 불가분 구조

### 4. auquel / duquel -- 결합형의 필연성

프랑스어는 전치사 + 정관사 결합 규칙(au, du 등)을 이미 갖고 있다. lequel은 정관사를 포함하므로, à/de와 결합할 때 같은 축약이 일어난다.

> (틀림) à lequel / de lequel

| 전치사 + 형태 | 결합 결과 |
|---|---|
| à + lequel | **auquel** |
| à + laquelle | à laquelle (변화 없음) |
| à + lesquels | **auxquels** |
| à + lesquelles | **auxquelles** |
| de + lequel | **duquel** |
| de + laquelle | de laquelle (변화 없음) |
| de + lesquels | **desquels** |
| de + lesquelles | **desquelles** |

규칙: 축약은 남성 단수와 복수에서만 일어난다. 여성 단수는 à laquelle / de laquelle 그대로 유지된다.

### 5. dont와의 분기점 (핵심)

언제 lequel(duquel)이고 언제 dont인가?

**dont를 쓰는 경우**: de가 **동사/형용사/명사에 의해 요구**될 때
- parler **de** / avoir besoin **de** / être fier **de**

> la chose **dont** je parle

**duquel을 쓰는 경우**: de가 **전치사(구) 자체로 의미를 가질** 때
- près **de** / à côté **de** / au sujet **de**

> la maison **près de laquelle** il habite
>
> le sujet **au sujet duquel** il parle

이 구분은 11-3에서 완전한 결정 트리로 확정된다.

## 비교: 한국어 / 영어 / 다른 로망스어

### 영어와의 비교

영어는 전치사 후치를 허용하고 관계대명사를 단일화한다:

> the table **which** I put the book **on**

프랑스어는 전치사 이동이 불가능하고, 자리마다 다른 대명사가 필요하다. 이것은 **통사적 엄격함의 대가**이다.

### 한국어와의 비교

한국어에는 lequel에 해당하는 체계가 없다. "~하는 탁자", "~하는 이유" 등 관계절이 조사 없이 명사를 직접 수식한다. 프랑스어의 성/수 일치와 전치사 유지는 한국어 화자에게 가장 낯선 문법 요소 중 하나다.

### 스페인어와의 비교

스페인어의 el cual / la cual / los cuales / las cuales 체계가 lequel과 거의 동일하다. 이탈리아어의 il quale / la quale도 같은 구조다. 로망스어의 공통 유산이다.

## 한국어 화자 주의점

1. **성/수 일치 누락**: lequel은 선행사의 성과 수에 반드시 일치해야 한다. 한국어에는 문법적 성이 없으므로, 선행사의 성을 항상 확인하는 습관이 필요하다.

2. **축약형 혼동**: auquel과 duquel은 각각 à+lequel, de+lequel의 축약이다. 여성 단수에서는 축약이 일어나지 않는다는 점(à laquelle, de laquelle)을 주의해야 한다.

3. **dont와의 혼동**: de가 동사 요구인지(dont) 전치사구 자체인지(duquel)를 판별하는 훈련이 필요하다.

## 핵심 정리

| 항목 | 설명 |
|---|---|
| lequel의 정체 | 전치사 보어 자리를 대체하는 관계대명사 |
| 사용 조건 | 전치사 뒤, 특히 사물이 선행사일 때 필수 |
| 형태 | 정관사 + quel, 4가지 성/수 변화 |
| auquel/duquel | à/de + lequel의 필연적 축약 |
| dont와의 차이 | de가 동사 요구 -> dont / 전치사구 자체 -> duquel |

```
전치사 + 관계대명사 형태 선택:
  1. 선행사가 사람인가?
     -> 예 -> 전치사 + qui (또는 lequel 계열)
     -> 아니오 -> 반드시 전치사 + lequel 계열
  2. 전치사가 à/de인가?
     -> 예 -> 축약 규칙 적용 (auquel, duquel 등)
     -> 아니오 -> 전치사 + lequel 그대로
```

## 연습 문제

**1. 빈칸을 채우시오: Le film ___ je pense est excellent.**

<details>
<summary>정답 및 해설</summary>

**auquel**

penser **à** quelque chose이므로 à + lequel = auquel. film은 남성 단수.
</details>

**2. 다음 문장이 틀린 이유를 설명하시오: La raison pour que il est parti est inconnue.**

<details>
<summary>정답 및 해설</summary>

전치사 뒤에서 que는 사용할 수 없다. raison은 여성 단수이므로 **pour laquelle**이 올바르다: La raison **pour laquelle** il est parti est inconnue.
</details>

**3. dont와 duquel 중 올바른 것을 고르시오: La personne (dont / de laquelle) je parle est absente.**

<details>
<summary>정답 및 해설</summary>

**dont**

parler **de** quelqu'un에서 de는 동사에 의해 요구되므로 dont를 사용한다. de laquelle은 이 경우 부자연스럽다.
</details>

**4. 빈칸을 채우시오: La montagne au sommet ___ se trouve un refuge est dangereuse.**

<details>
<summary>정답 및 해설</summary>

**de laquelle**

au sommet **de** la montagne에서 de는 전치사구(au sommet de) 자체의 일부이므로 dont가 아닌 de laquelle을 사용한다. montagne는 여성 단수.
</details>

**5. 다음을 프랑스어로 작성하시오: "그가 참여하는 회의들은 길다."**

<details>
<summary>정답 및 해설</summary>

Les réunions **auxquelles** il participe sont longues.

participer **à** quelque chose이므로 à + lesquelles = auxquelles. réunion은 여성 복수.
</details>
