# 5-3. 부사의 형성

> 부사는 왜 대부분 -ment로 끝나며, 왜 형용사의 여성형이 기준이 되는가?

## 핵심 개념

프랑스어 부사는 형용사의 **여성형**에 접미사 **-ment**를 붙여 형성된다. 이것은 자의적인 관습이 아니라 형태론적 논리에 기반한 규칙이다. 여성형이 기준이 되는 이유는 여성형이 형용사의 어간(stem)을 가장 명확하게 보존하기 때문이다.

```
기본 공식: 형용사 여성형 + -ment = 부사
```

## 상세 설명

### 1. 기본 규칙: 형용사 여성형 + -ment

프랑스어 부사의 90% 이상이 이 규칙을 따른다.

| 형용사 (남성) | 형용사 (여성) | 부사 |
| --- | --- | --- |
| lent | lent**e** | lent**ement** |
| heureux | heureus**e** | heureuse**ment** |
| sérieux | sérieuse | sérieuse**ment** |
| doux | douc**e** | douce**ment** |
| actif | activ**e** | active**ment** |
| long | longu**e** | longue**ment** |
| naturel | naturell**e** | naturelle**ment** |
| premier | premièr**e** | première**ment** |

### 2. 왜 여성형이 기준인가: 자음 보존과 어간 가시성

이 질문에 대한 답은 음운론과 형태론에 있다.

프랑스어의 역사적 발전 과정에서, 남성형은 발음 경제성 때문에 어말 자음을 자주 탈락시켰다. 반면 여성형은 -e를 붙이면서 그 자음을 보존하는 쪽으로 남았다.

**예시 1: lent / lente**

| 형태 | 발음 | 특징 |
| --- | --- | --- |
| lent (남성) | /lɑ̃/ | 어말 t 무음 |
| lente (여성) | /lɑ̃t/ | t 발음 복원 |
| lentement | /lɑ̃t.mɑ̃/ | t 포함 어간 유지 |

**예시 2: heureux / heureuse**

| 형태 | 발음 | 특징 |
| --- | --- | --- |
| heureux (남성) | /ø.ʁø/ | 어간이 heureu-인지 heureus-인지 불명확 |
| heureuse (여성) | /ø.ʁøz/ | z 자음 복원, 어간 heureus- 명확 |
| heureusement | /ø.ʁøz.mɑ̃/ | 여성형 자음 기반 |

**예시 3: grand / grande**

| 형태 | 발음 | 특징 |
| --- | --- | --- |
| grand (남성) | /ɡʁɑ̃/ | d 무음 |
| grande (여성) | /ɡʁɑ̃d/ | d 발음 |
| grandement | /ɡʁɑ̃d.mɑ̃/ | d 유지 |

결론: 여성형은 남성형에서 탈락/약화되는 자음을 보존하여 어간을 더 명확하게 드러낸다. 부사 형성의 기준형으로 여성형을 선택한 것은 형태 예측 가능성을 높이기 위한 합리적 결과다.

**여성형이 기준이 되면 생기는 장점**:
- 자음 복원 규칙이 불필요 (어간이 이미 명확)
- "숨은 자음 추측"을 하지 않아도 됨
- 부사는 한 형태로 고정되므로, 정보가 가장 풍부한 형태를 기준으로 삼는 것이 합리적

### 3. 남성형과 여성형이 동일한 경우 (-e로 끝나는 형용사)

형용사가 이미 -e로 끝나면 남성형 = 여성형이므로, 그대로 -ment를 붙인다.

| 형용사 | 부사 |
| --- | --- |
| rapide | rapid**ement** |
| facile | facil**ement** |
| simple | simpl**ement** |
| énorme | énorm**ément** |
| sincère | sincèr**ement** |
| difficile | difficil**ement** |

### 4. -ant / -ent 형용사: -amment / -emment 규칙

이 형용사군은 별도의 규칙을 따른다.

| 형용사 어미 | 부사 어미 | 예시 |
| --- | --- | --- |
| -ant | -**amment** | constant -> const**amment** |
| -ent | -**emment** | récent -> réc**emment** |

더 많은 예시:

| 형용사 | 부사 |
| --- | --- |
| abondant | abond**amment** |
| élégant | élég**amment** |
| courant | cour**amment** |
| évident | évidem**ment** |
| prudent | prudem**ment** |
| fréquent | fréquem**ment** |
| patient | patiem**ment** |
| intelligent | intelligem**ment** |

중요: 철자는 -amment과 -emment으로 다르지만, **발음은 둘 다 /amɑ̃/으로 동일**하다.

**왜 -amment과 -emment 두 가지로 갈라졌는가?**

이것은 프랑스어 내부에서 임의로 만든 규칙이 아니라, 라틴어에서 내려온 두 계열이 프랑스어에서 합류하면서 생긴 흔적이다.

라틴어 형용사에는 두 계열의 현재분사형이 있었다:

| 라틴어 계열 | 라틴어 예시 | 여성 형태 | 프랑스어 형용사 | 프랑스어 부사 |
| --- | --- | --- | --- | --- |
| -ans 계열 | constans | constanta | constant | constamment |
| -ens 계열 | recens | récente | récent | récemment |

이 두 라틴어 계열이 프랑스어에서 각각 -ant/-ent로 변형되었고, 부사 형성에서도 그 형태적 차이가 보존되었다.

```
라틴어 -ans + mente  ->  프랑스어 -amment
라틴어 -ens + mente  ->  프랑스어 -emment
```

철자가 형태를 보존하고 발음이 음운 단순화를 따른 결과, 철자는 다르지만 발음은 합류했다. 언어에서 흔히 나타나는 현상이다: 형태 규칙은 과거를 보존하고, 발음 규칙은 현재를 따른다.

학습자 관점에서의 실용적 이점: -ant로 끝나는 형용사를 보면 -amment, -ent로 끝나는 형용사를 보면 -emment이라는 예측이 자동으로 가능하다.

### 5. 불규칙 부사 (암기 대상)

매우 높은 빈도로 사용되는 부사 중 일부는 불규칙 형태를 가진다. 이들은 규칙으로 유도할 수 없으므로 암기해야 한다.

| 형용사 | 부사 | 의미 |
| --- | --- | --- |
| bon | **bien** | 좋은 -> 잘 |
| mauvais | **mal** | 나쁜 -> 나쁘게 |
| meilleur | **mieux** | 더 좋은 -> 더 잘 |
| pire | **pis** (제한적) | 더 나쁜 -> 더 나쁘게 |
| petit | **peu** | 작은 -> 적게 |

참고: pis는 현대 프랑스어에서 거의 쓰이지 않으며, 관용 표현(tant pis, de mal en pis)에서만 남아 있다. 일반적으로 "더 나쁘게"는 plus mal을 쓴다.

### 6. 형용사가 그대로 부사로 쓰이는 경우

일부 형용사는 형태 변화 없이 그대로 부사로 사용된다. 이때 성/수 일치를 하지 않는다.

| 표현 | 의미 | 참고 |
| --- | --- | --- |
| parler **fort** | 크게 말하다 | fortement과 의미 차이 |
| parler **bas** | 작게 말하다 | - |
| coûter **cher** | 비싸다 | - |
| travailler **dur** | 열심히 일하다 | durement과 의미 차이 |
| sentir **bon** | 좋은 냄새가 나다 | - |
| sentir **mauvais** | 나쁜 냄새가 나다 | - |
| chanter **faux** | 음을 틀리게 부르다 | - |
| chanter **juste** | 음을 정확히 부르다 | - |
| voir **clair** | 또렷이 보다 | - |
| couper **court** | 짧게 자르다 / 짧게 끊다 | - |

주의: 이들이 부사로 쓰일 때는 불변이지만, 형용사로 쓰일 때는 성/수 일치를 한다.
- Elle parle **fort**. (부사 - 불변)
- Elle est **forte**. (형용사 - 성 일치)

### 7. 특수 형태의 부사

일부 부사는 형용사 남성형 + -ment 형태를 취한다 (여성형이 아닌 남성형 기반).

| 형용사 | 부사 | 설명 |
| --- | --- | --- |
| vrai | vrai**ment** | 남성형 기반 (vraie + ment -> vraiment) |
| absolu | absolu**ment** | 남성형 기반 |
| poli | poli**ment** | 남성형 기반 |
| joli | joli**ment** | 남성형 기반 |

이들은 남성형이 이미 모음으로 끝나 어간이 충분히 명확하기 때문에 여성형을 거칠 필요가 없는 경우다. 형태적으로 볼 때 규칙의 예외라기보다, 기본 원리(어간 가시성)가 이미 충족된 사례다.

그 외:
- gentil -> genti**ment** (여성형 gentille에서 -lle 탈락)
- bref -> brieve**ment** (고어적 여성형 brieve 기반)

## 비교: 한국어 / 영어 / 다른 로망스어

**한국어와의 비교**

한국어 부사는 형용사에서 파생되는 경우가 있지만, 체계가 프랑스어와 다르다.

| 한국어 | 프랑스어 |
| --- | --- |
| 빠르다 -> 빠르**게** (접미사 -게) | rapide -> rapid**ement** (접미사 -ment) |
| 느리다 -> 느리**게** | lent(e) -> lent**ement** |
| 좋다 -> **잘** (불규칙) | bon -> **bien** (불규칙) |

두 언어 모두 대부분의 부사는 규칙적으로 파생되고, 고빈도 부사 일부가 불규칙이라는 공통점이 있다.

**영어와의 비교**

영어도 형용사 + -ly로 부사를 형성한다. 프랑스어의 -ment에 대응한다.

| 영어 | 프랑스어 | 공통점 |
| --- | --- | --- |
| slow -> slow**ly** | lent(e) -> lente**ment** | 접미사 파생 |
| happy -> happi**ly** | heureux/heureuse -> heureuse**ment** | 어간 변형 후 접미사 |
| good -> **well** (불규칙) | bon -> **bien** (불규칙) | 고빈도 불규칙 |
| fast -> **fast** (형용사=부사) | vite (형용사=부사) | 일부 동일형 |

**로망스어 비교**

-ment(프랑스어), -mente(스페인어/이탈리아어/포르투갈어)는 모두 라틴어 mente(탈격, "마음/방식으로")에서 왔다.

| 라틴어 | 프랑스어 | 스페인어 | 이탈리아어 |
| --- | --- | --- | --- |
| lenta + mente | lentement | lentamente | lentamente |
| rapida + mente | rapidement | rapidamente | rapidamente |
| felice + mente | heureusement | felizmente | felicemente |

스페인어와 이탈리아어는 여성형 + -mente 형태를 더 투명하게 보존하고 있다.

## 한국어 화자 주의점

1. **여성형 확인 습관**: 부사를 만들 때 남성형이 아닌 여성형에서 출발해야 한다. heureux에서 바로 -ment를 붙이면 heureuxment(틀림)가 된다. 반드시 heureuse -> heureusement.

2. **-amment/-emment 철자**: 발음이 같아서 철자를 혼동하기 쉽다. 형용사 어미를 확인하라: -ant이면 -amment, -ent이면 -emment.

3. **bon/bien, mauvais/mal 구분**: 한국어에서 "좋은/잘", "나쁜/나쁘게"는 다른 단어이듯, 프랑스어에서도 형용사와 부사가 완전히 다른 형태다. 특히 bon(형용사)과 bien(부사)의 혼동이 잦다.

4. **형용사=부사 형태 주의**: parler fort, coûter cher 같은 표현에서 fort, cher는 부사이므로 주어가 여성이어도 일치하지 않는다. Elle parle fort. (forte가 아님)

## 어원과 역사

프랑스어 부사 접미사 -ment는 라틴어 **mente**에서 왔다. mente는 라틴어 명사 mens("마음, 정신")의 탈격(ablative) 형태로, 원래 "~한 마음으로, ~한 방식으로"라는 뜻이었다.

라틴어 구조:
```
lenta mente = "느린 마음으로" -> "느리게"
clara mente = "맑은 마음으로" -> "분명하게"
```

이 구조에서 형용사는 mente와 성 일치를 해야 했고, mens가 여성 명사이므로 형용사는 **여성형**을 사용했다. 시간이 지나면서 이 두 단어 구조가 하나의 접미사(-ment)로 굳었지만, 여성형을 기준으로 한다는 형태적 흔적은 그대로 남았다.

불규칙 부사 bien(< 라틴어 bene), mal(< 라틴어 male)은 라틴어에서 이미 형용사와 별개의 부사 형태를 가지고 있었으며, -mente 구성을 거치지 않고 직접 계승되었다.

## 핵심 정리

### 부사 형성 규칙 종합표

| 유형 | 규칙 | 예시 |
| --- | --- | --- |
| 기본 (대부분) | 여성형 + -ment | lente -> lentement |
| -e로 끝나는 형용사 | 그대로 + -ment | rapide -> rapidement |
| -ant 형용사 | -ant -> -amment | constant -> constamment |
| -ent 형용사 | -ent -> -emment | récent -> récemment |
| 모음 끝 남성형 | 남성형 + -ment | vrai -> vraiment |
| 불규칙 | 개별 암기 | bon -> bien |
| 형용사=부사 | 형태 불변 | parler fort, coûter cher |

### -amment / -emment 결정 흐름

```
형용사가 -ant/-ent로 끝나는가?
    |
    +-- -ant -> -amment  (constant -> constamment)
    |
    +-- -ent -> -emment  (récent -> récemment)
    |
    발음은 둘 다 /amɑ̃/
```

### 여성형이 기준인 이유 (한 줄)

```
남성형: 어말 자음 탈락 (불명확) -> 파생 기준으로 부적합
여성형: 어말 자음 보존 (명확)  -> 파생 기준으로 적합
```

## 연습 문제

**1. 다음 형용사에서 부사를 만드시오.**

(a) heureux
(b) facile
(c) prudent
(d) constant
(e) actif

**답:**
(a) heureuse + ment = **heureusement** (여성형 기반)
(b) facile + ment = **facilement** (이미 -e로 끝남)
(c) prudent -> **prudemment** (-ent -> -emment)
(d) constant -> **constamment** (-ant -> -amment)
(e) active + ment = **activement** (여성형 기반)

---

**2. 다음 부사에서 원래 형용사를 유추하시오.**

(a) lentement
(b) naturellement
(c) évidemment
(d) abondamment
(e) doucement

**답:**
(a) lent/lente (여성형 lente + ment)
(b) naturel/naturelle (여성형 naturelle + ment)
(c) évident (-ent -> -emment)
(d) abondant (-ant -> -amment)
(e) doux/douce (여성형 douce + ment)

**4. 다음 문장에서 밑줄 친 단어가 형용사인지 부사인지 구분하시오.**

(a) Cette robe coûte **cher**.
(b) C'est une robe **chère**.
(c) Elle chante **faux**.
(d) C'est une **fausse** réponse.

**답:**
(a) 부사 - 동사 coûter를 수식, 불변
(b) 형용사 - 명사 robe를 수식, 성 일치 (chère)
(c) 부사 - 동사 chanter를 수식, 불변
(d) 형용사 - 명사 réponse를 수식, 성 일치 (fausse)
