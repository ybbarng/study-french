# 10-4. se + 동사 (중간태와 대명동사의 정체)

> 이 구조는 수동태도 아니고 재귀도 아니다. 행위자를 문법에서 지워버리는 방식이다.

## 핵심 개념

*se + 동사* 구문 중 일부는 행위자를 특정하지 않고, 사건을 "일반적/자동적/관행적"으로 제시하는 **비인칭화 구조**다. 이를 **중간태(voix moyenne)**라고 한다. 더 넓게 보면, "재귀동사(verbe pronominal)"라는 전통적 명칭으로 묶이는 se + 동사 구문은 사실 **서로 다른 문법 기능들의 집합**이다.

- 중간태: 행위자 제거 + 사건 일반화 (COD 승격 아님, être 아님)
- 재귀: 주어 = 대상 (se = COD 또는 COI)
- 상호: 주어들이 서로에게 (se = "서로")
- 어휘적 고착: se 없이 존재하지 않는 동사

"재귀동사"는 독립된 문법 범주가 아니라, **se라는 형태가 수행하는 서로 다른 기능들에 붙은 이름**이다.

## 상세 설명

### 1. 중간태 (voix moyenne): se의 핵심 기능

#### 대표 예시

- *La porte se ferme.* /la pɔʁt sə fɛʁm/ -- "문이 닫힌다."
- *Ce livre se vend bien.* /sə livʁ sə vɑ̃ bjɛ̃/ -- "이 책은 잘 팔린다."
- *Ça se dit souvent.* /sa sə di su.vɑ̃/ -- "그것은 자주 쓰이는 말이다."

이 문장들에 "누가?"라고 물으면 질문 자체가 무의미하다. 행위자가 존재하지 않거나 중요하지 않기 때문이다.

#### 수동태와의 결정적 차이

| 기준 | 수동태 (*La porte est fermée.*) | 중간태 (*La porte se ferme.*) |
|------|-------------------------------|------------------------------|
| 주어 이동 | COD -> 주어 승격 | 이동 없음 |
| 동사 형태 | être + 과거분사 | se + 동사 (능동 형태) |
| 초점 | 결과 상태 | 일반적 작동/현상 |
| 행위자 | 삭제 가능 (par...) | 애초에 존재하지 않음 |

중간태는 사건을 "누군가의 행위"가 아니라 "일어나는 현상"으로 제시한다.

#### 능동/수동/중간태 3자 대비

| 문장 | 초점 |
|------|------|
| *On ferme la porte.* | 불특정 행위자가 닫는다 |
| *La porte est fermée.* | 결과 상태 (닫혀 있다) |
| *La porte se ferme.* | 문이 (자체적으로) 닫힌다 -- 일반적 작동 |

의미가 비슷해 보여도 담화 초점이 다르다.

#### 중간태에서 se의 정체

이 se는 **대명사(COD/COI)가 아니다**. 대체, 질문, 강조가 불가능하다.

- *La porte se ferme elle-même.* -- 부자연스러움 (재귀 해석 강제)
- *Qui ferme la porte ?* -- 의미 붕괴 (행위자가 없으므로)

se는 여기서 **비인칭화 표지(marker)**일 뿐, 문법적 보어가 아니다.

#### 전형적 의미 영역

| 영역 | 예시 |
|------|------|
| 개폐/작동 | *se fermer, s'ouvrir, se casser* |
| 유통/평가 | *se vendre, se lire, se boire* |
| 관행/규칙 | *se faire, s'utiliser, se dire* |

이 동사들은 대부분 **타동사 출신**이지만, 중간태에서 타동사성(transitivity)은 사라진다.

#### 중간태가 쓰이는 담화 맥락

- 설명문: *Ça se fait comme ça.* (이렇게 하는 거다)
- 매뉴얼/설명서: *Ce produit se conserve au réfrigérateur.* (이 제품은 냉장고에 보관합니다)
- 일반 규칙: *Les billets se vendent en ligne.* (티켓은 온라인에서 판매됩니다)
- 객관적 서술: *Cette expression ne se dit plus.* (이 표현은 더 이상 쓰이지 않는다)

### 2. 재귀 (réfléchi): se = 보어

주어가 자기 자신을 행위 대상으로 삼을 때.

- *Je me lave.* /ʒə mə lav/ -- "나는 (나 자신을) 씻는다."
- *Elle se regarde.* /ɛl sə ʁə.ɡaʁd/ -- "그녀는 (자기 자신을) 본다."

이때 se(me, te, se, nous, vous, se)는 **진짜 COD 또는 COI**다:
- *Je me lave.* -- me = COD (나 자신을 씻는다)
- *Je me lave les mains.* -- me = COI (나에게 손을 씻긴다)

대명사 배열 규칙(6장)과 과거분사 일치 규칙이 그대로 적용된다.

### 3. 상호 (réciproque): se = "서로"

주어들이 서로에게 행위를 할 때.

- *Ils se parlent.* /il sə paʁl/ -- "그들은 서로 이야기한다."
- *Elles se regardent.* /ɛl sə ʁə.ɡaʁd/ -- "그녀들은 서로 본다."

문맥상 **복수 주어**가 필요하다. se는 "서로"를 의미한다.

### 4. 어휘적 고착 (verbes essentiellement pronominaux)

현대 프랑스어에서 se 없이 의미가 성립하지 않는 동사들.

- *se souvenir (de)* /sə su.və.niʁ/ -- "기억하다" (*souvenir* 단독 불가)
- *se méfier (de)* /sə me.fje/ -- "경계하다" (*méfier* 단독 불가)
- *s'en aller* /sɑ̃.n‿a.le/ -- "떠나다"
- *s'évanouir* /se.va.nwiʁ/ -- "기절하다"
- *se repentir* /sə ʁə.pɑ̃.tiʁ/ -- "뉘우치다"
- *s'abstenir* /sap.stə.niʁ/ -- "삼가다"

이 동사들에서 se는 분해 불가능하며, **사전 단위**로 취급해야 한다. 문법이 아니라 어휘 문제다.

### 5. 왜 이들을 하나로 묶으면 안 되는가

se + 동사를 "재귀동사"라는 단일 범주로 묶으면 잘못된 질문이 발생한다:

잘못된 질문:
- "재귀동사는 왜 être를 조동사로 쓰는가?"
- "재귀동사는 왜 과거분사가 일치하는가?"

올바른 질문:
- "이 문장에서 se는 **보어인가, 비인칭화 장치인가?**"
- "이 동사는 **어휘적으로 고착되었는가?**"
- "se가 COD라면 과거분사 일치가 필요한가?"

### 6. 조동사 être와 과거분사 일치

se + 동사의 복합 시제는 항상 **être**를 조동사로 사용한다.

- *Ils se sont lavés.* /il sə sɔ̃ la.ve/ -- "그들은 씻었다."
- *Ils se sont parlé.* /il sə sɔ̃ paʁ.le/ -- "그들은 서로 이야기했다."

être를 쓰는 이유는 "재귀동사라서"가 아니라, **대명사 se가 동사 앞에 위치하기 때문**이다 (보어 위치 문제).

과거분사 일치 규칙:
- se = COD -> **일치 있음**: *Ils se sont lav**és**.* (se = COD = 그들 자신을)
- se = COI -> **일치 없음**: *Ils se sont parl**é**.* (se = COI = 서로에게, parler **à** quelqu'un)
- *Ils se sont lav**é** les mains.* (se = COI, les mains = COD) -- 일치 대상은 les mains이지만 뒤에 있으므로 일치 없음.

이 규칙은 avoir 시제의 COD 일치 규칙(6-3)과 본질적으로 동일하다.

### 7. se의 기능 분류 총정리

| se의 역할 | 문법 범주 | 예시 |
|-----------|----------|------|
| 재귀 보어 (COD/COI) | 6장 (보어 체계) | *Je me lave.* |
| 상호 보어 | 6장 확장 | *Ils se parlent.* |
| 중간태 장치 | 10장 (문장 변형) | *Ça se vend bien.* |
| 어휘적 고착 | 사전 (어휘 문제) | *se souvenir, s'en aller* |

"재귀동사"라는 단일 범주는 존재하지 않는다.

## 비교: 한국어 / 영어 / 다른 로망스어

### 한국어

한국어에서 중간태에 해당하는 표현은 자동사화 접미사("-이다", "-리다") 또는 무주어 구문으로 처리된다.

| 프랑스어 | 한국어 |
|----------|--------|
| *La porte se ferme.* | "문이 닫힌다." (-히다) |
| *Ce livre se vend bien.* | "이 책이 잘 팔린다." (-리다) |
| *Ça se dit.* | "그렇게 말한다." / "그런 말이 있다." |

### 영어

영어에는 중간태에 직접 대응하는 구조가 없다. 유사 표현은 자동사 용법이나 수동적 해석의 능동 구문이다.

| 프랑스어 | 영어 |
|----------|------|
| *La porte se ferme.* | The door closes. (자동사 용법) |
| *Ce livre se vend bien.* | This book sells well. (중간 구문) |
| *Ça se dit.* | That is said. / People say that. |

영어의 "This book sells well"은 프랑스어 중간태와 가장 유사한 구조다.

### 스페인어/이탈리아어

로망스어 전반에서 se-구문(중간태/비인칭화)이 활발히 사용된다.

| 언어 | 중간태 예시 | 한국어 |
|------|------------|--------|
| 프랑스어 | *Ça se dit.* | 그렇게 말한다. |
| 스페인어 | *Se dice así.* | 그렇게 말한다. |
| 이탈리아어 | *Si dice così.* | 그렇게 말한다. |

스페인어와 이탈리아어는 프랑스어보다 se-구문(비인칭/수동 대용)을 더 빈번하게 사용한다.

## 한국어 화자 주의점

1. **se-중간태를 수동태로 번역하는 오류**: *La porte se ferme*는 "문이 닫혀진다"(수동)가 아니라 "문이 닫힌다"(자동적 현상)에 가깝다. 수동태(être + 과거분사)와는 다른 구조다.

2. **se를 항상 "자기 자신"으로 해석하는 오류**: 재귀 용법에서만 "자기 자신"이다. 중간태에서 se는 비인칭화 표지일 뿐이다.

3. **과거분사 일치 판단 오류**: se가 COD인지 COI인지를 판단해야 한다. *Ils se sont lavés* (se=COD, 일치) vs *Ils se sont parlé* (se=COI, 일치 없음).

4. **어휘적 고착 동사의 se 분리 시도**: *se souvenir*에서 se를 분리하여 *souvenir*로 쓰려는 오류. 이 동사들은 사전 단위로 암기해야 한다.

## 어원과 역사

프랑스어의 se-구문은 라틴어의 재귀 대명사 체계에서 유래했다.

- 라틴어에는 **중간태(medium)**가 동사 형태로 존재했다 (*lavatur* = "씻기다/씻어지다"). 속라틴어에서 이 합성 형태가 소멸하면서, 재귀 대명사 *se*를 이용한 분석적 구조로 대체되었다.
- 결과적으로 프랑스어의 *se*는 라틴어의 재귀 기능 + 중간태 기능 + 수동 기능을 모두 흡수했다.
- "재귀동사(verbe pronominal)"라는 전통적 명칭은, 형태(se + 동사)가 동일하다는 이유만으로 서로 다른 기능을 하나로 묶은 **분류 편의상의 용어**다.

## 핵심 정리

- se + 동사의 네 가지 기능: 재귀(보어), 상호(서로), 중간태(비인칭화), 어휘적 고착(사전)
- "재귀동사"라는 단일 범주는 존재하지 않는다
- 중간태: 행위자 제거 + 사건 일반화. 수동태와 다른 구조.
- se가 COD이면 과거분사 일치, COI이면 일치 없음
- 어휘적 고착 동사(se souvenir, s'en aller 등)는 분해 불가, 사전 단위로 암기

| se의 기능 | 판별 기준 | 과거분사 일치 |
|-----------|----------|--------------|
| 재귀 (COD) | "자기 자신을" = COD | 일치 |
| 재귀 (COI) | "자기 자신에게" = COI | 불일치 |
| 상호 | 복수 주어 + "서로" | COD/COI에 따름 |
| 중간태 | 행위자 질문 무의미 | 일치 (주어 = 문법적 주어) |
| 어휘적 고착 | se 없이 의미 불가 | 일치 (대부분) |

## 연습 문제

**1. 다음 문장에서 se의 기능을 판별하시오.**
*Ce plat se mange froid.*

> **답**: 중간태. "이 요리는 차갑게 먹는다." 행위자가 특정되지 않으며, 일반적 관행을 서술한다.

**2. 다음 문장에서 se의 기능을 판별하시오.**
*Elle se regarde dans le miroir.*

> **답**: 재귀 (COD). "그녀는 거울에 비친 자기 자신을 본다." se = COD (자기 자신을).

**3. 과거분사 일치를 판단하시오.**
*Elles se sont téléphoné___.*

> **답**: *téléphoné* (일치 없음)
> **해설**: *téléphoner **à** quelqu'un* -> se = COI. COI이므로 과거분사 일치 없음.

**4. 과거분사 일치를 판단하시오.**
*Elles se sont vu___ dans la rue.*

> **답**: *vu**es*** (여성 복수 일치)
> **해설**: *voir quelqu'un* -> se = COD. COD이므로 과거분사 일치.

**5. 다음 문장이 중간태인지 수동태인지 판단하시오.**
a) *La fenêtre est cassée.*
b) *La fenêtre se casse facilement.*

> **답**: a) 수동태 (être + 과거분사, 결과 상태). b) 중간태 (se + 동사, 일반적 속성 -- "이 창문은 쉽게 깨진다").

**6. 다음 동사가 어휘적 고착인지, 재귀/중간태인지 판단하시오.**
a) *se souvenir*
b) *se laver*
c) *se vendre*

> **답**: a) 어휘적 고착 (souvenir 단독 불가). b) 재귀 (laver + 자기 자신). c) 중간태 (vendre의 비인칭화).

## 다음 단계

문장 변형 파트를 완주했다:
- **10-1 수동태**: 구조 이동 (COD -> 주어)
- **10-2 간접화법**: 시점 이동 (인칭/지시어/시제 재배치)
- **10-3 강조 구조**: 초점 이동 (c'est...qui/que)
- **10-4 se + 동사**: 행위자 일반화/제거

이후 선택지:
- **11장. 전치사 + 관계대명사 (lequel 계열)**: 관계절의 고급 구조
- **13장. 접속법 과거**: 접속법에서 완료 인식 표현
