# 3-3. 비교급과 최상급

> 프랑스어에서 "더 ~하다 / 가장 ~하다"는 어떻게 문법화되는가?

## 핵심 개념

프랑스어의 비교와 최상급은 형용사 자체의 형태를 바꾸는 것이 아니라, 형용사 앞에 정도 부사(plus, moins, aussi)를 추가하는 분석적 방식으로 이루어진다. 형용사는 그대로 유지되고, 비교 정보는 별도 요소로 표시된다.

## 상세 설명

### 1. 비교급의 세 가지 구조

프랑스어 비교급은 정확히 세 가지다.

| 비교 유형 | 구조 | 의미 |
|---|---|---|
| 우등 비교 (comparatif de supériorité) | **plus** + 형용사 + **que** | ~보다 더 |
| 열등 비교 (comparatif d'infériorité) | **moins** + 형용사 + **que** | ~보다 덜 |
| 동등 비교 (comparatif d'égalité) | **aussi** + 형용사 + **que** | ~만큼 |

비교 대상은 **que**(~보다, ~만큼)로 연결한다.

#### 우등 비교

- Il est **plus grand que** moi. (그는 나보다 더 크다)
- Ce film est **plus intéressant que** l'autre. (이 영화가 다른 것보다 더 흥미롭다)

#### 열등 비교

- Ce livre est **moins cher que** celui-là. (이 책이 저것보다 덜 비싸다)
- Elle est **moins fatiguée que** hier. (그녀는 어제보다 덜 피곤하다)

#### 동등 비교

- Elle est **aussi intelligente que** lui. (그녀는 그만큼 지적이다)
- Paris est **aussi beau que** Rome. (파리는 로마만큼 아름답다)

주의: 부정문에서 동등 비교의 aussi는 **si**로 바뀔 수 있다.
- Elle n'est pas **si** grande **que** sa sœur. (그녀는 언니만큼 크지 않다)

### 2. 형용사는 변하지 않는다

비교급에서 형용사 자체는 형태 변화가 없다. 성수 일치는 수식하는 명사에 따라 여전히 적용되지만, 비교 자체가 형태를 바꾸지는 않는다.

- Elle est plus grand**e** que lui. (grande는 elle에 일치, 비교와 무관)
- Elles sont plus grand**es** que nous. (grandes는 elles에 일치)

### 3. 최상급의 구조

최상급은 비교급에 정관사를 추가하여 만든다.

| 유형 | 구조 |
|---|---|
| 우등 최상급 | **le/la/les + plus** + 형용사 |
| 열등 최상급 | **le/la/les + moins** + 형용사 |

정관사는 명사의 성수에 일치한다.

- **le plus** grand bâtiment (가장 큰 건물)
- **la plus** belle ville (가장 아름다운 도시)
- **les plus** intéressants livres (가장 흥미로운 책들)
- **les moins** chères options (가장 덜 비싼 선택지들)

### 4. 최상급의 위치

최상급 형용사의 위치는 원래 형용사의 위치를 따른다.

**전치 형용사(BANGS)의 최상급 -> 명사 앞:**
- la **plus belle** ville (가장 아름다운 도시)
- le **plus grand** homme (가장 위대한 사람)
- le **plus petit** enfant (가장 작은 아이)

**후치 형용사의 최상급 -> 명사 뒤:**
- le film **le plus intéressant** (가장 흥미로운 영화)
- la solution **la moins chère** (가장 저렴한 해결책)

후치 최상급에서는 정관사가 두 번 나타난다는 점에 주의: le film **le** plus intéressant.

### 5. 최상급 뒤의 전치사: de

최상급 뒤에 범위를 나타낼 때 전치사 **de**를 사용한다 (영어의 in/of에 해당).

- le plus grand **de** la classe (반에서 가장 큰)
- la plus belle ville **du** monde (세상에서 가장 아름다운 도시)
- le meilleur restaurant **de** Paris (파리 최고의 레스토랑)

### 6. 불규칙 비교형 (핵심 암기 대상)

#### 형용사 bon의 비교급/최상급

| 원급 | 비교급 | 최상급 |
|---|---|---|
| bon(ne) (좋은) | **meilleur(e)** (더 좋은) | **le/la meilleur(e)** (가장 좋은) |

- Ce gâteau est **meilleur** que l'autre. (이 케이크가 다른 것보다 더 맛있다)
- C'est **la meilleure** étudiante de la classe. (그녀가 반에서 최고의 학생이다)

주의: plus bon은 사용하지 않는다 (문법 오류).

meilleur는 형용사이므로 성수 일치한다: meilleur, meilleure, meilleurs, meilleures.

#### 형용사 mauvais의 비교급/최상급

| 원급 | 비교급 | 최상급 |
|---|---|---|
| mauvais(e) (나쁜) | **pire** 또는 plus mauvais(e) | **le/la pire** 또는 le/la plus mauvais(e) |

- La situation est **pire** qu'avant. (상황이 이전보다 나쁘다)
- C'est **le pire** film de l'année. (올해 최악의 영화다)

pire는 더 강한 뉘앙스이고, plus mauvais는 더 중립적이다. 둘 다 문법적으로 정확하다.

#### 부사 bien의 비교급/최상급 (형용사와 구분 중요)

| 원급 | 비교급 | 최상급 |
|---|---|---|
| bien (잘, 부사) | **mieux** (더 잘) | **le mieux** (가장 잘) |

- Elle chante **mieux** que moi. (그녀가 나보다 더 잘 노래한다)
- C'est lui qui travaille **le mieux**. (그가 가장 잘 일한다)

핵심 구분:
- **bon -> meilleur**: 형용사 (명사를 수식)
- **bien -> mieux**: 부사 (동사를 수식)

혼동 방지:
- Ce gâteau est **meilleur**. (이 케이크가 더 좋다 -- bon의 비교급)
- Elle cuisine **mieux**. (그녀가 더 잘 요리한다 -- bien의 비교급)

#### 부사 mal의 비교급

| 원급 | 비교급 | 최상급 |
|---|---|---|
| mal (나쁘게, 부사) | **pis** (문어) / plus mal (구어) | le pis / le plus mal |

pis는 현대 프랑스어에서 관용 표현(tant pis = 어쩔 수 없지)에 주로 남아 있다.

### 7. plus의 발음

- plus + 자음: /ply/ -- plus grand /ply ɡʁɑ̃/
- plus + 모음: /plyz/ -- plus intéressant /ply.z‿ɛ̃.te.ʁɛ.sɑ̃/ (연음)
- ne...plus (부정): /ply/ -- Je ne mange plus /ʒə nə mɑ̃ʒ ply/
- 단독 사용(더 많이): /plys/ -- J'en veux plus /ʒɑ̃ vø plys/

### 8. 비교 강조 표현

비교를 강조할 때 사용하는 부사:

- **beaucoup** plus grand (훨씬 더 큰)
- **bien** plus intéressant (훨씬 더 흥미로운)
- **un peu** plus cher (조금 더 비싼)
- **encore** plus beau (더욱 더 아름다운)
- **de loin** le meilleur (단연 최고의)

## 비교: 한국어 / 영어 / 다른 로망스어

### 한국어와의 비교

한국어는 "더"와 "가장"을 형용사 앞에 추가하여 비교를 표현한다: "더 크다", "가장 크다". 비교 대상은 조사 "~보다"로 연결한다. 구조적으로 프랑스어의 plus...que와 유사하다.

- 더 크다 = plus grand
- ~보다 = que
- 가장 크다 = le plus grand

차이점: 한국어에는 bon/meilleur 같은 불규칙 비교형이 없다. "더 좋다"로 충분하다.

### 영어와의 비교

영어는 두 가지 방식을 혼용한다:
- 짧은 형용사: 어미 변화 (big -> bigger -> biggest)
- 긴 형용사: 분석적 (more interesting, most interesting)

프랑스어는 영어의 "긴 형용사" 방식만 사용한다. 어미 변화 방식(bigger 같은)은 존재하지 않는다. 다만 bon/meilleur는 영어의 good/better와 유사한 불규칙 패턴이다.

| 영어 | 프랑스어 |
|---|---|
| good -> better -> best | bon -> meilleur -> le meilleur |
| bad -> worse -> worst | mauvais -> pire -> le pire |
| well -> better -> best | bien -> mieux -> le mieux |

### 스페인어

프랑스어와 거의 동일한 분석적 구조.
- más grande que (plus grand que)
- menos interesante que (moins intéressant que)
- tan inteligente como (aussi intelligent que -- 동등 비교에서 como 사용, 프랑스어의 que와 다름)
- bueno -> mejor (bon -> meilleur)
- malo -> peor (mauvais -> pire)

### 이탈리아어

역시 분석적 구조.
- più grande di (plus grand que -- di 사용, 프랑스어의 que와 다름)
- meno interessante di (moins intéressant que)
- buono -> migliore (bon -> meilleur)

### 독일어

형용사 어미 변화 방식을 사용하여 영어와 유사.
- gross -> grösser -> am grössten (big -> bigger -> biggest)
- gut -> besser -> am besten (good -> better -> best)

요약: 프랑스어는 로망스어 형제들과 분석적 비교 구조를 공유하며, 영어/독일어의 종합적(어미 변화) 방식과 대비된다.

## 한국어 화자 주의점

### 1. plus bon이라고 말하는 실수

한국어에서 "더 좋다"가 자연스러우므로, bon에 plus를 붙이고 싶어진다.

- (틀림) Ce gâteau est plus bon.
- (맞음) Ce gâteau est **meilleur**.

bon -> meilleur, bien -> mieux는 반드시 외워야 한다.

### 2. meilleur와 mieux를 혼동하는 실수

둘 다 "더 좋은/잘"이라 한국어로는 구분이 어렵다.

판단 기준: 수식 대상이 명사면 meilleur(형용사), 동사면 mieux(부사).

- Ce gâteau est **meilleur**. (형용사: 케이크가 더 좋다)
- Il cuisine **mieux**. (부사: 더 잘 요리한다)

### 3. 최상급에서 정관사를 빠뜨리는 실수

한국어 "가장 큰"에는 관사가 없으므로 le/la/les를 잊기 쉽다.

- (틀림) C'est plus belle ville.
- (맞음) C'est **la** plus belle ville.

### 4. 최상급 범위의 전치사를 in으로 쓰는 실수

영어 "the best in the world"의 영향으로 dans을 쓰고 싶어지지만, 프랑스어는 **de**를 사용한다.

- (틀림) le meilleur dans le monde
- (맞음) le meilleur **du** monde

### 5. 후치 형용사의 최상급에서 정관사 중복을 빠뜨리는 실수

- (틀림) le film plus intéressant
- (맞음) le film **le plus** intéressant

후치 최상급에서는 정관사가 명사 앞과 형용사 앞에 각각 한 번씩, 총 두 번 나타난다.

## 어원과 역사

라틴어는 형용사 비교급과 최상급을 어미 변화로 표현했다:
- altus (높은) -> alt**ior** (더 높은) -> alt**issimus** (가장 높은)

이 종합적(synthetic) 체계가 속라틴어를 거치면서 붕괴하고, 분석적(analytic) 체계로 대체되었다:
- plus + 형용사 (라틴어 plus에서 직접 계승)
- magis + 형용사 (스페인어의 más, 포르투갈어의 mais가 여기서 유래)

다만 라틴어 불규칙 비교형 일부가 화석처럼 남았다:
- bonus -> melior -> meilleur (라틴어 비교급 melior가 직접 살아남음)
- malus -> peior -> pire (라틴어 비교급 peior에서)
- bene -> melius -> mieux (라틴어 부사 비교급 melius에서)

이 불규칙형들이 살아남은 이유는 사용 빈도가 극히 높아서, 분석적 체계로 교체되기 전에 이미 화석화되었기 때문이다.

## 핵심 정리

### 비교급/최상급 구조표

| 유형 | 우등 | 열등 | 동등 |
|---|---|---|---|
| 비교급 | plus + 형 + que | moins + 형 + que | aussi + 형 + que |
| 최상급 | le/la/les + plus + 형 | le/la/les + moins + 형 | -- |

### 불규칙 비교 핵심표

| 원급 | 품사 | 비교급 | 최상급 |
|---|---|---|---|
| bon | 형용사 | meilleur(e)(s) | le/la/les meilleur(e)(s) |
| mauvais | 형용사 | pire / plus mauvais | le/la pire / le/la plus mauvais |
| bien | 부사 | mieux | le mieux |
| mal | 부사 | pis / plus mal | le pis / le plus mal |

### 핵심 원칙

- 비교는 plus / moins / aussi + 형용사 (형용사 자체는 변하지 않는다).
- 비교 대상은 que로 연결한다.
- 최상급은 정관사 + plus/moins + 형용사이다.
- 최상급 범위는 de로 표현한다 (in이 아님).
- bon -> meilleur, bien -> mieux는 반드시 구분한다.
- 후치 최상급에서 정관사가 두 번 나타난다.

## 연습 문제

### 문제 1: 빈칸 채우기

1. Marie est ________ (plus / intelligent) ________ Paul.
2. Ce restaurant est ________ (bon, 비교급) ________ l'autre.
3. C'est la ville ________ (plus / beau) ________ France.
4. Il court ________ (bien, 비교급) ________ moi.
5. Cette solution est ________ (moins / cher) ________ l'autre.

**답과 해설:**

1. Marie est **plus intelligente que** Paul. -- intelligente는 Marie(여성)에 일치.
2. Ce restaurant est **meilleur que** l'autre. -- bon의 비교급은 meilleur, plus bon 아님.
3. C'est la ville **la plus belle de** France. -- 후치 최상급이므로 정관사 중복. 범위는 de.
4. Il court **mieux que** moi. -- bien(부사)의 비교급은 mieux.
5. Cette solution est **moins chère que** l'autre. -- 여성형 일치: cher -> chère.

### 문제 2: 한국어를 프랑스어로

1. 이 책이 저것보다 더 흥미롭다.
2. 그녀는 반에서 가장 좋은 학생이다.
3. 이 문제는 저것만큼 어렵다.
4. 상황이 어제보다 나쁘다.
5. 그가 가장 잘 노래한다.

**답과 해설:**

1. **Ce livre est plus intéressant que celui-là.** -- 우등 비교 기본 구조.
2. **Elle est la meilleure étudiante de la classe.** -- bon의 최상급 meilleure, 범위는 de.
3. **Ce problème est aussi difficile que celui-là.** -- 동등 비교.
4. **La situation est pire qu'hier.** -- mauvais의 비교급 pire. que + hier -> qu'hier (모음 축약).
5. **C'est lui qui chante le mieux.** -- bien의 최상급 le mieux.

### 문제 3: 오류 교정

1. Elle est plus bonne que sa sœur.
2. C'est le plus intéressant film.
3. Il est le plus grand dans la classe.
4. Ce gâteau est plus mieux.
5. La plus belle ville dans le monde.

**답과 해설:**

1. Elle est **meilleure** que sa sœur. -- plus bonne은 틀림, meilleure가 맞음.
2. C'est le film **le plus intéressant**. -- intéressant은 후치 형용사, 명사 뒤에서 최상급.
3. Il est le plus grand **de** la classe. -- 범위는 dans이 아닌 de.
4. Ce gâteau est **meilleur**. 또는 Ce gâteau est **mieux**. -- plus mieux는 이중 비교, 틀림.
5. La plus belle ville **du monde**. -- dans le monde가 아닌 du monde (de + le = du).

### 문제 4: meilleur vs mieux 구분

올바른 형태를 고르시오.

1. Ce vin est (meilleur / mieux) que l'autre.
2. Elle travaille (meilleur / mieux) que son collègue.
3. C'est la (meilleure / mieux) idée.
4. Il va (meilleur / mieux) aujourd'hui.
5. Les (meilleurs / mieux) résultats sont ici.

**답과 해설:**

1. **meilleur** -- 형용사 (vin을 수식).
2. **mieux** -- 부사 (travaille를 수식).
3. **meilleure** -- 형용사 (idée를 수식, 여성형).
4. **mieux** -- 부사 (va를 수식).
5. **meilleurs** -- 형용사 (résultats를 수식, 남성 복수).

### 문제 5: 비교 강조

다음 문장에 적절한 강조 부사를 넣으시오.

1. Il est ________ plus grand que moi. (훨씬)
2. C'est ________ le meilleur. (단연)
3. Elle est ________ plus âgée que lui. (조금)

**답과 해설:**

1. Il est **beaucoup** (또는 **bien**) plus grand que moi. -- beaucoup/bien + 비교급 = 훨씬.
2. C'est **de loin** le meilleur. -- de loin + 최상급 = 단연.
3. Elle est **un peu** plus âgée que lui. -- un peu + 비교급 = 조금 더.
