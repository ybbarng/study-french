# 3-1. 형용사 성수 일치 규칙

> 프랑스어 형용사는 왜, 어떻게 명사의 성과 수에 맞춰 형태가 변하는가?

## 핵심 개념

프랑스어 형용사는 수식하는 명사의 성(genre)과 수(nombre)에 반드시 일치해야 한다. 이것은 선택이나 스타일의 문제가 아니라 문법적 강제 규칙이다.

- 성수 정보의 출처는 항상 명사다.
- 형용사는 그 정보를 표면에 반영하는 수동적 요소다.
- 관사 유무와 무관하게 일치는 항상 작동한다.

## 상세 설명

### 1. 왜 형용사는 일치해야 하는가

프랑스어에서 성과 수는 명사와 다른 요소를 연결하는 문법적 기준(anchor)이다. 형용사는 명사의 속성을 설명하고, 명사와 같은 정보 단위로 묶인다. 따라서 형용사는 명사의 성수를 복제한다.

정확한 표현은 "형용사가 변한다"가 아니라, "형용사가 명사의 성수 정보를 표면에 반영한다"이다.

### 2. 기본 일치 규칙

#### 성 일치: 여성형은 -e를 추가

| 남성형 | 여성형 | 뜻 |
|---|---|---|
| petit | petit**e** | 작은 |
| grand | grand**e** | 큰 |
| intelligent | intelligent**e** | 지적인 |
| content | content**e** | 만족한 |
| bleu | bleu**e** | 파란 |

이미 -e로 끝나는 형용사는 남녀 동형이다:

| 남성형 | 여성형 | 뜻 |
|---|---|---|
| jeune | jeune | 젊은 |
| triste | triste | 슬픈 |
| difficile | difficile | 어려운 |
| rouge | rouge | 빨간 |

#### 수 일치: 복수형은 -s를 추가

| 단수 | 복수 | 뜻 |
|---|---|---|
| grand | grand**s** | 큰 |
| petite | petite**s** | 작은 (여성) |
| intéressant | intéressant**s** | 흥미로운 |

이미 -s 또는 -x로 끝나는 형용사는 복수에서도 변하지 않는다:

| 단수 | 복수 | 뜻 |
|---|---|---|
| gros | gros | 뚱뚱한 |
| heureux | heureux | 행복한 |

#### 성 + 수 동시 적용

| 형태 | 예시 |
|---|---|
| 남성 단수 | un livre intéressant |
| 여성 단수 | une table intéressant**e** |
| 남성 복수 | des livres intéressant**s** |
| 여성 복수 | des tables intéressant**es** |

### 3. 주요 불규칙 여성형 유형

기본 규칙(-e 추가) 외에 체계적인 변환 패턴이 있다.

#### 자음 겹침 + e

| 남성형 | 여성형 | 패턴 |
|---|---|---|
| bon | bon**ne** | -n -> -nne |
| ancien | ancien**ne** | -n -> -nne |
| gros | gros**se** | -s -> -sse |
| bas | bas**se** | -s -> -sse |
| muet | muet**te** | -t -> -tte |
| gentil | gentil**le** | -l -> -lle |

#### 어미 변환형

| 남성형 | 여성형 | 패턴 |
|---|---|---|
| heureux | heureu**se** | -eux -> -euse |
| sérieux | sérieu**se** | -eux -> -euse |
| sportif | sporti**ve** | -if -> -ive |
| actif | acti**ve** | -if -> -ive |
| premier | premiè**re** | -er -> -ère |
| léger | légè**re** | -er -> -ère |
| blanc | blan**che** | -c -> -che |
| sec | sè**che** | -c -> -che |
| long | lon**gue** | -g -> -gue |

#### 완전 불규칙형 (암기 대상)

| 남성형 | 여성형 | 뜻 |
|---|---|---|
| beau (bel) | belle | 아름다운 |
| nouveau (nouvel) | nouvelle | 새로운 |
| vieux (vieil) | vieille | 오래된 |
| fou (fol) | folle | 미친 |
| mou (mol) | molle | 부드러운 |

괄호 안의 형태(bel, nouvel, vieil 등)는 모음이나 무성 h 앞의 남성 단수에서 사용한다: un bel homme, un nouvel ami, un vieil arbre.

#### 불규칙 복수형

| 단수 | 복수 | 패턴 |
|---|---|---|
| beau | beau**x** | -eau -> -eaux |
| nouveau | nouveau**x** | -eau -> -eaux |
| normal | normau**x** | -al -> -aux |
| national | nationau**x** | -al -> -aux |

### 4. 관사가 없어도 일치는 유지된다

이것은 성수 일치가 관사에 의존하는 것이 아니라 명사 자체의 속성임을 보여준다.

- beaucoup de livres intéressant**s** (관사 없이도 남성 복수 일치)
- beaucoup de tables intéressante**s** (관사 없이도 여성 복수 일치)

### 5. 형용사가 여러 개인 경우

모든 형용사가 각각 명사의 성수에 일치한다.

- une **grande** table **ronde** (여성 단수 두 번 일치)
- des livres **anciens** **intéressants** (남성 복수 두 번 일치)

### 6. 혼성 복수의 일치 규칙

남성 명사와 여성 명사가 함께 올 때, 형용사는 남성 복수형을 취한다.

- Paul et Marie sont content**s**. (남성 + 여성 -> 남성 복수)
- Les garçons et les filles sont intelligent**s**. (남성 복수 우선)

## 비교: 한국어 / 영어 / 다른 로망스어

### 한국어와의 비교

한국어 형용사는 성수 일치가 없다. "작은 고양이"든 "작은 집"이든 "작은"의 형태는 동일하다. 따라서 프랑스어의 형용사 일치는 한국어 화자에게 완전히 새로운 개념이다.

### 영어와의 비교

영어 형용사도 성수에 따라 변하지 않는다. a small cat / a small house / small houses -- 형태 변화 없음. 프랑스어를 영어 감각으로 접근하면 일치를 잊기 쉽다.

### 스페인어

프랑스어와 거의 동일한 체계. 형용사가 성수에 일치한다.
- un gato pequen**o** / una casa pequen**a** / unas casas pequen**as**
- 차이점: 스페인어는 남성 -o, 여성 -a 패턴이 더 규칙적이다.

### 이탈리아어

마찬가지로 성수 일치가 작동한다.
- un gatto piccol**o** / una casa piccol**a** / delle case piccol**e**
- 복수 어미가 -i(남성), -e(여성)로 프랑스어와 다르다.

### 독일어

성(3개: 남성/여성/중성) + 수 + 격에 따라 형용사가 변하며, 프랑스어보다 더 복잡하다.

요약: 프랑스어의 형용사 일치는 로망스어권에서는 보편적이고, 영어/한국어와 비교하면 추가 학습이 필요한 영역이다.

## 한국어 화자 주의점

### 1. 일치 자체를 잊는 실수

한국어에 형용사 일치 개념이 없으므로, 형용사를 쓸 때 성수를 확인하는 습관이 없다. 모든 형용사를 쓸 때 "명사의 성은? 수는?"을 먼저 확인하는 절차를 습관화해야 한다.

- (틀림) une table intéressant
- (맞음) une table intéressant**e**

### 2. 여성형 -e 추가 시 발음 변화를 놓치는 실수

-e를 추가하면 이전에 묵음이었던 자음이 발음된다. 이 발음 변화가 성 구분의 핵심 단서다.

- petit /pti/ -> petite /ptit/ (t가 발음됨)
- grand /ɡʁɑ̃/ -> grande /ɡʁɑ̃d/ (d가 발음됨)
- content /kɔ̃.tɑ̃/ -> contente /kɔ̃.tɑ̃t/ (t가 발음됨)

### 3. 불규칙 패턴을 개별 암기하는 비효율

불규칙처럼 보이지만 대부분 체계적 패턴이다. -eux/-euse, -if/-ive, -er/-ère 같은 패턴을 그룹으로 기억하면 효율적이다.

### 4. 혼성 복수에서 여성형을 쓰는 실수

남성 + 여성이 함께 올 때 남성 복수가 우선한다는 규칙을 잊기 쉽다.

## 어원과 역사

프랑스어 형용사 일치는 라틴어에서 직접 계승되었다. 라틴어는 형용사가 명사의 성(남/여/중), 수(단/복), 격(주격/목적격 등 6개)에 모두 일치했다. 프랑스어는 이 중 격 변화를 잃었지만 성수 일치는 유지했다.

라틴어 bonus, bona, bonum(좋은)이 프랑스어 bon, bonne으로 변한 과정에서, 중성이 남성에 흡수되고 격 어미가 탈락했지만, 성수 구분은 살아남았다. 이는 관사 체계 발달과 함께 성수 정보를 다중으로 표시하려는 프랑스어의 경향을 보여준다.

beau/bel/belle 같은 이중 남성형은 고프랑스어(ancien français)에서 모음 앞 형태가 별도로 보존된 흔적이다. 라틴어 bellus에서 왔으며, 자음 앞과 모음 앞에서 음운 변화가 다르게 진행된 결과다.

## 핵심 정리

### 기본 규칙 요약표

| 변환 | 규칙 | 예시 |
|---|---|---|
| 남성 -> 여성 | -e 추가 | grand -> grande |
| 단수 -> 복수 | -s 추가 | grand -> grands |
| 남성 단수 -> 여성 복수 | -es 추가 | grand -> grandes |

### 주요 불규칙 패턴 요약

| 패턴 | 남성 | 여성 | 예시 |
|---|---|---|---|
| 자음 겹침 | -on, -el, -et | -onne, -elle, -ette | bon/bonne |
| 어미 변환 | -eux, -if, -er | -euse, -ive, -ère | heureux/heureuse |
| 완전 불규칙 | beau, nouveau, vieux | belle, nouvelle, vieille | -- |
| 남녀 동형 | -e로 끝남 | 변화 없음 | jeune/jeune |

### 핵심 원칙

- 형용사는 명사의 성수에 반드시 일치한다.
- 정보의 출처는 명사이고, 형용사는 수동적으로 반영한다.
- 관사 유무와 무관하게 일치가 작동한다.
- 형용사가 여러 개면 전부 일치한다.
- 혼성 복수에서는 남성 복수가 우선한다.

## 연습 문제

### 문제 1: 빈칸 채우기

다음 빈칸에 형용사의 올바른 형태를 넣으시오.

1. une femme ________ (intelligent)
2. des livres ________ (nouveau)
3. une ________ maison (beau)
4. les filles ________ (heureux)
5. un ________ arbre (vieux)

**답과 해설:**

1. **intelligente** -- 여성 단수이므로 -e 추가.
2. **nouveaux** -- 남성 복수이므로 -eau -> -eaux.
3. **belle** -- 여성 단수이므로 beau -> belle.
4. **heureuses** -- 여성 복수이므로 -eux -> -euses.
5. **vieil** -- 모음으로 시작하는 남성 단수 명사 앞이므로 vieux -> vieil.

### 문제 2: 한국어를 프랑스어로 번역

1. 긴 거리 (rue = 여성)
2. 아름다운 꽃들 (fleur = 여성)
3. 재미있는 영화 (film = 남성)
4. 행복한 아이들 (enfant = 남성)
5. 새로운 친구 (여성) (amie = 여성)

**답과 해설:**

1. **une longue rue** -- long의 여성형은 longue. 전치사적 형용사.
2. **de belles fleurs** -- beau의 여성 복수형은 belles.
3. **un film intéressant** -- 남성 단수, 기본형 그대로.
4. **des enfants heureux** -- 남성 복수, -eux는 복수에서도 변하지 않음.
5. **une nouvelle amie** -- nouveau의 여성형은 nouvelle.

### 문제 3: 오류 교정

다음 문장에서 틀린 부분을 찾아 고치시오.

1. Elle est intelligent.
2. Des tables blanc.
3. Un beau histoire.
4. Les garçons et les filles sont contentes.
5. Une situation sérieux.

**답과 해설:**

1. Elle est intelligent**e**. -- elle(여성)에 일치해야 하므로 -e 추가.
2. Des tables blanc**hes**. -- table은 여성 복수, blanc -> blanche -> blanches.
3. Une bell**e** histoire. -- histoire는 여성이므로 beau -> belle. 또한 전치 위치.
4. Les garçons et les filles sont content**s**. -- 혼성 복수에서는 남성 복수 우선.
5. Une situation sérieu**se**. -- situation은 여성이므로 -eux -> -euse.

### 문제 4: 문장 완성

주어진 요소로 문장을 완성하시오.

1. (Marie / être / sportif) ->
2. (les idées / être / bon) ->
3. (un / vieux / ami / fidèle) ->

**답과 해설:**

1. **Marie est sportive.** -- 여성이므로 sportif -> sportive.
2. **Les idées sont bonnes.** -- 여성 복수이므로 bon -> bonnes.
3. **Un vieil ami fidèle.** -- 모음 앞 남성 단수이므로 vieux -> vieil, fidèle은 남녀 동형.

### 문제 5: 선택형

올바른 형태를 고르시오.

1. une robe (blanc / blanche / blancs)
2. des hommes (vieux / vieille / vieilles)
3. un (nouvel / nouveau / nouvelle) appartement

**답과 해설:**

1. **blanche** -- robe는 여성 단수.
2. **vieux** -- hommes는 남성 복수, vieux는 남성 단복수 동형.
3. **nouvel** -- appartement은 모음으로 시작하는 남성 단수 명사.
