# 2-2. 관사 I -- 정관사 / 부정관사

> 프랑스어 명사는 왜 거의 항상 관사와 함께 등장하는가?

## 핵심 개념

**공리**: 프랑스어에서 명사는 원칙적으로 **관사와 함께** 쓰인다. 관사는 명사의 성/수 정보를 외부에 표시하고, 화자와 청자 사이의 정보 상태(특정/불특정/일반)를 조절하는 장치다.

## 상세 설명

### 1. 관사는 왜 필요한가

흔한 오해: "관사 = 영어의 a/the를 번역하는 단어"

이것은 틀렸다. 관사의 진짜 역할은 다음과 같다:

> 관사는 명사를 문장 안에서 **'참조 가능한 대상'으로 만드는 장치**다.

프랑스어에서 명사 단독으로는 문장에 등장할 수 없다:
- *Livre est intéressant.* -- 비문
- ***Le** livre est intéressant.* -- 정문

관사가 명사에 부여하는 기능:
1. **성/수 표시**: 명사의 문법적 성과 수를 외부에 드러낸다 (2-1과 직접 연결)
2. **참조 지정**: 청자가 어떤 대상을 떠올려야 하는지를 지시한다
3. **담화 기능**: 새 정보(부정관사) vs. 구 정보(정관사)를 구분한다

### 2. 부정관사 (article indéfini)

#### 형태

| 성/수 | 형태 | 발음 |
|-------|------|------|
| 남성 단수 | **un** | /ɛ̃/ |
| 여성 단수 | **une** | /yn/ |
| 복수 | **des** | /de/ |

#### 핵심 의미

> un / une / des = "청자에게 아직 특정되지 않은 대상"

부정관사는 다음 상황에서 사용한다:
- **처음 등장**: 담화에서 아직 언급되지 않은 대상을 도입할 때
- **하나의 예시**: 종류 중 하나를 가리킬 때
- **존재 도입**: "~가 있다"라는 맥락

예문:
- *J'ai **un** livre.* /ʒe ɛ̃ livʁ/ -- 어떤 책 하나 (특정되지 않음)
- *Elle a **une** idée.* /ɛl a yn i.de/ -- 막 떠오른 아이디어
- *J'ai acheté **des** pommes.* /ʒe aʃ.te de pɔm/ -- 몇 개의 사과

### 3. 정관사 (article défini)

#### 형태

| 성/수 | 형태 | 발음 |
|-------|------|------|
| 남성 단수 | **le** | /lə/ |
| 여성 단수 | **la** | /la/ |
| 모음 앞 단수 | **l'** | /l/ |
| 복수 | **les** | /le/ |

**축약형 (contraction)**: 전치사 à / de와 결합할 때

| | à + | de + |
|--|-----|------|
| le | **au** /o/ | **du** /dy/ |
| les | **aux** /o/ | **des** /de/ |
| la | à la | de la |
| l' | à l' | de l' |

#### 핵심 의미

> le / la / les = "청자와 공유되는, 특정 가능한 대상"

정관사는 다음 상황에서 사용한다:

**(1) 이미 언급된 대상 (재지시/anaphore)**
- *J'ai un chien. **Le** chien est gentil.* -- 앞에서 도입한 개를 다시 가리킨다

**(2) 맥락으로 특정되는 대상**
- *Ferme **la** porte.* /fɛʁm la pɔʁt/ -- (이 방의) 그 문을 닫아라

**(3) 유일한 대상**
- ***Le** soleil brille.* /lə sɔ.lɛj bʁij/ -- 태양은 하나뿐이므로 정관사

**(4) 일반화 (매우 중요)**
- ***Le** chat aime **le** lait.* /lə ʃa ɛm lə lɛ/ -- 고양이(종 전체)는 우유를 좋아한다

이 네 번째 용법이 한국어/영어 화자에게 가장 낯선 부분이다.

### 4. 특정 vs. 불특정: 정확한 이해

이것은 현실 세계에서의 존재 유무 문제가 아니라, **담화 흐름(discourse flow)** 기준이다.

```
J'ai un chien.       (개의 존재를 도입 -- 부정관사)
Le chien est gentil.  (이미 도입된 대상을 재지시 -- 정관사)
```

첫 문장에서 '개'는 청자에게 새로운 정보이므로 부정관사를 쓴다. 두 번째 문장에서는 이미 공유된 정보이므로 정관사를 쓴다.

### 5. 정관사의 일반화 용법 (한국어/영어와의 핵심 차이)

프랑스어에서 일반적 진술(generic statement)에는 **정관사**를 쓴다:

- *J'aime **le** café.* /ʒɛm lə ka.fe/ -- 커피(라는 것)를 좋아한다
- ***Les** enfants aiment **le** chocolat.* -- 아이들(일반)은 초콜릿을 좋아한다
- ***La** musique adoucit **les** mœurs.* -- 음악은 풍속을 부드럽게 한다

이것은 영어와 크게 다르다:
- 영어: *I like **coffee**.* (관사 없음) / *Children like **chocolate**.* (관사 없음)
- 프랑스어: *J'aime **le** café.* / *Les enfants aiment **le** chocolat.* (정관사 사용)

프랑스어는 '일반적 범주'를 말할 때도 정관사를 쓴다. 관사 없이 명사만 쓰는 것은 허용되지 않는다.

### 6. 관사는 성을 '보이게' 만든다

2-1에서 명사의 성은 사전에 고정되어 있다고 했다. 그 성이 문장에서 표면에 드러나는 주된 수단이 관사다.

- *un livre* / *une table* -- 부정관사가 성을 드러낸다
- *le livre* / *la table* -- 정관사가 성을 드러낸다

관사가 없으면 성이 표면에 드러나지 않고, 형용사/대명사의 일치 연쇄도 끊어진다.

### 7. 관사 생략이 가능한 경우 (예고)

프랑스어에서 관사 생략은 예외적이다. 현 단계에서는 원칙만 알아둔다:

- 직업/신분 (être 뒤): *Il est médecin.* /il ɛ med.sɛ̃/ (그는 의사다)
- 일부 고정 표현: *avoir faim* (배고프다), *avoir soif* (목마르다)
- 전치사 + 명사 구: *en France* (프랑스에서), *par exemple* (예를 들어)

상세 규칙은 2-4와 이후 단원에서 다시 다룬다.

## 비교: 한국어 / 영어 / 다른 로망스어

### 한국어와의 차이

한국어에는 관사가 없다. 특정/불특정의 구분은 지시어(이/그/저)나 문맥으로 처리한다.

| 기능 | 한국어 | 프랑스어 |
|------|--------|----------|
| 불특정 | "책 한 권" / "어떤 책" | *un livre* |
| 특정 | "그 책" | *le livre* |
| 일반화 | "고양이는..." | *Le chat...* |
| 관사 필수 여부 | 없음 | 필수 |

한국어 화자가 가장 어려워하는 것: **모든 명사 앞에 관사를 붙여야 한다는 것**. 한국어에서는 아무 표지 없이 "책이 재미있다"라고 말할 수 있지만, 프랑스어에서는 *Le livre est intéressant.* 처럼 반드시 관사가 필요하다.

### 영어와의 비교

| | 영어 | 프랑스어 |
|--|------|----------|
| 부정관사 | a / an (성 구분 없음) | un / une (성 구분) |
| 정관사 | the (성/수 구분 없음) | le / la / les (성/수 구분) |
| 일반화 | 무관사 (*I like coffee.*) | 정관사 (*J'aime le café.*) |
| 복수 일반 | 무관사 (*Dogs are loyal.*) | 정관사 (*Les chiens sont fidèles.*) |
| 관사 생략 | 비교적 자유 | 매우 제한적 |

핵심 차이: 영어에서 일반적 진술에 관사를 쓰지 않는 경우(zero article)가 프랑스어에서는 정관사로 표현된다. 이것은 영어와 프랑스어의 관사 체계에서 가장 중요한 차이 중 하나다.

### 다른 로망스어와의 비교

| | 프랑스어 | 스페인어 | 이탈리아어 |
|--|----------|----------|------------|
| 부정관사 남단 | un | un | un/uno |
| 부정관사 여단 | une | una | una/un' |
| 정관사 남단 | le | el | il/lo |
| 정관사 여단 | la | la | la |
| 정관사 복수 | les | los/las | i/gli/le |
| 일반화에 정관사 | 사용 | 사용 | 사용 |

로망스어들은 공통적으로 일반화에 정관사를 사용한다. 이것은 라틴어의 지시대명사 *ille*에서 유래한 정관사가 로망스어 전체에서 동일한 기능으로 발전했기 때문이다.

차이점: 스페인어/이탈리아어에서는 복수 정관사가 성에 따라 구분되지만 (los/las, i/le), 프랑스어에서는 복수 정관사 *les*가 성 구분 없이 하나만 사용된다.

## 한국어 화자 주의점

1. **관사 누락**: 한국어에 관사가 없으므로, 프랑스어에서도 관사를 빠뜨리는 실수가 가장 잦다. 특히 단독으로 쓰이는 명사 앞에서: *Chat est mignon.* (비문) - *Le chat est mignon.* (정문)

2. **일반화에서 관사 선택 실수**: 영어식으로 관사 없이 일반화하려는 경향. *J'aime café.* (비문) - *J'aime **le** café.* (정문). 프랑스어에서 일반적 범주도 정관사를 요구한다.

3. **부정관사/정관사 혼동**: "그 책"이라고 해석할 수 없는데도 정관사를 쓰거나, "어떤 책"인데 부정관사를 잊는 경우. 핵심: 처음 등장 = un/une, 이미 알려진 것 = le/la.

4. **des의 혼동**: 복수 부정관사 *des*와 전치사 *de* + 정관사 *les*의 축약형 *des*가 형태가 같다. *J'ai des amis.* (부정관사, 친구들이 있다) vs. *Je parle des enfants.* (de + les, 아이들에 대해 말한다). 문맥으로 구분해야 한다.

5. **축약형(au, du, aux, des) 누락**: *à + le*를 *à le*로 쓰는 오류. *Je vais à le marché.* (비문) - *Je vais **au** marché.* (정문). 축약은 필수이며 선택이 아니다.

## 어원과 역사

### 관사의 탄생

라틴어에는 관사가 없었다. *Liber est bonus.* (책은 좋다) -- 관사 없이 명사가 문장에 등장했다.

### 정관사의 기원

정관사는 라틴어의 지시대명사 **ille, illa, illud** (그, 그녀, 그것)에서 유래했다.

```
ille  →  (il)le  →  le   (남성 단수)
illa  →  (il)la  →  la   (여성 단수)
illos →  (il)los →  les  (복수)
```

지시 의미("그")가 약화되면서, 단순히 "특정된 대상"을 가리키는 관사로 문법화(grammaticalisation)되었다.

이 과정은 모든 로망스어에서 동일하게 일어났다:
- 프랑스어: le / la
- 스페인어: el / la
- 이탈리아어: il(lo) / la
- 포르투갈어: o / a

### 부정관사의 기원

부정관사는 라틴어의 수사 **unus, una** (하나)에서 유래했다.

```
unus  →  un   (남성)
una   →  une  (여성)
```

'하나'라는 수적 의미가 약화되면서 '불특정한 하나'를 가리키는 관사로 발전했다. 복수 부정관사 *des*는 전치사 *de* + 정관사 *les*의 축약에서 문법화된 것이다.

### 관사 체계의 확립

고프랑스어(9-13세기)에서 관사 사용은 아직 불규칙적이었다. 중프랑스어(14-16세기)를 거치며 관사 사용이 의무화되었고, 17세기에 현대적 관사 체계가 완성되었다.

## 핵심 정리

프랑스어 관사 체계 종합표:

| | 남성 단수 | 여성 단수 | 모음 앞 | 복수 |
|--|-----------|-----------|---------|------|
| **부정관사** | un | une | un/une | des |
| **정관사** | le | la | l' | les |

관사 선택 기준:

```
명사를 사용할 때:
  처음 등장 / 불특정 / 존재 도입  →  부정관사 (un / une / des)
  이미 알려진 / 특정 / 유일      →  정관사 (le / la / les)
  일반적 범주 / 총칭              →  정관사 (le / la / les)
```

전치사 축약:

```
à + le = au       de + le = du
à + les = aux     de + les = des
à + la = à la     de + la = de la
à + l' = à l'     de + l' = de l'
```

## 연습 문제

**1. 빈칸에 적절한 관사를 넣으시오: *J'ai ______ sœur et ______ frère.*

> **답**: *J'ai **une** sœur et **un** frère.*
> **해설**: 처음 등장하는 불특정 대상이므로 부정관사. *sœur*는 여성(une), *frère*는 남성(un).

**2. 다음 문장이 올바른지 판단하시오: *J'aime café.*

> **답**: 비문. *J'aime **le** café.*가 올바르다.
> **해설**: 일반적 범주(커피 전체)에 대한 취향을 말할 때도 프랑스어에서는 정관사가 필요하다.

**3. 부정관사와 정관사를 사용하여 두 문장을 만드시오: "어떤 남자가 들어왔다. 그 남자는 키가 크다."**

> **답**: ***Un** homme est entré. **L'**homme est grand.* /ɛ̃.n‿ɔm ɛ.t‿ɑ̃.tʁe. lɔm ɛ ɡʁɑ̃/
> **해설**: 첫 문장에서 부정관사(un)로 도입하고, 두 번째 문장에서 정관사(l')로 재지시한다.

**4. *à + le marché*를 올바르게 축약하시오.**

> **답**: ***au** marché* /o maʁ.ʃe/
> **해설**: 전치사 *à* + 정관사 *le*는 반드시 *au*로 축약한다. *à le*는 비문이다.

**5. 다음 문장을 프랑스어로 번역하시오: "아이들은 놀기를 좋아한다." (일반적 진술)**

> **답**: ***Les** enfants aiment jouer.* /le.z‿ɑ̃.fɑ̃ ɛm ʒu.e/
> **해설**: 일반적 범주(아이들 전체)에 대한 진술이므로 정관사 *les*를 사용한다. 영어에서는 *Children like to play.*로 관사 없이 말하지만, 프랑스어에서는 정관사 필수.

**6. *des*가 부정관사인지, 전치사+정관사인지 구분하시오.**
a) *J'ai des amis.*
b) *Je parle des enfants.*

> **답**: a) 부정관사 des (친구들을 가지고 있다 -- 불특정 복수), b) de + les = des (그 아이들에 대해 말한다 -- 특정 대상)
> **해설**: a)에서 *des amis*는 "몇몇 친구들"(부정관사), b)에서 *des enfants*는 *de les enfants*의 축약으로 "그 아이들에 대해"(전치사+정관사).
