# 2-1. 성(gender)의 개념

> 프랑스어에서 명사는 왜 반드시 남성/여성 중 하나를 가져야 하는가?

## 핵심 개념

**공리**: 프랑스어의 모든 명사는 **문법적 성(genre grammatical)**을 가진다. 이 성은 의미에서 오는 경우도 있고, 의미와 무관한 경우도 있으며, **문장 구조를 작동시키는 필수 정보**다.

성은 관사, 형용사, 대명사, 과거분사 일치의 기준이 된다. 성이 없으면 프랑스어 문장은 조립 자체가 불가능하다.

## 상세 설명

### 1. 문법적 성이란 무엇인가

두 종류의 '성'을 명확히 구분해야 한다.

#### 의미적 성 (genre sémantique / natural gender)

실제 생물학적 성별을 반영하는 경우:
- *un homme* /ɛ̃.n‿ɔm/ (남자) -- 남성
- *une femme* /yn fam/ (여자) -- 여성
- *un chat* /ɛ̃ ʃa/ (수고양이) -- 남성
- *une chatte* /yn ʃat/ (암고양이) -- 여성

#### 문법적 성 (genre grammatical / grammatical gender)

문법 시스템이 요구하는 분류로, 실제 성별과 무관하다:
- *une table* /yn tabl/ (테이블) -- 여성
- *un livre* /ɛ̃ livʁ/ (책) -- 남성
- *une voiture* /yn vwa.tyʁ/ (자동차) -- 여성
- *un problème* /ɛ̃ pʁɔ.blɛm/ (문제) -- 남성

테이블과 책에는 성별이 없지만, 문법적으로는 반드시 하나를 가진다. 이것이 문법적 성의 본질이다.

### 2. 왜 성이 필요한가

성은 의미 전달이 아니라 **문장 내부 연결 장치(mécanisme d'accord)**로 기능한다. 성은 명사와 다른 요소를 연결하는 일치 기준(anchor) 역할을 한다.

#### 관사와의 일치

- *un livre* / *une table*
- *le livre* / *la table*

#### 형용사와의 일치

- *un livre **intéressant*** / *une table **intéressante***
- *un petit chat* / *une petite chatte*

#### 대명사와의 일치

- *Je **le** vois.* (le = 남성 대상) / *Je **la** vois.* (la = 여성 대상)

#### 과거분사와의 일치

- *Elle est **arrivée**.* (여성이므로 -e 추가)
- *Il est **arrivé**.* (남성이므로 변화 없음)

성이 없으면 관사 선택, 형용사 일치, 대명사 대체, 과거분사 일치 모두 불가능해진다.

### 3. 성은 언제 결정되는가

명사의 성은 문장 안에서 결정되는 것이 아니라, **사전에 고정(fixe dans le lexique)**되어 있다. 따라서 명사를 배울 때 의미와 성을 함께 배워야 한다.

- *table* 만 외우면 불완전하다
- ***une** table* 로 외워야 한다

이것은 프랑스어 학습의 가장 기본적인 원칙 중 하나다.

### 4. 성을 예측할 수 있는가

부분적으로 가능하지만, 완벽하지는 않다.

#### 어미 기반 경향 (확률적 지표)

**여성 경향이 강한 어미:**

| 어미 | 예시 | 의미 |
|------|------|------|
| -tion / -sion | *la nation, la décision* | 국가, 결정 |
| -ité | *la qualité, la cité* | 품질, 도시 |
| -ure | *la nature, la culture* | 자연, 문화 |
| -ence / -ance | *la science, la chance* | 과학, 기회 |
| -ée | *la journée, l'année* | 하루, 해 |
| -ette | *la fourchette, la cassette* | 포크, 카세트 |
| -esse | *la politesse, la richesse* | 예절, 부유함 |

**남성 경향이 강한 어미:**

| 어미 | 예시 | 의미 |
|------|------|------|
| -age | *le village, le voyage* | 마을, 여행 |
| -ment | *le mouvement, le moment* | 움직임, 순간 |
| -isme | *le tourisme, le réalisme* | 관광, 사실주의 |
| -eur | *le bonheur, le malheur* | 행복, 불행 |
| -ier | *le cahier, le papier* | 공책, 종이 |
| -eau | *le chapeau, le gâteau* | 모자, 케이크 |

**주의: 예외가 많다.** *une page* (여성, -age로 끝남), *une image* (여성, -age로 끝남), *la mer* (여성, 짧은 어미). 이 표는 경향일 뿐, 규칙이 아니다.

### 5. 특수한 경우

#### 성에 따라 의미가 달라지는 명사

| 남성 | 의미 | 여성 | 의미 |
|------|------|------|------|
| *le livre* | 책 | *la livre* | 파운드 (화폐/무게) |
| *le tour* | 차례, 둘레 | *la tour* | 탑 |
| *le poste* | 직위 | *la poste* | 우체국 |
| *le mode* | 방식 | *la mode* | 유행 |
| *le vase* | 꽃병 | *la vase* | 진흙 |

이런 단어들은 성이 의미를 구분하는 기능을 하므로, 관사 없이 외우면 혼동이 생긴다.

#### 양성 명사 (noms épicènes)

일부 명사는 남녀 모두에 사용되며, 관사로만 성을 구분한다:
- *un/une élève* (학생)
- *un/une artiste* (예술가)
- *un/une journaliste* (기자)

#### 직업 명사의 여성형 (현대적 변화)

전통적으로 남성형만 있던 직업 명사에 여성형이 만들어지는 추세다:
- *un professeur* - *une professeure* (교수)
- *un auteur* - *une auteure / autrice* (저자)
- *un médecin* - *une médecin / médecine* (의사, 여전히 논쟁 중)

## 비교: 한국어 / 영어 / 다른 로망스어

### 한국어와의 차이

한국어에는 문법적 성이 없다. 명사에 성별 정보가 부여되지 않으며, 관사도 없으므로 성에 따른 일치 현상이 존재하지 않는다.

| | 한국어 | 프랑스어 |
|--|--------|----------|
| 문법적 성 | 없음 | 남성/여성 |
| 관사 | 없음 | 성에 따라 변화 |
| 형용사 일치 | 없음 | 성에 따라 변화 |
| 명사 학습 | 의미만 | 의미 + 성 |

한국어 화자에게 문법적 성은 완전히 새로운 개념이다. '왜 테이블이 여성이고 책이 남성인가'라는 질문에 논리적 답은 없다. 역사적 관습으로 받아들여야 한다.

### 영어와의 비교

영어는 고대 영어 시기에 3성 체계(남/여/중)를 가졌으나, 중세 영어를 거치며 소멸했다. 현대 영어에서 성은 대명사(he/she/it)에만 남아 있고, 명사와 형용사에는 성 구분이 없다.

| | 영어 | 프랑스어 |
|--|------|----------|
| 명사의 성 | 없음 | 남성/여성 |
| 관사 | the/a (성 무관) | le/la, un/une (성 반영) |
| 형용사 | 불변 (big) | 성에 따라 변화 (grand/grande) |
| 대명사 | he/she/it | il/elle |

### 다른 로망스어와의 비교

| 언어 | 성 체계 | 특이사항 |
|------|---------|----------|
| 프랑스어 | 남성/여성 (2성) | 라틴어 중성이 남성에 흡수 |
| 스페인어 | 남성/여성 (2성) | -o 남성, -a 여성 경향 강함 |
| 이탈리아어 | 남성/여성 (2성) | -o 남성, -a 여성 경향 강함 |
| 포르투갈어 | 남성/여성 (2성) | 스페인어와 유사 |
| 루마니아어 | 남성/여성/중성 (3성) | 로망스어 중 유일하게 중성 보존 |
| 독일어 (참고) | 남성/여성/중성 (3성) | 인도유럽어족 3성 체계 유지 |

스페인어와 이탈리아어는 어미로 성을 예측하기가 프랑스어보다 쉽다. -o로 끝나면 대부분 남성, -a로 끝나면 대부분 여성이다. 프랑스어는 어미의 성 예측력이 상대적으로 낮아, 개별 암기의 비중이 더 크다.

## 한국어 화자 주의점

1. **성의 존재 자체를 잊는 것**: 한국어에 문법적 성이 없으므로, 명사를 배울 때 성 정보를 빠뜨리기 쉽다. 새 명사를 배울 때 반드시 **관사(un/une 또는 le/la)와 함께** 외우는 습관을 들여야 한다.

2. **의미로 성을 추측하려는 것**: "꽃은 예쁘니까 여성이겠지", "문제는 딱딱하니까 남성이겠지" 같은 추측은 위험하다. *la fleur*(꽃)는 여성이 맞지만, *le problème*(문제)가 남성인 것에 논리적 이유는 없다.

3. **형용사 일치를 잊는 것**: "재미있는 책" = *un livre intéressant*, "재미있는 영화" = *un film intéressant*이지만, "재미있는 이야기" = *une histoire intéressant**e***. 형용사 어미가 명사의 성에 따라 바뀐다는 것을 놓치기 쉽다 (3장에서 상세히 다룬다).

4. **성이 다른 동음이의어 혼동**: *le tour*(차례) vs. *la tour*(탑)처럼 성만 다르고 발음이 같은 단어를 혼동하기 쉽다. 관사를 정확히 써야 의미가 구분된다.

## 어원과 역사

### 인도유럽어의 3성 체계

인도유럽조어(Proto-Indo-European)는 남성/여성/중성의 3성 체계를 가졌다. 이 체계는 산스크리트어, 그리스어, 라틴어, 게르만어 등에 전승되었다.

### 라틴어의 3성

라틴어에는 남성(masculinum), 여성(femininum), 중성(neutrum)이 있었다.
- *dominus* (주인, 남성)
- *regina* (여왕, 여성)
- *templum* (신전, 중성)

### 프랑스어에서 중성의 소멸

속라틴어(latin vulgaire)에서 중성 명사는 점차 남성 또는 여성에 흡수되었다.

중성 소멸의 주요 원인:
- 중성 복수형 -a가 여성 단수형 -a와 형태가 같았다
- 격변화 체계 붕괴와 함께 성 구분의 형태적 근거도 약화

결과적으로:
- 중성의 대부분이 **남성으로 흡수**되었다
- 일부는 여성으로 흡수 (*la mer* < *mare*, 원래 중성)
- 남성이 '기본값(default)'으로 기능하게 되었다

이 '남성 = 기본값'의 경향은 현대 프랑스어에서도 나타난다:
- 혼합 성별 집단은 남성 복수로 표현: *les étudiants* (남녀 학생들)
- 불특정 주어는 남성: *on est content* (남성형 형용사)

다만 이 관행은 현대에 논쟁의 대상이 되고 있으며, 포용적 표기(écriture inclusive)를 사용하는 움직임도 있다.

## 핵심 정리

프랑스어 명사 성의 핵심:

| 항목 | 내용 |
|------|------|
| 성의 종류 | 남성(masculin) / 여성(féminin) |
| 성의 성격 | 문법적 분류 (의미와 무관할 수 있음) |
| 성의 기능 | 관사, 형용사, 대명사, 과거분사의 일치 기준 |
| 성의 결정 시점 | 사전에 고정 (문장 내에서 결정되지 않음) |
| 성의 예측 | 어미로 부분적 예측 가능, 예외 많음 |
| 학습 원칙 | 명사 + 관사(un/une)를 함께 암기 |

관사를 통한 성 표시:

| | 남성 | 여성 |
|--|------|------|
| 부정관사 단수 | un | une |
| 정관사 단수 | le | la |
| 정관사 (모음 앞) | l' | l' |
| 복수 (성 구분 없음) | les / des | les / des |

## 연습 문제

**1. 다음 명사의 성을 관사와 함께 쓰시오: table, livre, nation, village, liberte**

> **답**: *une table* (여성), *un livre* (남성), *une nation* (여성), *un village* (남성), *la liberté* (여성)
> **해설**: -tion 어미는 여성 경향, -age 어미는 남성 경향, -té 어미는 여성 경향. 하지만 개별 암기가 기본이다.

**2. 다음 문장의 오류를 교정하시오: *Un table est grande.*

> **답**: ***Une** table est grande.*
> **해설**: *table*은 여성 명사이므로 부정관사 *un*이 아니라 *une*을 사용해야 한다.

**3. *le tour*와 *la tour*의 의미 차이를 설명하시오.**

> **답**: *le tour* (남성) = 차례, 둘레. *la tour* (탑, 여성) = 탑. 같은 발음이지만 성에 따라 의미가 다른 동음이의어다.

**4. 왜 *Livre est intéressant.*이 비문인지 설명하시오.**

> **답**: 프랑스어에서 명사는 원칙적으로 관사와 함께 쓰인다. 관사 없이 명사만 사용하면 성/수 정보가 표면에 드러나지 않고, 문장 내 일치 관계가 성립하지 않는다. *Le livre est intéressant.*이 올바르다. (관사에 대해서는 2-2에서 상세히 다룬다.)

**5. 다음 형용사를 명사의 성에 맞게 변형하시오.**
a) *un gâteau (délicieux)*
b) *une maison (grand)*
c) *un problème (important)*

> **답**: a) *un gâteau délicieux*, b) *une maison grande*, c) *un problème important*
> **해설**: 여성 명사 앞에서 형용사에 -e가 추가된다. *grand* - *grande*. 남성 명사 앞에서는 기본형 유지. (형용사 일치의 상세 규칙은 3장에서 다룬다.)

**6. 프랑스어에서 중성이 소멸한 역사적 이유를 간략히 설명하시오.**

> **답**: 라틴어의 중성 복수 어미 -a가 여성 단수 어미 -a와 형태적으로 동일해지면서, 중성과 여성/남성의 구분이 모호해졌다. 격변화 체계의 붕괴와 함께 중성은 대부분 남성에 흡수되었다.
