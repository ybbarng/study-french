## 메타 토론 — 어간 공유

### imparfait + participe présent 어간 공유

**규칙**: 둘 다 nous 현재 - ons에서 출발 → 거의 같은 어간.

| 동사 | nous 현재 | imparfait | participe |
|---|---|---|---|
| faire | faisons | faisait | faisant ⭕ |
| prendre | prenons | prenait | prenant ⭕ |
| voir | voyons | voyait | voyant ⭕ |
| boire | buvons | buvait | buvant ⭕ |
| écrire | écrivons | écrivait | écrivant ⭕ |
| finir (2군) | finissons | finissait | finissant ⭕ |
| **avoir** | avons | av**ait** | **ay**ant (다름!) |
| **savoir** | savons | sav**ait** | **sach**ant (다름!) |
| être | sommes | ét**ait** | ét**ant** ⭕ (특수 어간) |

**예외**: avoir (분사 ayant), savoir (분사 sachant). être는 같은 어간 ét-.

### 보너스 — 접속법 nous/vous도 같은 어간 (대부분)

이중 어간 동사:
- boire 접속법 nous = buvions (buv-)
- prendre 접속법 nous = prenions (pren-)
- 위 동사들은 imparfait/participe/접속법 nous/vous가 모두 같은 어간

특수 어간 동사:
- faire 접속법 = fassions (fass-, 완전 다름)
- aller 접속법 = aillions ❌ → allions (이중 어간, nous/vous만 all-)
- pouvoir 접속법 = puissions (puiss-, 완전 다름)
- savoir 접속법 = sachions (sach-, 완전 다름)

→ 어간 패턴이 동사군별로 다양. 5/24 동사 6변화 체크에서 정리 예정.
