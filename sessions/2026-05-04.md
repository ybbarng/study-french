## 메타 토론 (사용자 깊이 흥미)

### 격 시스템
- 인도유럽어 특유 X — 다양한 가족 (우랄어, 셈어 등)
- 표현 방법: 굴절(라틴어/독일어) / 조사(한국어) / 위치+전치사(영어/프랑스어)
- 프랑스어도 격 잔재 (qui=주격, que=대격, dont=속격, où=장소격) — 분리 단어 형태
- 영어 who/whom/whose = 3격 (대명사만 보존)

### 사용자 도출 통찰 (학습 효율 관점)
- 격 보유 언어 (독일어/러시아어): 일관성 ↑, 학습 곡선 초기 가파름→후 안정
- 격 손실 언어 (영어/프랑스어): 어순 고정 + 예외 多, 후기 가파름
- 사용자 선호 (일관 규칙) → **독일어가 후보로 가치 ↑** (2027 새 언어)

### 로망스어 시너지 (어휘 70~89%)
- 프랑스어 후 스페인어/이탈리아어 = 6개월에 B1 가능
- 다국어 학습자가 로망스어 묶음으로 학습하는 이유

### 동사 굴절 vs 명사 굴절
- 명사 굴절 = 격 + 성수
- 동사 굴절 = 활용 (인칭/수/시제/법/태)
- 분사 = 하이브리드 (형용사 기능 + 동사 기능)

### être + p.p. 세 가지 의미
- 상태 (형용사): La porte est fermée (닫혀 있다)
- 수동 현재: La porte est fermée par Paul (닫고 있다)
- PC (être 동사): Elle est partie (떠났다)
- → 시제 변환 시 단서로 식별 필요 (par/시점부사/지속부사)

### 의문문 변환 원리 (qu/ce qu)
- 명확한 의문사 (où, quand) → 그대로
- 모호한 형태 (없음/que/qui) → 보강 (si/ce que/ce qui)
- 영어 what = 통합형, 프랑스어 ce que = 분리형

### dire 6변화 발음 정리
- 단수 모두 /di/ (어미 -s/-t 묵음)
- nous /diz/ (z), vous /dit/ (t, 예외), ils /diz/ (z)
- → vous만 t, 나머지 z 발음으로 disent vs ditent 구별
