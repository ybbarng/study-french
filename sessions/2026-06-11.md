## 3. 약점 4묶음 드릴

### 묶음 A — être 동사 식별 (3/3 ✅)
- *rester* → sommes restés / *tomber* → est tombée / *devenir* → sont devenus
- 핵심: 이동 아닌 "머묾·됨·넘어짐"도 être. 어제 naître에 이어 정착.

### 묶음 B — 불규칙 과거분사 (회복 + 신규 학습 ✅)
- *courir* → couru (Q19 회복) / *croire* → cru
- 🆕 **vivre → vécu** (신규 학습, 어간 véc-, 라틴 victus). 즉시 재적용 *ont vécu* ⭕
- 사용자 추측 분석: *vu* = voir ⭕ 정확히 구분. *vivu/vivru* ❌

### 묶음 C — 분사구문 시제 (3/3 ✅ + 규칙 학습)
- 선행: *Ayant lu* / 동시: *Souriant* (추론 성공) / Q8 재도전 *Ayant terminé* ⭕
- 🆕 **현재분사 형성 규칙**: nous 현재형 어간 + -ant (souri-ant, finiss-ant). 불규칙 3개: étant/ayant/sachant
- 메타: 사용자 "동사 현재형(nous)이 허브" 자가 도출 ⭐ — 현재분사·반과거·접속법 모두 nous 어간

### 묶음 D — 가정문 3형 조건법 과거 (D-1 ❌ → D-2·D-3 ✅, 자동화 미달)
- D-1 *sois venu* ❌ → **serais venu** (접속법 과거 vs 조건법 과거 조동사 혼동, 5/28 재발)
- D-2 serais venu ⭕ / D-3 aurait réussi ⭕ (회복)
- **판별 스위치**: que+트리거 → 접속법(sois/ait) / Si 결과절 → 조건법(serais/aurais)
- 사용자 진단: "조건법인 줄 알면서 별 생각 없이 접속법 형태가 나온다" — 오늘 워밍업 접속법 과거 다량 연습의 잔상 효과. 다음 세션 가정문 3형 워밍업으로 반사 자동화 필요.

## 프랑스어 표현 교정

| 사용자 | 정답 | 비고 |
|---|---|---|
| il a resté | il est resté | rester = être 동사 |
| Si j'avais... je lirais | Si j'avais eu... j'aurais lu | 3형 가정 (과거 비현실) |
| plus temps | plus de temps | 수량 표현 + de |
| a couri | a couru | courir 과거분사 (-u 가족) |
| Terminant (선행 의미) | Ayant terminé | 분사구문 선행 = ayant + p.p. |
| sois venu (가정문 주절) | serais venu | 조건법 과거 (접속법 아님) |

## 메타 인지·자가 도출 ⭐

- naître/mourir = "세상에 옴/떠남" → 왕래발착 être 동사로 묶음
- "동사 현재형(nous)이 허브" — 현재분사·반과거·접속법 통합 인출점
- 접속법/조건법 혼동의 원인 자가 진단 ("별 생각 없이 접속법이 나온다")
- vu(voir) vs vécu(vivre) 동음 혼동 구분

## 교사측 정정

- "S6에서 *Elle vit*(vivre) 봤다"는 오설명 → 그 *vit*는 voir 단순과거였음. 동음이의(vit = voir 3sg / vivre 3sg) 혼선. vécu는 vivre 과거분사로 정확.
