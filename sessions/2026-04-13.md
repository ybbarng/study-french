## 프랑스어 표현 교정
- *ferent* (faire ils futur) → **feront** (futur ils = -ont, 현재 = -ent)
- *irions* (faire nous cond.) → **ferions** (faire = fer-, aller = ir-)
- *viendriez* (nous) → **viendrions** (nous = -ions, vous = -iez)
- *voudrons* (ils) → **voudront** (ils = -ont, nous = -ons)
- *envoirrait* → **enverrait** (envoyer 미래 어간 = enverr-)
- *saurras* → **sauras** (savoir = saur-, r 하나. rr 그룹 아님)
- *puvais* → **pouvais** (pouvoir 어간 = pouv-, ou 유지)

## 반복 필요 항목
- **faire(fer-) vs aller(ir-) 혼동** — 재발 (4/8, 4/13). f로 시작하면 faire.
- **futur ils = -ont vs 현재 ils = -ent** — ferent ❌. 미래 어미는 avoir에서 유래 (ont).
- **nous(-ons/-ions) vs vous(-ez/-iez) vs ils(-ont/-aient)** — 인칭별 어미 구분 실수 2회.
- **envoyer 미래 어간 = enverr-** — envoir- 아님. 현재 envoie와 완전히 다른 형태.
- **savoir = saur- (r 하나)** — rr 그룹(pourr-, verr-, enverr-)과 구분.
