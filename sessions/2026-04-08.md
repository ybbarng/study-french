## 새로 배운 개념

### 7-4 futur simple
- **형태**: 부정사(infinitif) + avoir 어미 (-ai, -as, -a, -ons, -ez, -ont)
- **역사적 기원**: 속라틴어 "부정사 + habere(avoir)" → 의무 → 의도 → 미래
  - parler + ai → parlerai (문자 그대로 부정사 + avoir)
  - 의무/의지 → 미래로 문법화 (grammaticalization)
- **-re 동사**: 마지막 e 탈락 (prendre → prendrai)
- **3가지 용법**: 예측/전망, 약속/의지, 격식 명령
- **불규칙 미래 어간 (11개)**:
  - aller→ir-, avoir→aur-, être→ser-, faire→fer-
  - venir→viendr-, pouvoir→pourr-, vouloir→voudr-
  - voir→verr-, devoir→devr-, envoyer→enverr-, savoir→saur-
  - rr 겹침 그룹: pourr-, verr-, enverr- (+ mourr-, courr-)

## 연습 내용과 결과

### 불규칙 어간 개별 드릴 (5개 기본)
- aller je → irai ✔️ (처음에 allerai ❌)
- avoir tu → auras ✔️
- être il → sera ✔️
- faire nous → ferons ✔️
- venir ils → viendront ✔️ (처음에 viendrai ❌, 인칭 혼동)

### 5개 섞기 테스트
- être je → serai ✔️
- aller nous → irons ❌ (ferons — faire와 혼동)
- faire tu → feras ✔️
- avoir elle → aura ✔️
- venir vous → viendrez ✔️

### 추가 불규칙 어간 드릴 (6개)
- pouvoir je → pourrai ✔️
- vouloir il → voudra ✔️
- voir nous → verrons ✔️
- devoir tu → devras ✔️
- envoyer elle → enverra ✔️ (철자 오타: eonverra)
- savoir ils → sauront ✔️

## 프랑스어 표현 교정
- *allerai* → **irai** (aller는 불규칙 어간 ir-)
- *viendrai* (ils) → **viendront** (ils = -ont 어미)
- *ferons* (aller nous) → **irons** (faire와 aller 혼동)
- *eonverra* → **enverra** (철자 오타)
- *s'est levées* (elles) → **se sont levées** (elles = 복수 → se sont)
- *Tu le lui donnes* → **Tu le lui as donné** (passé composé 요청에 현재형 사용)

## 반복 필요 항목
- **se sont (복수) vs s'est (단수)** — 대명동사 복수형 여전히 자동화 안 됨. 워밍업에서 재발.
- **tu + avoir = as** — 복합시제에서 계속 a로 쓰는 경향. 워밍업 재발.
- **aller 미래 어간 ir-** — allerai로 쓰려는 경향. 불규칙 암기 필요.
- **faire(fer-) vs aller(ir-) 혼동** — nous ferons ≠ nous irons
