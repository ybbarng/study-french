### 3-2. 형용사 위치 (복습 + 심화)
1. **기본 위치**: 명사 뒤 (후치). 의심되면 뒤가 안전.
2. **항상 후치**: 색상(rouge, bleu), 국적(français, coréen), 기술적(électrique, médical)
3. **BANGS 규칙**: Beauty, Age, Number, Goodness, Size → 명사 앞
4. **위치별 의미 변화 8개**: grand, ancien, cher, pauvre, propre, seul, dernier, certain
   - 전치 = 주관적/평가적/개념적
   - 후치 = 객관적/사실적/물리적
5. **추가 4개**: brave(착한/용감한), curieux(이상한/호기심), sacré(빌어먹을/신성한), simple(그저~/소박한)
6. **여러 형용사**: 전치(BANGS) + 명사 + 후치(색상 등)
7. **암기법**: "Mon grand-père, ancien professeur, cher et pauvre, vit seul dans sa propre maison. C'est son dernier jour — c'est certain."

## 틀린 것과 교정

### 위치 관련
- **ancien과 dernier 혼동**: "이전의"에 dernier를 사용 → ancien이 맞음
- **certain 위치**: 확실한(객관적) = 후치. 어떤(주관적) = 전치
- **petit을 후치**: BANGS(Size)이므로 전치가 기본

### 어휘 관련
- **propre = 자신의** (전치), ≠ 적합한(approprié)
- **clair ≠ propre**: clair = 밝은, propre = 깨끗한
- **dress → robe**: 영어 혼용 주의

### 철자/문법
- pouvre → **pauvre**
- semain → **semaine** (+ 여성명사: la semaine)
- le objectif → **l'objectif** (élision 여전히 자유 발화에서 누락)
- "chacun mots" → **chaque mot**
- "le maniere" → **une manière** (여성명사)

- *le maniere* → **une manière** (manière는 여성)
