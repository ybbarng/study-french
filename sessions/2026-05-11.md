## 11-1 전치사 + 관계대명사 (5문제, 60%)

### 핵심
- 동사가 전치사 요구 → 관계절에서도 전치사 유지
- 사람: 전치사 + qui (또는 lequel)
- 사물: 전치사 + lequel 계열
- 장소/시간: où
- de + 동사 요구: dont
- 영어 후치 vs 프랑스어 전치 차이

### 결과
- ⭕: Q1 (다른 방향), Q2 (qui), Q4 부분
- ❌: Q3 (de laquelle → laquelle), Q5 (de lequel → dont)

## 11-2 lequel 계열 + 드릴 (12문제, 75%)

### 핵심
- lequel/laquelle/lesquels/lesquelles (성수 일치)
- à/de 축약: auquel/duquel/auxquels/auxquelles/desquels/desquelles
- 여단 변화 X (à laquelle, de laquelle)
- dont vs duquel 분기 (동사 요구 vs 위치 전치사구)

### 점진적 드릴 (8형태 + 랜덤 4)
**à 4가지 (모두 ⭕)**: auquel / à laquelle / auxquels / auxquelles
**de 위치 (모두 ⭕)**: au-dessous duquel / près de laquelle / desquels / autour desquelles
**랜덤 (3⭕ 1❌)**: dont (rêver de) / dont (s'occuper de) / où ❌ → à laquelle / auquel

### 약점
- où vs à laquelle (선행사 장소/시간 vs 추상/사건) — Q11 conférence

## 메타 토론 (사용자 깊이)

### dont/duquel 차이
- dont: de가 동사/형용사/명사에 직접 요구
- duquel: "명사구/위치 전치사구 + de"
- 사용자 통찰 (앞에 다른 구성요소 있냐?) ⭕

### au-dessous de 위치 관계
- 선행사 = 항상 기준점
- 다른 사물의 위/아래는 사용된 전치사구로 결정
- 같은 사물, 다른 관점 (강 vs 다리)

### 도치 규칙 (관계절 안)
- 대명사 주어 → 도치 X
- 명사 주어 → 도치 ⭕ (선택, 격식)
- 보어 多 → 부자연 (가능 but 안 씀)

### à vs de 비대칭
- de = dont (광범위) / duquel (위치 표현)
- à = 선행사 종류만으로 단순 판단 (사람/사물/장소)
- où는 dont처럼 일반 X — 선행사가 장소/시간일 때만

### 원어민 발화 메커니즘
- 발화 = 계획 + 산출 동시
- 자주 쓰는 형태 자동 (qui/que/dont)
- 복잡한 형태 (duquel) 잠깐 멈춤 + 회피 전략 (단순화/두 문장 분리)
- false starts 흔함

### 언어별 "look-ahead" 부담
- 매우 높음: 라틴어, 독일어, 러시아어
- 높음: 프랑스어, 스페인어
- 중간: 영어 (who/whom, 시제 일치)
- 낮음: 한국어/일본어 (조사 + 자유 어순)
- 매우 낮음: 중국어
