## 오답 분석
| 틀린 단어 | 오답 패턴 | 규칙 |
|-----------|-----------|------|
| problème | -e 어미 → f 추측 | -ème = 남성 (système, thème) |
| page | -age → m 추측 | -age 예외: page, image = 여성 |
| exercice | -e 어미 → f 추측 | un exercice = 남성 |
| heure | 직감 오류 | -eure = 여성 |
| charme | -e 어미 → f 추측 | un charme = 남성 |
| marché | -é 어미 직감 오류 | -é = 남성 |

**핵심 패턴**: -e로 끝나면 여성이라고 추측하는 경향. charme, problème, exercice처럼 -e로 끝나도 남성인 단어 주의.
