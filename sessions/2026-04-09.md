## 새로 배운 개념

### 7-5 futur proche & passé récent
- **futur proche**: aller (현재형) + infinitif = 현재에 걸린 미래
  - 용법: 즉각적 계획, 현재 징후, 임박한 미래, 경고
  - 부정문: ne...pas가 aller를 감쌈 (Je ne vais pas partir.)
- **passé récent**: venir (현재형) + de + infinitif = 현재에 여파가 남은 과거
  - 용법: 방금 완료, 직후 설명, 결과 지속
  - de + 모음 → d' (vient d'arriver)
- **구조적 대칭**: aller(가다→미래) / venir(오다→과거)
- **futur simple과의 차이**: futur simple은 현재와 분리된 전망, futur proche는 현재에 걸린 미래
- **passé composé와의 차이**: PC는 사건 닫아 제시(거리 있음), passé récent는 여파 남음(거리 없음)
- **대명사 위치**: 부정사 앞 (Je vais le faire. / Je viens de le voir.)
  - aller/venir 앞이 아님 — semi-auxiliaire이므로
- **semi-auxiliaire 개념**: aller/venir는 조동사로 가는 문법화 과정 중이지만 아직 완전한 조동사가 아님
  - 증거: 대명사가 부정사 앞에 옴 (avoir/être 조동사는 앞에 옴)
  - futur simple의 역사와 비교: parler + ai → parlerai (융합 완료) vs aller + inf (아직 분리)
  - 접미사 기반 굴절 언어라서 aller + inf는 한 단어로 합쳐지기 어려움
- **imparfait 전환**: allais + inf (떠나려 하고 있었다), venais de + inf (막 먹은 참이었다)

## 프랑스어 표현 교정
- *poudrai* → **pourrai** (pouvoir 미래 어간은 pourr-, rr 겹침)
- *soie* → **sois** (soie = 실크, sois = être 명령형 tu)
- *vien* → **viens** (je에는 항상 -s)
- *du finir* → **de le finir** (le는 대명사, 관사 축약 du 적용 불가)
- *vient téléphoné* → **vient de téléphoner** (de 필수 + 부정사 사용, 과거분사 아님)

## 반복 필요 항목
- **pourr- (rr 겹침)** — poudrai로 씀. pouvoir/voir/envoyer는 rr 그룹.
- **sois vs soie 혼동** — être 명령형 tu = sois (-s). soie = 실크. 재발.
- **passé récent = venir de + infinitif** — de 빠뜨림 + 과거분사/부정사 혼동. 핵심 구조 반복 필요.
