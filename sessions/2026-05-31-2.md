## 14-1 ce qui / ce que / ce dont 학습 (5/5 ⭐⭐⭐)

### 핵심 개념
- 특정 명사 X → **'내용 전체'** 지시
- 11장 qui/que/dont 논리 + ce (추상 표지)
- 영어 *what* 하나에 대응

### 결정 트리
```
지시 대상 = 내용 전체
관계절 안 역할?
- 주어 → ce qui
- COD → ce que
- de 보어 → ce dont
```

## 14-2 ce + être 학습 (5/5 ⭐⭐⭐)

### 핵심 개념
- **ce + être** = 정의/평가/분류 연결
- 분열문 (cleft structure)
- 14-1과 결합되어 정형 표현 형성: *Ce qui/que/dont ..., c'est ...*

### ⭐ c'est vs il est 구분
| 구조 | 사용 |
|---|---|
| **c'est + 명사/대명사/절** | 정의, 동격 |
| **il est + 형용사 + de/que** | 성질 서술 (격식) |

### ce vs cela vs ça
- ce: ce + être / 관계대명사 구조에서만
- cela: 격식, 단독 주어 ⭕
- ça: 구어, 만능

### 연습 결과
| # | 답 | 결과 |
|---|----|----|
| Q1 | c'est (정의) | ⭕ |
| Q2 | Il est (~하는 것이 중요하다 + 부정사) | ⭕ ⭐ |
| Q3 | c'était (imparfait 시제 변경) | ⭕ |
| Q4 | Ce n'est pas (부정 + 엘리지옹) | ⭕ |
| Q5 | Ce que j'aime, c'est son sourire (14-1+14-2 완전 통합) | ⭕ ⭐⭐⭐ |
