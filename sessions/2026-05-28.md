## 13-2 접속법 반과거 (인지 위주, 5문제)

### 핵심 개념
- **사용 X, 인식 ⭕** — 현대 프랑스어에서 거의 소멸
- 의미 = 접속법 현재와 **동일** (시간 일치만)
- 등장 = 주절 과거 + 종속절 접속법 (옛 격식)
- 표지 = **^ + 어미 패턴 (-ass-/-iss-/-usse-/-insse-) + que 트리거**

### 7 핵심 불규칙 (인지 필수)
| 동사 | 접속 반과거 3sg | 현대 (접속 현재) |
|------|-----------------|------------------|
| être | **fût** | soit |
| avoir | **eût** | ait |
| faire | **fît** | fasse |
| aller | **allât** | aille |
| pouvoir | **pût** | puisse |
| vouloir | **voulût** | veuille |
| savoir | **sût** | sache |

### 사용자 메타 전략 (탁월)
"parla... 보면 parler 가족이라고 생각하고, que 있으면 접속법 의심" — 정확한 인지 알고리즘
- 어간 → 동사 추정
- 어미 4 패턴 (-ass-/-iss-/-usse-/-insse-)
- que + ^ → 접속법 반과거 확신
→ **6변화 외울 필요 X**, 이 알고리즘만으로 충분

### 변환 연습 결과 (5문제, 3⭕ + 2❌)
| # | 문장 | 답 | 결과 |
|---|------|----|----|

| M2 | "Bien qu'il eût raison" | avoir, ait | ⭕ |
| M3 | "qu'on le fît exécuter" | faire, fasse | ⭕ |

| M5 | "qui pût l'aider" | pouvoir, pouve ❌ → puisse | ❌ (어간 puiss- 회수 미달, 5/24 ⭕ → 5/28 망각) |

### 신규 학습 (예고)
- **ne explétif** (15-1): craindre que + ne (부정 X)
- **사역동사 faire + 부정사** (S2): faire exécuter
- **^ = 옛 -s 자취** (어원): parlast → parlât, hôpital(옛 hospital), être(옛 estre)

## S1 futur antérieur (5문제, 4⭕ + 1△)

### 핵심 개념
- **avoir/être 미래 + 과거분사**
- 의미: **미래 시점 이전 완료**
- 4 합성 시제 가족 완성: plus-que-parfait / subjonctif passé / **futur antérieur** / infinitif passé
- 접속법 과거 ↔ futur antérieur 평행 (둘 다 "상대 완료")

### 사용 환경 2가지
1. **시간 접속사 + 미래 시점 이전 완료**
   - quand / dès que / une fois que / lorsque / après que + futur antérieur + futur simple
   - *Quand tu auras fini, je partirai.*
2. **과거 추정 (구어 자주)**
   - *Il aura oublié.* (잊었을 것이다)
   - *Elle sera partie sans nous prévenir.*

### 연습 결과
| # | 답 | 결과 |
|---|----|----|
| Q1 | auras fini | ⭕ |
| Q2 | sera arrivée | ⭕ |
| Q3 | aura oublié (추정) | ⭕ ⭐ |
| Q4 | 한국어 번역 (시제 ⭕, prêter→준비해줄게 ❌) | △ |
| Q5 | aurez lu + comprendrez | ⭕⭕ ⭐⭐⭐ |

### 신규 약점
- **prêter (빌려주다) vs préparer (준비하다) 혼동** — Q4
- 형태 비슷 (pr- + -er), 의미 완전 다름. emprunter(빌리다)와 함께 정리.

### 접속법 반과거 vs 과거 단순 ^ 표지 어원
- 옛 프랑스어: il parla**st** (s 있음)
- 중세 후반 s 탈락
- 현대: 사라진 s 자리에 ^ (보충 부호)
- 같은 원리: hôpital ← hospital, être ← estre, forêt ← forest

### prêter 어휘 분석
- prêter = 빌려주다 (lend)
- préparer = 준비하다 (prepare)
- emprunter = 빌리다 (borrow)
- 형태 함정 (pr- + -er) 경계

### S2 사역동사 (5/5 ⭐⭐⭐)

3 그룹:
1. **faire + inf** (사역, 강제)
2. **laisser + inf** (방임, 허용)
3. **voir/entendre/sentir + inf** (지각)

**핵심 규칙**:
- 대명사 = **모동사 앞** (Je **la** fais réparer)
- 직접목적어 두 개 불가 → 행위자 à/par로 밀림
- 과거분사 일치 (2016 학술원):
  - faire = **불변** (Je l'ai *fait* réparer)
  - laisser = 권장 불변
  - 감각동사 = **일치 ⭕** (Je l'ai *vue* partir)
