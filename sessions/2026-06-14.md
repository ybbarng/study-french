# 2026-06-14 — 워밍업 + 고급 단계별 평가 재시험 2회차 ✅ 통과 🏆

대조 5형태 + 자잘한 형태 워밍업 → 고급 단계별 평가 재시험 2회차 **19/20 (95%) 통과**. 단계별 평가 3개(초급·중급·고급) 전부 통과 완료. 다음 단계는 모의시험.

## 1. 워밍업 (대조 종합 + 형태 점검) — 8문제

- **모니터링: 엘리지옹** (오늘 3회 qu'elle·d'amis·j'avais → Q7/8 회복)

## 2. 고급 단계별 평가 재시험 2회차 — 19/20 (95%) ✅ 통과 🏆

- 1차 80% → 재시험1 84% → **재시험2 95%**, 꾸준한 향상
- **L5 자유 작문 4/4** (어제 약점 영역 완전 회복)
- **대조(Q6·Q11·Q19) 전부 정확** — 어제 학습 첫 시험에서 정착 확인 ⭐
- 유일 오답: Q10 분열문 인칭 일치 (a→as)
- 경계선 통과 — Q20 법 대비는 완벽, 표현 alors que/mais는 모니터 (사용자 정직성 확인 후 통과 인정)

## 프랑스어 표현 교정

| 사용자 | 정답 | 비고 |
|---|---|---|
| Alors que elle | Alors qu'elle | 엘리지옹 (que+모음) |
| à la compagne | à la campagne | 시골(a) ≠ 동반자(o) |
| beaucoup de amis | beaucoup d'amis | 엘리지옹 (de+모음) |
| je te serais aidé | je t'aurais aidé | aider=avoir 동사 + 엘리지옹 |
| voyager le monde | voyager dans le monde | 자동사 (재료 부실 — 교사 책임) |
| C'est toi qui a fait | ...qui as fait | 분열문 인칭 일치 (2sg) |

## 교사측 개선 (사용자 지적 반영)

- **출제 범위 위반 또 발생**: Q9 워밍업 manquer/le train/plus tôt (미등록 어휘) → 무효. 재시험은 **재료를 어휘 풀에서 확인하며** 출제 (개선)
- **재료는 collocation으로** (사용자 지적): voyager **dans** le monde처럼 전치사 결합까지 줘야 함. 자동/타동·전치사는 어휘 지식이라 추측 강요 금지
- 재시험 전 과정 어휘 사전 확인 적용 (grep 대조)
