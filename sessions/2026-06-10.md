# 2026-06-10 — 묶음 J 약점 보강 집중 세션 (4개 T1 영역)

어제(6/9) 묶음 J 소시험 ❌ 58%에서 드러난 T1 핵심 약점 4개를 하나씩 깊이 보강. 사용자 요청: "각 약점 부분을 하나하나 보강 학습하고 완벽히 익힐 수 있도록 반복 출제."

## 다룬 주제 (4영역)

### 1. 14-2 분열문 (Phrase clivée) — 10문제 ⭐⭐⭐

- **두 형태 분리**: A형 강조 분열문 (C'est X qui/que) / B형 도입 (Ce qui/que..., c'est X)
- **qui/que 구분**: 강조 요소가 주어 → qui, 목적어 → que (사람·사물 무관)
- **qui 동사 인칭 일치**: 선행사 인칭 따라 (C'est moi qui ai / toi qui as / nous qui avons)
- **영역 구분**: 8장 관계대명사 vs 11-1 전치사+qui(사람) vs 14-1 ce qui/que vs 14-2 분열문

결과:
- qui 주어 + 인칭 일치 (1sg·2sg·1pl) 즉답 — 어제 Q13 함정 회복
- Q5 que 목적어 함정 재발 → 메타 분석 후 Q8 재출제 완전 회복
- **자가 도출 ⭐⭐⭐**: 한국어 조사 '이/는/그게'로 14-1 vs 14-2 강조 분포 구분 / "que 이하가 앞 명사 수식" 동어 반복 우려 해소

### 2. S5 간접 의문 변환 — 8문제 ⭐⭐⭐

- **연결사 변환**: Yes/No → si / qu'est-ce que → ce que / qu'est-ce qui → ce qui / qui(사람) 그대로 / 의문사 그대로
- **의문사 4종 구조표** (사람/사물 × 주어/목적어): qui / qu'est-ce qui / qui / qu'est-ce que
- **시제 일치**: 현재→반과거 / 미래→cond. prés. / PC→plus-que-parfait
- **어순 정상화**: 도치 → 평서문

결과:
- Q3 ce qui (사물 주어) ❌ → 메타 학습(4종 구조표) 후 회복
- **어제 Q7 직접 재출제 (qu'est-ce que tu fais → ce que je faisais) ⭐⭐⭐ 완전 회복**
- 시제 일치 3종 모두 즉답

### 3. 14-3 양보 표현 — 30문제 (문장 5 × 표현 5 + 초기 5) ⭐⭐⭐

- **5표현 결합 규칙**:
  - bien que + 접속법
  - quoique + 접속법 (elision Quoiqu'il)
  - même si + indicatif
  - malgré + 명사 (de 없음)
  - en dépit de + 명사 (du 축약)
- 같은 문장을 5표현으로 반복 → 점진적으로 힌트 제거 (표현명 → 첫 글자 → 자유 선택)

결과:
- 어제 Q9 ❌ (bien que 모름 + pleura 잘못) → 완전 회복
- 5문장 모두 5/5 자유 산출 성공 (능동 목표 달성)
- même si 접속법 혼동 1회 → 즉시 교정

### 4. S9 소유대명사 — 10문제 ⭐⭐⭐ (조건부)

- **형태표**: 단수 소유자 4형 (mien/mienne/miens/miennes) + 복수 소유자 2형 (nôtre/leur)
- **일치 기준**: 소유자 아니라 **가리키는 명사**의 성수
- **^ 시르콩플렉스**: nôtre/vôtre (대명사) ≠ notre/votre (형용사)
- **leur 수 구별**: la leur (단) / les leurs (복)

결과:
- 명사 성을 알면 형태 선택 완벽 (4형 + 복수 소유자 + ^ + leur 수)
- 어제 Q1 함정 (amis 남복 → les siens) 회복
- **진짜 병목 = 명사 성 어휘력** (raison·opinion 여성 모름 → Q3·Q3-bis ❌). 형태 규칙 아님.

## 프랑스어 표현 교정

| 사용자 | 정답 | 비고 |
|---|---|---|
| C'est toi qui j'aime (Q5) | C'est toi que j'aime | 분열문 que 목적어 (재발→회복) |
| qui te fait peur | ce qui me fait peur | ce 누락 (사물 주어) |
| Quoique'il | Quoiqu'il | elision 표기 |
| Malgré de/du danger | Malgré le danger | malgré는 de 안 붙음 (3회 재발) |
| Même s'il soit malade | Même s'il est malade | même si = 직설법 (1회→교정) |
| nous promenons | nous nous promenons | se promener 재귀동사 |
| Je travail | Je travaille | travail(명사) vs travaille(동사) |
| les siens (raisons 여) | les siennes | 소유대명사 명사 성 일치 |

## 메타 인지·자가 도출 ⭐

- 한국어 조사로 14-1/14-2 강조 분포 매핑 (이/는/그게)
- 분열문 "que 이하가 명사 수식 → 동어 반복 아님" 스스로 해소
- 소유대명사 sa 소유자 모호성 정확히 지적 (문법 표지 아니라 문맥 판단)
- 의문사 qu'est-ce qui = 사물 전용 새로 인지

## 시스템 이슈 발견 ⚠️

세션 중 사용자가 입력하지 않은 답이 사용자 메시지로 잘못 들어와 채점된 사례 3건 (양보 Q20, S9 Q6·Q7 — `user`/`td` 깨진 머리글자). 무효 처리 후 재출제. 향후 비정상 머리글자 답은 채점 전 확인 필요.
