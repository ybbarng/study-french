## 10-2 간접화법 — 진행 상황

### 도입 + 메타 토론
- 직접 vs 간접 개념 정립
- 보고 동사 종류 4그룹 (dire/penser/demander/ordonner)
- 시제 일치 = 모든 과거 시제 보고 동사에 동일 적용
- 종속절 포함 평서문 처리 (que + 전체 + 시제 일치)
- 쉼표 규칙 (앞 부사절 양쪽 쉼표 / 뒤 부사절 쉼표 X)
- conditionnel의 두 기능 (법 + 시제, futur du passé)

### 변환 단계별 진행

**단계 1: 인칭만** (5/5 ⭕)
- Marie/Paul/parents/소유형용사/tu→je 변환

**단계 2: 시제만** (5/5 ⭕, 재시도 2번)
- 현재 → 반과거 ⭕ (재시도 후)
- 미래 → 조건법 ⭕ (재시도 후) + 강화 1문제
- PC → 대과거 ⭕
- 종속절 + 반과거 ⭕ (재시도 후, 누락 보강)
- 대과거 → 변화 X ⭕

**단계 3: 지시어만** (5/5 ⭕, 재시도 1번)
- demain → le lendemain ⭕
- hier → la veille ⭕
- ici/maintenant 동시 ⭕ (alors 어순 평가 실수 → 정정)
- cette semaine → cette semaine-là ⭕ (재시도 후, semaine 여성 + 시제)
- aujourd'hui → ce jour-là ⭕

### 남은 단계
- 단계 4: 의문문 (si/의문사/que→ce que/qu'est-ce qui→ce qui)
- 단계 5: 명령문 (de + 부정사)
- 단계 6: 종합
