## 12-3 핵심 개념

### 분사구문 정의
부사절/관계절을 분사 하나로 압축. **문법 축약의 최종 형태**.
- 접속사 제거 / 관계대명사 제거 / 시제 표지 제거

### 두 가지 유형

**A. participe présent 기반** (12-1 응용)
- *Étant fatigué, il est parti.* (원인)
- *Travaillant, il écoutait...* (동시)
- 의미는 문맥으로 결정

**B. participe passé 기반** (12-2 응용, 신규)
- *Le travail terminé, il est parti.* (완료 후 상태)
- *La nuit tombée, ils sont rentrés.*
- *Les documents signés, la réunion a commencé.*

### 두 가지 타입

**A. 주어 공유형 (12-1)**: 분사 주어 = 주절 주어
- *Étant malade, il est resté chez lui.*

**B. 절대 분사구문 (construction absolue) — 신규 ⭐**
- *La nuit tombée, ils sont partis.*
- 분사가 자기 주어 명시
- 라틴어 ablativus absolutus에서 직접 계승

### dangling participle vs 절대 분사구문
| 형태 | 분석 |
|---|---|
| ❌ *Ayant fini, la porte était fermée.* | 분사 주어 ≠ 주절 주어 + 분사 주어 명시 X → 오류 |
| ⭕ *Le travail fini, la porte était fermée.* | 분사 자기 주어 명시 (le travail) → 절대 분사구문 |

### 절대 분사구문 허용 조건 (3가지)
1. **상태로 해석 가능한 동사** (tomber/terminer/fermer/finir 등). aimer 같은 지속 상태 동사 X.
2. **주어가 사건 단위** (la nuit/le travail/la réunion). 일상 사람 주어 매우 제한적.
3. **담화적 배경 역할** (주절의 전제/시간/원인 프레임)

→ 절대 분사구문 = **거의 항상 participe passé**.

### 선행 완료 분사구문 (ayant + p.p.)
- *Ayant fini ses devoirs, elle est sortie.*
- 주어 공유형 + 선행 완료
- 종속절 plus-que-parfait + 주절 PC → ayant 분사

### 선행 완료 판별 (3 신호)
1. **시제**: plus-que-parfait → 선행 (ayant)
2. **접속사**: après que / une fois que / dès que → 선행
3. **부사절 의미**: "~한 후에" → 선행

### *une fois que* 의미
- "한 번 ~하면 / ~하자마자" (영어 once)
- 시간 + **완료/전제** 의미
- participe passé 분사구문과 자연스러움
