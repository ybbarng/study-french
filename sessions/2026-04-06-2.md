## 새로 배운 개념

### 7-2 imparfait
- **의미**: 과거의 열린 상태/배경 (im-parfait = 완료되지 않은)
- **형태**: 어간(nous 현재형 - ons) + -ais/-ais/-ait/-ions/-iez/-aient
  - être만 예외: ét- (나머지 전부 nous 현재형에서 유도)
- **4가지 용법**:
  1. 배경 묘사: Il pleuvait. (비가 오고 있었다)
  2. 지속 상태: J'habitais à Paris. (파리에 살고 있었다)
  3. 습관/반복: Chaque matin, elle prenait un café.
  4. 진행 중 동작 + 사건: Je lisais quand il est entré.
- **passé composé와의 대비**:
  - imparfait = 열림, 배경, 상태 / passé composé = 닫힘, 사건, 결과
  - imparfait는 서사를 전진시키지 못함

## 연습 내용과 결과

### imparfait 활용 (Q1~Q5)
- je mangeais ✔️, il avait ✔️, j'étais ✔️, nous faisions ✔️, ils prenaient ✔️ — 전부 정답

### passé composé vs imparfait 판별 (Q6~Q9)
- Q6: pleuvait(배경) + est entré(사건) ✔️
- Q7: dormait ✔️, est sonné ❌ (sonner는 être 그룹 아님 → a sonné)
- Q8: été ❌ (과거분사와 imparfait 혼동 → étais), mangeais ✔️
- Q9: habitait(상태) + est parti(사건) ✔️

## 프랑스어 표현 교정
- *met* → **mis** (mettre 과거분사, met는 현재형)
- *est sonné* → **a sonné** (sonner = avoir, être 그룹 아님)
- *j'été petit* → **j'étais petit** (été = 과거분사, étais = imparfait)

## 반복 필요 항목

- **sonner → avoir** (être 그룹 14개에 포함 안 됨)
