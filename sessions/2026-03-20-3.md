## 새로 배운 개념

### 4-1. 주어 대명사 체계
1. **주어 대명사 전체 목록**: je, tu, il/elle/on, nous, vous, ils/elles
2. **주어 대명사 필수**: 동사 발음이 같아서 (je/tu/il/ils parle(nt) = 모두 /parl/) 주어 없이는 인칭 구분 불가
3. **on의 3가지 의미**: 불특정 일반인 / nous 대체(구어) / 수동적(장소 규칙)
4. **on 활용**: 항상 3인칭 단수 (On va ✔️, ~~On allons~~ ❌), 형용사는 의미상 복수 일치 가능 (On est prêts)
5. **tu vs vous**: tu=비격식(친구/가족/아이), vous=격식(공식/처음/윗사람). 불확실하면 vous
6. **강세형 대명사**: moi/toi/lui/elle/soi/nous/vous/eux/elles — 강조, 전치사 뒤, 독립 사용
7. **강세형은 주어 불가**: ~~Moi parle~~ ❌ → Moi, **je** parle ✔️
8. **ils vs elles**: 혼성 복수(남+여)는 항상 ils, elles는 전원 여성일 때만
9. **é → è 변환 규칙**: 닫힌 음절(자음으로 끝남)에서는 é가 è로 바뀜 (préférer → je préfère)
10. **열린/닫힌 모음 쌍**: é/è, o/ɔ, eu(ø)/eu(œ) — 3개 쌍

## 프랑스어 표현 교정
- *le nom de un homme* → **le nom d'un homme** (de + un → d'un, élision)
- *Le sujet n'est pas existe* → **Le sujet n'existe pas** (exister = 단독 동사)
- *préfére* → **préfère** (accent grave è, 닫힌 음절)

## 반복 필요 항목
- elles vs ils 구분 (전원 여성 = elles, 혼성 = ils)
- on의 수동적 용법 (ici + on = 장소 규칙)
- BANGS 최상급 어순 복습 (au meilleur restaurant, 명사 앞)
- é → è 변환 (닫힌 음절 규칙)
