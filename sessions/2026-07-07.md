# 2026-07-07 — /write: 모델 모방(Claude Code) + ce qui/ce que 깊이 파기

화요일 모델 모방(36단어). 워밍업에서 ce qui↔ce que를 "왜 ce가 필요한가(선행사)"까지 파고들어 정착. 새 T1 = 모음 앞 엘리지옹.

## 1. 워밍업 — ce qui↔ce que 몰아서 (+ 깊이)
| # | 카드 | 결과 |
|---|------|------|
| 1 | ce qui (주어) | ❌ C'est important qui… (비문) → **Ce qui est important, c'est la régularité** 재생산 ✅ |
| 2 | ce que (목적어) | ✅ Ce que j'aime, c'est le calme |
| 3 | ce que + 엘리지옹 | △ Ce que il veut → **Ce qu'il** (que+il 엘리지옹만) |
- **핵심 개념 정착**: qui는 **선행사(명사)가 필요** → 형용사(important)엔 못 붙음 → **ce가 중성 선행사** 역할("ce qui"에서 ce가 qui의 선행사). 사용자 자가 실험(C'est important ce qui…)까지 하며 이해.
- 대체 구조: **C'est la régularité qui est importante**(명사 선행사 + 도치)도 유효.

## 2. 작문 — Claude Code 예찬 (모델 모방, 36단어)
- 상세·총평·교정은 `writing/exercises/2026-07-07-작문-claude-code-모방.md`.
- **잘한 것 ⭐**: **Ce qui me plaît le plus dans X, c'est que…**(모델 틀 정확 모방 — 오늘 T1 ce qui 능동!) · il me permet de+부정사 · De plus · faire des progrès · régulièrement.
- **고칠 것**: ❌ 모음 앞 엘리지옹 누락(le utilisation→**l'**, de apprendre→**d'**) · exerciser→**s'entraîner** · français facile→**facilement**(부사) · de compétence pour→**en** français. △ 활동 주어면 permet de faire.

## 3. 곁다리 문법
- **ce qui의 ce = qui의 선행사**(중성). qui는 선행사 필수, 형용사엔 못 붙음.
- **aimer bien(좋아하다) vs aimer(사랑하다)** — 사물엔 큰 차이 없지만 사람엔 확 다름. "좋다" 느낌엔 j'aime bien / J'aime ça / cette idée-là.
- **calme = /kalm/** — 프랑스어는 **l 발음됨** (영어 calm은 l 묵음).
