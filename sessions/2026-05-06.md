## 10-3 강조 구조 — 학습 완료 ✅

### 핵심
- c'est + 강조 성분 + qui (주어) / que (그 외)
- 시간/장소/방법 등도 que (où 사용 X — 부사 선행사 X)
- 대명사 강조 시 강세형 (moi/toi/lui)

### 메타 토론 (사용자 깊이 토론)
- 강조 구문 vs 관계절 소개 — 형태 비슷, 의미 거의 동일 (학술적으로 같은 분열 구문 가족)
- où/dont 사용 조건: **선행사가 명사**일 때만 (부사 hier는 X)
- *C'est le jour où ~* (관계절) ≈ *C'est le jour que ~* (강조) — 의미 거의 같음
- 한국어 강조 형태 다양 (어순 변경 / 분열 구문 / 보조사 / 강조 부사)
- 한국어 강조 위치: 형태별 다름 (어순=앞쪽 / 분열=뒤쪽 / 보조사=무관)

### 연습 (5/5)
- Q1 주어 (qui) ⭕
- Q2 COD (que) — cherchees 동사 활용 일치 시도 (잘못, 동사는 일치 X) 정정
- Q3 시간 (que) ⭕
- Q4 장소 (à la gare que) ⭕
- Q5 강세대명사 (moi qui ai fait) ⭕

## 10-4 se + 동사 — 학습 완료 ✅

### 핵심: se의 4가지 기능
1. **재귀 (réfléchi)**: 자기 자신을/에게 (se = COD 또는 COI)
2. **상호 (réciproque)**: 서로 (복수 주어)
3. **중간태 (moyen)**: 행위자 제거 + 일반화 (La porte se ferme)
4. **어휘적 고착 (lexicalisé)**: se 없이 의미 X (se souvenir 등)

### PC 일치 규칙 (5/6 메타 토론과 연결)
- 조동사 = 항상 être
- 일치 규칙 = avoir 규칙 (COD 위치)
- se = COD → 일치 ⭕ (Elle s'est lavée)
- se = COI → 일치 X (Elle s'est lavé les mains)

### 메타 토론
- réfléchi vs lexicalisé 구분 = "동사 단독 사용 가능?"
- *laver* 단독 가능 → réfléchi
- *souvenir* (verbe seul) 거의 X → lexicalisé
- 명사 souvenir vs 동사 souvenir 구별 (명사로는 자주 사용)

### 연습 (5/5)
- Q1 *Cette voiture se vend* → moyen
- Q2 *Marie et Paul se regardent* → réciproque
- Q3 *Je me souviens* → lexicalisé
- Q4 *Elle se lave les cheveux* → réfléchi (COI + 신체 부위 COD)
- Q5 *Les enfants se sont endormis* → réfléchi

## 메타 토론 (사용자 깊이)

### PC 일치 본질 (5/6 통찰)
- 사용자 통찰: "분사 성수 일치 = 행위가 작용하는 대상과 일치"
- avoir = COD가 작용 대상 (단, COD가 동사 앞일 때만 일치)
- être 자동사 = 주어가 변화 대상 → 주어와 일치
- être 대명동사/수동 = 주어가 작용 대상 → 주어와 일치
- 사용자가 학술적 본질을 직관적으로 도출 ⭐

### 격 시스템 (5/6)
- 인도유럽어 특유 X — 다양한 가족 (우랄어, 셈어 등)
- 표현 방법: 굴절 / 조사 / 위치+전치사
- 프랑스어 관계대명사 (qui/que/dont/où/lequel) = 격 잔재 (분리 단어 형태)
- 사용자 학습 효율 선호 = 일관 규칙 → 독일어 후보 부상

### 한국어 분류 — 격 조사 시스템
- 한국어 = 격 조사 시스템 (분리 형태소형)
- 유럽어 격(굴절)과 같은 기능, 다른 표현
- 사용자 정리 정확

### 다국어 학습 계획 (사용자 결정)
- 2027 새 언어: 스페인어 1순위 / 일본어 2순위
- 일본어 = 환경 노출 ↑, 한자 부담 ↑
- 스페인어 = 프랑스어 시너지, 라틴어 확장 가능
- 3~5년 단위 누적 학습 계획
- 동시 학습 X (회사 + 분기 시스템)
- 일본어 입력 부담 (한자 인지 필요) — 한국어 입력보다 어려움
- 프랑스어 유지 = 주 2~3시간 (수동 노출 + 월 1회 의식 점검)

### 강조 구문 vs 관계절 (5/6 사용자 통찰)
- 두 구조 본질 같음 (분열 구문 가족)
- 차이는 표현 형태 선택 (부사+que / 명사+où)
- où 사용 조건: 선행사가 명사
- 한국어 강조 위치는 형태별 다름
