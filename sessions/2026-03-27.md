## 새로 배운 개념

### 5-1. 전치사의 의미 범주 (Les prépositions)
1. **전치사 = 관계 표시어**: 명사를 문장의 다른 요소와 연결. 전치사 + 명사 = 전치사구.
2. **6개 필수 전치사**:
   - **à**: 방향, 도달점, 시점, 목적/수혜 (Je vais à Paris.)
   - **de**: 출발, 기원, 소유, 내용 (Je viens de Corée.)
   - **dans**: 내부, 구체적 공간 (dans la chambre)
   - **sur**: 표면 접촉 (sur la table)
   - **avec**: 동반, 수단 (avec Marie, avec une fourchette)
   - **sans**: 부재, 결여 (sans sucre)
3. **à와 de의 구조적 대립**: 도착점(à) ↔ 출발점(de)
   - une tasse à café = 커피용 컵 (용도)
   - une tasse de café = 커피 한 잔 (내용물)
4. **관사 축약 (contraction)**: 필수!
   - à + le = **au**, à + les = **aux**
   - de + le = **du**, de + les = **des**
   - à la, à l', de la, de l' → 축약 없음
5. **기타 전치사**:
   - **pour** (목적), **par** (수단/경유), **en** (추상적/무관사), **chez** (사람의 집)
6. **dans vs en**:
   - dans + article = 구체적 공간 (dans la voiture)
   - en = 추상적/수단, 관사 없음 (en voiture)
7. **chez + 사람만**: chez Marie ✔️, ~~chez le restaurant~~ ❌ → au restaurant
8. **나라 + 전치사**: en + 여성/모음 (en France), au + 남성 (au Japon), aux + 복수 (aux États-Unis)

### 5-2. 부사의 기능과 위치 (Les adverbes)
1. **부사 = 불변화사**: 성/수 일치 없음. 명사 수식 불가.
2. **4가지 수식 대상**:
   - 동사: Il parle **bien**.
   - 형용사: Il est **très** content.
   - 다른 부사: Il parle **assez** vite.
   - 문장 전체: **Heureusement**, il fait beau.
3. **기본 위치 규칙**:
   - 동사 수식 → 동사 **뒤** (한국어와 반대!)
   - 형용사 수식 → 형용사 **앞**
   - 문장 수식 → **문두**
4. **très vs beaucoup**:
   - très + 형용사/부사 (très fatiguée)
   - 동사 + beaucoup (Il mange beaucoup)
   - ~~très beaucoup~~ ❌, ~~Il mange très~~ ❌
5. **bon(형용사) vs bien(부사)**:
   - un bon film (형용사, 명사 수식)
   - Il chante bien (부사, 동사 수식)
6. **의미 분류**: 방식, 정도, 빈도, 시간, 장소, 긍정/부정, 담화

## 프랑스어 표현 교정
- *de* (설탕 없이) → **sans** sucre (sans = 부재, de ≠ 없이)
- *J'habitude* → **J'habite** (habitude=명사 습관, habiter=동사 살다)
- *à France* → **en France** (나라 여성 → en)
- *Fais prudent* → **Aie confiance** (avoir confiance, 아직 혼동)

## 반복 필요 항목
- **faire attention 암기**: 3회 연속 기억 못함. Fais/Faisons/Faites attention ! 반복 필요
- **confiance ↔ prudent 혼동**: avoir confiance vs être prudent. 월말 평가 + 이번 세션 연속 오류
- **sans (부재)**: de와 혼동. sans = 없이, de = ~의/~로부터
- **en + 나라 여성**: à France ❌ → en France ✔️
- **habiter ≠ habitude**: 동사 vs 명사 구분
