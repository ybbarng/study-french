## 새로 배운 개념

### 5-3. 부사의 형성 (La formation des adverbes)
1. **기본 공식**: 형용사 여성형 + -ment = 부사
   - lent → lente → lente**ment**
   - heureux → heureuse → heureuse**ment**
   - actif → active → active**ment**
2. **왜 여성형?**: 여성형이 어말 자음을 보존하여 어간이 더 명확. lent /lɑ̃/ vs lente /lɑ̃t/ → lentement /lɑ̃tmɑ̃/
3. **-e로 끝나는 형용사**: 그대로 + -ment (rapide → rapidement, facile → facilement)
4. **-ant / -ent 형용사**: 별도 규칙
   - -ant → -amment (constant → constamment)
   - -ent → -emment (prudent → prudemment)
   - 철자는 다르지만 발음은 둘 다 /amɑ̃/
5. **불규칙 부사**: bon→bien, mauvais→mal, meilleur→mieux, petit→peu
6. **형용사=부사 (불변)**: parler fort, coûter cher, chanter faux (성수 일치 없음)
7. **bon/bien/meilleur/mieux 2×2 구조**:
   - 형용사: bon (기본) → meilleur (비교급)
   - 부사: bien (기본) → mieux (비교급)

### 5-4. 부사 위치 규칙 (La position des adverbes)
1. **단순 시제**: 방식/빈도 부사 → 동사 뒤, 정도 부사 → 형용사 앞, 담화 부사 → 문두+콤마
2. **복합 시제 (passé composé 등)**: 조동사 + 과거분사 구조에서
   - **짧은 부사** (bien, déjà, beaucoup, trop, encore, vraiment) → 조동사 뒤, 과거분사 앞 (J'ai bien compris.)
   - **긴 부사** (-ment 부사) → 과거분사 뒤 (J'ai mangé rapidement.)
3. **부정문 + 부사**: ne + 조동사 + pas + 부사 + 과거분사 (Je n'ai pas bien compris.)
4. **문두 가능 부사**: 시간(hier, demain), 빈도(souvent), 담화(heureusement) — 문두는 강조, 의무 아님
5. **문두 불가 부사**: 방식(bien, mal), 정도(très, trop)

## 프랑스어 표현 교정
- *couremment* → **couramment** (courant은 -ant → -amment, -ent가 아님)
- *Il chante bien que moi* → **mieux** (비교 que → 비교급 필요)
- *Elle parle trop vite* → **très** vite (trop=너무 too, très=매우 very)
- *je ne comprend pas* → **comprends** (je에는 항상 -s)
- *lui* (그것을) → **ça** (직접목적어 = ça, lui = 간접목적어/강세형)
- 현재시제로 과거 표현 → passé composé 사용 (n'ai pas compris)

## 반복 필요 항목
- **-ant → -amment, -ent → -emment**: courant→couramment 오류. 형용사 어미 확인 습관 필요
- **bien(기본) vs mieux(비교급)**: que가 있으면 비교급!
- **très ≠ trop**: très=매우(very), trop=너무(too)
- **beaucoup 복합시제 위치**: 조동사 뒤, 과거분사 앞 (짧은 부사)
- **comprends 철자**: je에는 항상 -s (je comprends, pas comprend)
- **ça vs lui**: 직접목적어 = ça/le/la, 간접 = lui
