## 새로 배운 / 강조한 개념

### 간접화법 변환 4축 체크리스트
1. ✓ 인칭 (주어 + 청자 + 소유형용사)
2. ✓ 시제 (현재→imparfait, PC→plus-que-parfait, 미래→cond.)
3. ✓ 시간 부사 (-là 후치 / 형용사 치환 / 단어 치환)
4. ✓ 장소 부사 (ici → là 필요 시)

### 시간 부사 변환 — 두 카테고리
| 패턴 | 변환 |
|---|---|
| ce matin (지시) | ce matin-**là** (ce 유지, -là 후치) |
| la semaine dernière (정관사+형) | la semaine **précédente** |
| la semaine prochaine (정관사+형) | la semaine **suivante** |
| aujourd'hui (부사) | ce jour-là |
| hier (부사) | la veille |
| demain (부사) | le lendemain |

### prochain vs dernier 방향 유지
- 미래 방향: prochain → suivant (잇따른)
- 과거 방향: dernier → précédent (이전의)
- 방향을 바꾸지 X — 기준점만 "지금"→"그때"로 이동

### 대명동사 PC 일치 (10-4 핵심)
- 항상 être 조동사
- COD 위치 기준 일치:
  - COD가 동사 뒤 → 일치 X (lavé)
  - COD가 동사 앞 → 일치 ⭕ (les mains qu'elle s'est lavées)
  - 자기 자신을 직접 COD로 → 일치 ⭕ (Elle s'est lavée)

### 단순 전치사 + lequel vs 전치사구 + duquel
- 단순 전치사 (sous, sur, devant, dans, entre): + lequel
- 전치사구 끝이 de (au-dessous de, près de, autour de): + duquel
- 전치사 자체가 de로 끝나는지 확인

### lequel 계열 철자
- 남단 lequel / 여단 **laquelle** (l 두 개) / 남복 lesquels / 여복 **lesquelles**
- à 축약: auquel / à laquelle / auxquels / auxquelles (여단만 축약 X)
- de 축약: duquel / de laquelle / desquels / desquelles (여단만 축약 X)

### 수동태 시제 보존
- 능동의 시제를 그대로 유지
- 능동 PC *a fait* → 수동 PC *a été fait* (être가 PC 시제 잡음)
- 일치는 주어 기준 (être 조동사)
- *sont écrites* (현재 수동) ≠ *ont été écrites* (PC 수동)
