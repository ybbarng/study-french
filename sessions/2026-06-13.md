# 2026-06-13 — 약점 워밍업 + 고급 단계별 평가 재시험 1회차 (미통과)

가정문 3형/접속법 판별 집중 워밍업 → 고급 단계별 평가 재시험 1회차 (16.5/20 = 82.5%, 미통과). 어제 약점은 회복됐으나 **새 약점(가정절 Si를 분사구문 Ayant으로 대체)**이 드러남. 사용자 요청으로 저장 후 재시험 2회차 예정.

## 1. 워밍업 (가정법·접속법 혼합) — 첫 시도 5/10, 전부 즉시 회복/학습

| # | 항목 | 결과 |
|---|------|------|
| 1 | 가정문 3형 (aurais acheté) | ✅ 어제 핵심 약점 회복 ⭐ |
| 2 | 접속법 판별 (attendiez) | ✅ que→접속법, 조건법 침범 차단 |
| 3→4 | 조건법 과거 être조동사 | ❌ *aurait venu* → *serait venue* / Q4 *serais parti* ✅ 회복 |
| 5→6 | 분사구문 ayant+과거분사 | ❌ *Ayant comprennant* → *Ayant compris* / Q6 *Ayant fini* ✅ |
| 7→8 | vivre=avoir 조동사 + 철자 | ❌ *sont véçus* → *ont vécu* / Q8 *ai vécu* ✅ |

## 2. 고급 단계별 평가 재시험 1회차 — 16.5/20 (82.5%) ❌

- 어제 1차 80%와 비슷, 통과(95%) 미달
- **L1~2: 5.5/6 / L3~4: 9.5/9 / L5: 1.5/4** — 약점이 L5 자유 작문에 집중
- 틀린 문제: Q2(△ aies 철자), Q17(△ acheter 부정사), Q18(❌), Q20(❌)
- **Q11 무효 처리** — *alors que/tandis que*(대조 종속접속사)를 요구했으나 **미학습 형태** (배운 대조는 cependant/pourtant/néanmoins 부사). 사용자 답 *mais*는 합리적. 출제 오류 → 약점 아님. 재시험 전 대조 접속사 보강 필요 or 배운 형태로만 출제

### 🎯 새 약점 — 가정절 Si를 분사구문 Ayant으로 대체 (Q18·Q20 2회)
- Q18 *Ayant plus de temps* / Q20 *Ayant le temps* — 조건법 **주절은 맞힘**(aurais visité/visiterais), 가정절에서 Si+반과거/대과거 대신 Ayant 사용
- 원인: 어제 배운 분사구문이 활성화돼 가정문 침범 (접속법 잔상과 동일 현상)
- **개념 정리** (사용자 질문 "분사가 si 대신한다면서 왜 가정법엔 못 써?"): 분사구문은 **mood(법) 정보가 없어** 1형(실현가능 조건)만 대체 가능. 2·3형 반사실 가정은 시제(Si+반과거/대과거+조건법)로만 표현 → 분사 불가. 사용자 개발자 비유 "mood 비트 없음"으로 이해 ⭐

### ✅ 어제 약점 회복 확인
- 조건법/접속법 판별 (Q9 serions arrivés, Q19 법 대비 완벽 + ne explétif ⭐)
- plus de temps의 de (어제 ❌ → Q18 ⭕)
- 분사구문 ayant+과거분사 (Q8 Ayant perdu ⭕)
- 과거분사 일치 (Q16 écrites — 불규칙 écrit + 여복)

## 프랑스어 표현 교정

| 사용자 | 정답 | 비고 |
|---|---|---|
| aurait venu | serait venue | venir=être 동사 (조건법 과거도 être) |
| sont véçus | ont vécu | vivre=avoir 동사 + cédille 불필요 |
| est mouri | est mort | mourir 과거분사 불규칙 mort |
| beaucoup de l'eau | beaucoup d'eau | 수량부사 + de (관사 제거) |
| ais réussi | aies réussi | avoir 접속법 2sg = aies |
| il ne peut pas achete | ...acheter | pouvoir + 부정사 |
| Méme | Même | accent grave (è) |
| Ayant le temps (가정) | Si j'avais le temps | 분사구문은 비현실 가정 불가 |
| je aurais | j'aurais | 엘리지옹 (je + 모음) |

## 메타 인지·자가 도출 ⭐

- "분사가 si 대신한다면서 왜 가정법엔 못 써?" — 분사구문의 의미 한계를 정확히 질문 (mood 부재 원리 도출)
- naître=né / mourir=mort, vivre=avoir / naître=être 짝 정리
- "프랑스어는 알수록 비논리적" → 자의적 불규칙(성별) vs 복잡하지만 일관된 시스템(가정법/분사) 구별 제시

## 3. 대조(opposition) 표현 5형태 — 신규 학습 ⭐ (사용자 요청)

사용자 지적으로 시작: Q11(alors que/tandis que)이 미학습 형태였음 → 양보처럼 대조도 5형태 묶어 정식 학습. 사용자 원칙 제시: **"3개월 학습엔 빈 곳이 많다. 시험이 배운 범위를 벗어나면 안 된다 (미학습 예외 변화·연결 표현·특수 표현 출제 금지)."**

### 대조 5형태 (영어 매핑 — 사용자 자가 mnemonic ⭐)
| 프랑스어 | 영어 | 품사 | 결합 |
|---|---|---|---|
| mais | but | 등위접속사 | A, mais B |
| alors que / tandis que | whereas/while | 종속접속사 | **+ 직설법** |
| en revanche / par contre | on the other hand | 부사 | A. ~, B (문장 끊고) |
| cependant / pourtant | however | 부사 | A. ~, B (문장 끊고) |

- 사용자가 mais=but, cependant=however 매핑 도출 → 전체 5형태가 영어와 1:1 대응 확인 (노르만 정복 + 라틴 수사 전통). whereas가 직설법 받는 것 = alors que 직설법.
- **출제 약속**: 동의어(par contre/pourtant/néanmoins)는 "읽기 인지만", 시험엔 en revanche/cependant 2개만.

### 드릴 결과 (전부 ✅)
- **alors que** 3회: 대조(현재+현재) / 시간(반과거+PC) / 엘리지옹(qu'elle). 시제 패턴으로 시간·대조 구별 정착
- **tandis que** 2회: 직설법 정착. (vive ❌는 접속법 착각이 아니라 vivre 직설법 활용 미숙 — 사용자 정정)
- **en revanche** (대비) / **cependant** (역접): 각 명확 문맥 + 구별 드릴 2회 (역접→cependant, 대비→en revanche) 모두 ⭕

### 부수 학습
- **vivre 직설법 현재**: 단수 vi-(vis/vis/vit) / 복수 viv-(vivons/vivez/vivent). -re 동사 1인칭 = -s (e 아님)
- **boot vs 단복 2분할 구별** (사용자 지적): vivre는 ils=viv-라 boot 아님 → 단복 2분할 (écrire/lire/dire류). boot은 ils가 단수쪽(venir/prendre)
- **대과거 vs PC** (사용자 질문): 선후 있다고 무조건 대과거 아님. "기준 과거 시점 + 그 이전 완료"일 때만 대과거 (간접화법·가정3형·시간배경). 단순 순차 나열은 PC
