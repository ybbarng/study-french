## S5 간접 의문문 본 학습

### 핵심: 변환 3단계
1. **연결사 선택** (의문 유형에 따라)
2. **어순 정상화** (도치 제거)
3. **시제 일치** (주절 과거 시)

### 단위 A — Yes/No → si

| 직접 의문 | 간접 의문 |
|----------|-----------|
| Tu viens? / Est-ce que tu viens? / Viens-tu? | **si** |

**s' 축약**: si + il/ils만 (모음 충돌 회피)
- s'il, s'ils ⭕
- si elle, si on, si je 그대로

### 단위 B — 사물 의문사 → ce que / ce qui

| 직접 의문 | 간접 의문 | 역할 |
|----------|---------|------|
| que / qu'est-ce que (사물 목적어) | **ce que** | 목적어 |
| qu'est-ce qui (사물 주어) | **ce qui** | 주어 |

**판별 트릭**: 종속절에 명시적 주어 있나?
- YES → 의문사 = 목적어 → ce que
- NO → 의문사 = 주어 → ce qui

### 단위 C — 사람 qui + 부사 의문사 (그대로)

**사람 vs 사물 비대칭 (시험 함정)**:
| | 주어 | 목적어 |
|-|------|--------|
| 사물 | ce qui | ce que |
| 사람 | qui | qui |

→ **사람 qui에 절대 ce 붙이지 말 것**

**부사 의문사 5개**: où / quand / comment / pourquoi / combien — **그대로 유지** (도치만 풀기)

### 단위 D — 시제 일치 (주절 과거 시)

| 직접 의문 시제 | 간접 의문 시제 |
|--------------|--------------|
| présent | imparfait |
| passé composé | plus-que-parfait |
| futur | conditionnel présent |
| futur antérieur | conditionnel passé |

→ 10-2 간접화법과 동일 원리 (5/22 정착 영역)

### demander의 두 용법 ⚠️

| 용법 | 구조 | 법 | 의미 |
|------|------|----|------|
| 요구 | demander que + 절 | **접속법** | "~할 것을 요구" |
| 간접 의문 | demander si / 의문사 + 절 | **직설법** | "~인지 묻다" |

→ 연결사로 구별: que → 접속법, si/의문사 → 직설법
