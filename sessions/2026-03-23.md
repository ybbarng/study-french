## 새로 배운 개념

### 4-2. 현재시제 활용
1. **현재시제 3가지 용법**: 현재 진행/상태, 일반적 사실/습관, 가까운 미래
2. **진행형 없음**: ~~Je suis mangeant~~ ❌ → Je mange (= I eat / I am eating). 강조 시 être en train de + 부정사
3. **동사 3군 분류**:
   - 1군(-er): 90%, 규칙적. 어미 -e/-es/-e/-ons/-ez/-ent
   - 2군(-ir, -issons): 규칙적. 어미 -is/-is/-it/-issons/-issez/-issent
   - 3군: 불규칙. boot pattern (부츠 패턴)
4. **1군 특수 케이스**:
   - -ger → nous에서 e 삽입 (mangeons, nageons, voyageons)
   - -cer → nous에서 cédille (commençons, lançons, plaçons)
   - -é_er → 닫힌 음절에서 è (préfère, espère)
   - -eler/-eter → è (gèle, achète), 단 appeler/jeter만 자음 겹침 (appelle, jette)
5. **3군 boot pattern**: je/tu/il/ils = radical 1, nous/vous = radical 2
6. **주요 3군 동사 활용 암기**: prendre, venir, partir, pouvoir, vouloir, devoir, faire, aller
7. **faire 불규칙**: vous faites (pas faisez), ils font (pas faisent)
8. **aller 완전 불규칙**: 3개 어근 (v-, all-, vont)
9. **pouvoir/vouloir**: je/tu에서 -x (pas -s)

## 반복 필요 항목
- -al → -aux 남성 복수 (finaux, résultats=남)
- 최상급 범위 de (dans ❌)
- film = 남성 (Ce film, intéressant)
- vous faites, ils font (faire 불규칙)
- prennent, viennent — ils에서 double n
- venons — nous에서는 n 하나
- appeler → appelle, jeter → jette (자음 겹침)
- préférez (nous/vous에서는 é 유지)
