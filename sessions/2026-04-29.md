## 새로 배운 개념

### 9-4 subjonctif présent 활용 (정확한 규칙)

**핵심 규칙 (2부분):**
- 어미 (모든 동사 동일): -e / -es / -e / -ions / -iez / -ent
- 어간:
  - je/tu/il/ils → **ils 어간** (현재 ils형 - ent)
  - nous/vous → **nous 어간** (현재 nous형 - ons)

**단일 어간 동사** (parler, finir 등): nous 어간 = ils 어간 → 모든 인칭 동일 어간
**이중 어간 동사** (prendre, venir, boire, devoir, recevoir): 두 어간 분리 적용

### 핵심 통찰 (사용자 발견)

1. **현재형 boot pattern = 접속법 이중 어간**
   - 현재형에서 nous/vous 어간이 다른 동사 = 접속법에서도 그 패턴 그대로
   - "현재형 잘 알면 접속법도 자동"

2. **subjonctif nous/vous = imparfait nous/vous**
   - 둘 다 nous 현재 어간 + -ions/-iez
   - 5개 예외 (être, avoir, faire, savoir, pouvoir)만 다름

3. **모국어 화자는 형태 인지보다 트리거 자동 반응**
   - 규칙 동사는 직설법과 형태 같음 → 트리거가 모드 결정

### 7개 불규칙 동사 소개
- être (sois/soyons/soyez/soient ...)
- avoir (aie/ayons/ayez/aient ...)
- faire (fass-)
- aller (aill- / all-)
- pouvoir (puiss-)
- savoir (sach-)
- vouloir (veuill- / voul-)

명령형과 부분 일치:
- être/avoir: 명령형 = 접속법 tu/nous/vous (완전 일치)
- savoir/vouloir: 명령형 ≠ 접속법 nous/vous (부분 일치)

### 학습 우선순위
3인칭 단수 5개 먼저 자동화: soit, ait, fasse, puisse, aille

## 메타 토론 (언어학)

### Zipf의 법칙 + 빈도 효과
- 단어 빈도는 power law 분포 (1/순위)
- 고빈도 = 짧음 (Zipf brevity law)
- 고빈도 = 불규칙 보존 (frequency effect)
- → être/avoir는 천 년 후도 불규칙일 것

### 1군 동사 평준화 가능성
- 신어는 거의 100% -er
- 3군은 죽은 클래스 (신어 없음)
- 천 년 단위로 일부 평준화 가능, 고빈도 동사는 보수적

### 음운 진화 vs 선점 모델
- 사용자 가설: 짧은 형태를 빈도 높은 단어가 선점
- 실제: 음운 변화는 모두 적용, 고빈도가 더 빨리 침식
- 결과 동일 (고빈도 = 짧음), 메커니즘 다름

### 학습 방법론 토론
- 자료 저자: 형태 먼저 → 사용 → 뉘앙스
- 우리 접근: 개념 먼저 → 형태 → 사용
- 결론: 9-3까지 개념 충분, 9-4부터 형태 + 출력
